강해설교

정의

성경 본문의 핵심 메시지와 저자의 의도를 효과적으로 전달하는 설교 방법론. 본문 중심의 원리를 기본으로 하되, 청중의 삶에 적용되고 변화를 일으키는 것을 목표로 한다.

핵심 원리

하나의 중심 아이디어

  • "잘 준비된 설교는 하나의 핵심적인 사상의 구현이요 발전이며 완전한 진술이다" (데이비스)
  • "설교자는 본문을 강해할 때 단 하나의 지배적인 테마와 관련하여 발전시켜야 한다" (알렌 스팁스)
  • "어떠한 설교이든 한 가지 중심 아이디어를 가져야만 한다" (밀러)

로빈슨의 3구조

  1. 설명 ("이것이 무슨 뜻인가")
  2. 증명 ("그것이 사실인가")
  3. 적용 ("그것이 나와 무슨 상관이 있는가")

설교의 권위

  • "설교자의 단 하나의 권위는 하나님의 말씀을 신실하게 전달하는 데서 나온다" (존 파이퍼)
  • 텍스트가 최후의 결정적인 권위

강해설교의 목표

  • 지적 변화: 무지함을 깨우침
  • 감정적 변화: 가슴에 호소
  • 의지적 변화: 삶의 변화

강해설교 vs. 다른 설교 형태

  • 주제설교: 주제 중심, 본문은 보조
  • 강해설교: 본문 중심, 주제는 본문에서 도출
  • 내러티브 설교(신설교학): 귀납적·이야기 구조

이미지 설교와의 관계

이미지 설교(이광재)는 강해설교의 기본 범주 안에서 이미지를 활용하는 방법론

강해설교준비실습 — 1강: 설교자로 헌신하기

설교자란 누구인가

정의 — 4인의 목소리:

  • 스티븐 스미스: "설교자란 자신이 죽어서 하나님의 말씀이 살아나게 하는 자이다." — 자기 말이 아니라 하나님 말씀을 하는 것
  • 다니엘 도리아니: "설교자는 영적인 산파와 같다. 아기를 낳는 자가 아니라 하나님이 말씀을 통해 영적 생명을 만들어 내시는 것을 돕는 자다."
  • 어거스틴 (팀 켈러의 설교 인용): 설교자의 의무는 ① 프로바레(probare) — 지도하고 증명하는 것, ② 델렉타레(delectare) — 마음을 사로잡고 즐겁게 하는 것, ③ 플렉테레(flectere) — 사람들의 마음을 휘젓고 감동시켜 마침내 행동하게 하는 것
  • 스펄전 (마틴로이드존스 인용): "만약 다른 일을 할 수 있다면 그 일을 하십시오." — 소명이 얼마나 그를 압박하는지 "난 다른 일을 못한다. 설교만 해야 한다"라고 말하게 된다.

설교자는 무엇이 아닌가:

아닌 것 이유
선지자 하나님 말씀을 직접 대언하는 직접적 권위는 없음 (벧후 1:21)
사도 사도의 권위는 열두 제자와 바울에게만 국한
거짓선지자 본문과 무관하게 자기 말을 선포, 대중의 비위를 맞추는 자
말쟁이 한 조각씩 끌어 모은 사상을 유통시키는 자, 자신의 생각이 없음 (행 17:18)
강단꾼 내용보다 형식, 웅변과 쇼맨십으로 청중 감정을 조작하는 자

"강단꾼이 미친 악영향: 내용보다 형식이 중요해졌고, 설교는 결국 오락의 한 형태로 전락했다." — 마틴로이드존스

설교자의 5가지 필수 이해

김운용 설교의 새로운 패러다임 — 설교자는 예언자 전통 안에 서 있는 동시에 시인이어야 한다:

  1. 텍스트 이해 — 텍스트가 스스로 목소리를 낼 수 있도록 허용하는 것 (월터 브루그만)
  2. 설교 이해 — 설교는 성육신적 사건. 하늘의 메시지가 이 땅에 구현되는 것
  3. 시대(문화) 이해 — 그 시대를 아파하고 하나님 말씀이 효과적으로 들려지기를 추구
  4. 회중 이해 — 회중은 단 하나: "하나님의 은혜가 필요한 죄인, 은혜로만 살아갈 수 있는 성도"
  5. 복음 이해 — 설교자의 가장 중요한 임무는 복음의 비밀을 이 시대에 어떻게 전할 것인가

존 파이퍼 — 강단에서 기억해야 할 6가지

  1. 본문의 내용을 충실히 보여주지 못하면, 영광스러운 진리를 드러내는 데 실패한 것이다.
  2. 성경을 기뻐하지 않는 설교자는 게으르거나, 영적으로 둔감하거나, 무감각하다는 것을 드러낸다.
  3. 설교자의 단 하나의 권위는 하나님의 말씀을 신실하게 전달하는 데서 나온다.
  4. 본문에 뿌리를 두지 않은 설교는 교인들에게 성경·교리적 결함을 남긴다.
  5. 본문의 세부 내용에 주의를 기울이지 않으면, 영적·도덕적 권면의 특수성에 둔한 교인을 기른다.
  6. 본문과 동떨어진 설교는 교인에게서 예수 그리스도께서 말씀하신 기쁨을 빼앗는다.

2주 토론 — 보충 인사이트

존 스토트:

"설교야말로 교회의 주된 임무이자 목회자의 주된 임무이며, 그 밖의 모든 것은 보조적인 장치로서 외곽에서 설교를 받쳐 주고 매일의 삶을 통해 실천하는 방편이다"

설교자의 유혹과 권위: 성경을 떠나 정치적 이념·경제이론·종교철학·현대 심리학 등을 강단에서 전하는 유혹에 빠지는 순간 설교자는 스스로 권위를 포기하는 것. 하나님은 성경을 통해 말씀하시고, 성경 강해설교가 신적 권위를 가장 온전하게 전달한다 (딤후 3:15-17).

설교 그립 비유

야구공은 그립 없이도 던질 수 있다. 그러나 어떤 그립으로 어떤 구질의 공을 던지는지 아는 것이 설교자를 성장시킨다. 강해설교 = 직구 그립, 이미지 설교 = 변화구 그립.

결국 설교는 설교자의 인격에 의해 전달된다 — 에토스(신뢰), 파토스(공감)

설교 = 해석 + 전달

해석 (무엇을 전달할 것인가) 전달 (어떻게 전달할 것인가)
① 문자적·역사적·문법적·문맥적 연구 ① 개념 전달
② 중심명제(개념) 찾기 ② 효과적·현실적 의사전달
③ 개념 적용 (설교목적+대지 찾기) ③ 삶의 변화를 목적으로

두 현실(성경 본문의 현실 ↔ 우리가 사는 현실) 사이에 브릿지를 놓는 것이 적용이자 설교다.

강해설교의 결론 — 3 기둥

기둥 내용
본문중심 본문의 장르·무드를 파악하고, 역사적·문법적·정황적 배경 위에서 본문을 벗어나지 않는 설교
개념중심 성경저자의 의도를 파악하고, 본문을 하나로 묶는 핵심주제를, 그 책의 전체주제 안에서 해석
적용중심 원청중에게 전하려는 메시지를 → 오늘 청중에게 분명하고 기억하기 쉬운 언어로 → 지·정·의의 변화를 이끌어냄

강해설교가 아닌 것 5가지

  1. 종교적 담화 — 성경과 무관한 제목설교
  2. 문맥 무시 설교 — 석의 없이 해석하는 설교
  3. 단편적 나열 설교 — 단어·구절을 관계없이 열거
  4. 단순한 석의설교 — 석의는 과정이지 목적이 아님
  5. 관주적 설교 — 성경구절만 잔뜩 연결해 나열

강해설교준비실습 — 2주차 수퍼비전: 본문 선택 & 본문 연구

본문 선택 방식

  • 산발적 선택: 다양한 주제·상황 대응, 유연성 있음
  • 집중적 선택: 권별 전체 / 권의 일부 / 장별 / 주제 시리즈
  • 권별 전체: 전체 시각에서 부분을 볼 수 있음
  • 권의 일부(한 인물 중심): 전체 주제 안에서 인물의 행동·변화를 볼 수 있음
  • 연속 본문 중에도 교회력·국가력(어린이주일·어버이주일)·특별행사·예기치 않은 사건 고려 필요

→ 항상 성도(청중)를 염두에 두어야 한다. 아무리 좋은 설교라도 때에 맞는 말이 아름답다 (잠 15:23).

본문 연구 2단계

1단계 — 보는 작업 (관찰)

성경 본문을 지적으로, 목적 의식을 가지고, 순종하려는 자세로 연구하는 것. - 긴 단어, 특이한 단어, 반복되는 단어 주시 - 관계 탐색: - 문법적 관계: 단어들 사이 바른 이해를 위해 - 논리적 관계: 원인과 결과, 대조, 비교, 조건, 목적 → 저자의 요점·논리적 주장 파악 - 시대적·지리적 관계: 본문 구조 이해에 도움 - 심리적 관계: 비유·기적·인간관계를 포함한 특정 문학양식 본문의 올바른 이해를 위해 - 문맥적 관계: 책 전체를 이해해야 저자가 표현하는 심각한 문제들을 올바르게 이해할 수 있음

2단계 — 찾는 작업 (해석)

관찰한 내용으로 질문을 던지고 답변하는 과정. 해석의 목적 = 저자의 의도 이해.

던져야 할 질문 4종류: 배경에 관한 질문 / 사실에 관한 질문 / 의미에 관한 질문 / 적용에 관한 질문

해석 오류 유형: 본문을 틀리게 읽는 것 / 성경을 왜곡하는 것 / 본문에 모순되는 해석 / 주관적 해석 / 상대주의적 해석 / 지나친 자신감

묵상맵

문자로 된 것을 입체화하는 도구. 본문을 다르게 볼 수 있게 만든다.

과정: 1. 등장인물 찾기 2. 인물의 말과 행동 정리 3. 질문 던지기 (Why? What? How?) 4. 비교·강조·반복되는 부분 탐색 5. 질문에 대한 묵상 (해석 + 상상력) 6. 전체를 하나로 연결할 주제 찾기

장점: 본문을 정확하게 이해 / 한눈에 비교·분석 / 사고 확장 / 저자의 의도와 목적 발견

이상협 자기 성찰 — 주제설교의 한계

지금까지 주제 특강 중심 → 본문은 주제에 맞게 산발적 선택 → 깊은 해석 없이 표면적 의미 인용 → 설교자의 생각과 의도 전달이 우선이었음.

장점은 있었다: 성경 전체 문맥은 고려됐고, 청중의 필요에 민감했으며, 교육적이었다.

그러나:

"주제의 근원이 비록 설교자의 개인적인 생각이나 회중의 삶·필요에 의한다 할지라도, 그 주제를 제어할 수 있는 본문과의 연관이 깊어야 한다. 주제를 풀어나가기 위해 본문을 사용하거나, 설교자의 말을 합리화하기 위해 본문을 사용하는 오류를 꼭 피해야 한다."

강해설교준비실습 — 3주차 강의: 본문연구와 묵상맵 심화

2주차 수퍼비전에서 다룬 본문 선택·연구·묵상맵의 심화 버전. 구체적인 방법론과 학자별 도구를 추가로 정리한다. 연결: 성경해석학 · 해돈로빈슨 · 존스토트

본문 선택 — 집중적 선택 예시들

  • 권별 전체: 칭의→성화→영화의 큰 틀에서 각 부분을 바라보며 설교 준비. 전체 시각에서 부분을 볼 수 있음
  • 권의 일부 (인물 중심): "껍질을 깨고 비상하라" 같은 전체 주제 안에서 인물의 행동과 변화를 추적. 성경 전체주제로 각 사건을 바라볼 때 새로움이 있다
  • 주제 시리즈: 자녀·부부 공감·하모니·화목 등 주제별. 강단 배경·소품·드라마·영상까지 분위기에 맞춤. 현장에서 진행하는 경우도 있다
  • 기타: 큐티 → 은혜 → 주일설교 연결 / DBS(귀납적 성경공부) 활용

본문 관찰 4가지 요소 (심화)

2주차의 관찰 2단계에서 더 구체화한 분류

  1. 단어들 — 반복되는 단어, 중요한 의미를 가진 단어
  2. 관계 — 문법적·논리적·시대적·지리적·심리적·문맥적·인과적 관계
  3. 장르 — 문학 양식 (이야기·역사·비유·예언 등)
  4. 분위기 — 저자의 의도

하워드 핸드릭스 (『삶을 변화시키는 성경연구』) — 효과적인 관찰 6가지 실마리: 1. 강조되고 있는 것들 2. 반복되고 있는 것들 3. 서로 연관된 것들 4. 서로 비슷한 것들 5. 서로 상반된 것들 6. 실생활에 도움이 되는 것들

관찰 시 주의사항: - 본문을 빠뜨리지 말고 제대로 읽어라 - 선경험과 선이해를 다 버려라 - 성경본문을 설교의 바탕(basis)에 깔아라 - 저자가 우리를 어디로 끌고 가는가를 기억해야 한다 - 본문의 근거를 제시하라 - 자연스럽게 본문에 접근하라

월렌 위어스비 — 설교 준비를 위한 관찰 7가지 질문: 1. 본문은 무엇을 말하는가? 2. 본문은 그것을 어떻게 말하는가? 3. 본문은 처음 청중들(성경시대)에게 무엇을 의미하였는가? 4. 본문이 오늘의 교회에는 무슨 의미가 있는가? 5. 본문이 내게 의미하는 것은 무엇인가? 6. 본문이 청중들에게 무엇을 의미하는가? 7. 설교자가 청중들에게 어떻게 본문을 의미 있게 만들 것인가?

묵상맵 — 추가 장점

2주차 수퍼비전의 묵상맵 6단계 원리와 함께 보라

  • 이야기나 줄거리가 없어도 묵상맵을 그릴 수 있다 — 하나님과 우리(나)의 구도로도 가능
  • 묵상맵 훈련을 지속하면 나중에 묵상맵을 그리지 않아도 본문을 바라보는 사고 자체가 확장된다

강해설교준비실습 — 4주차: 메인 아이디어(중심 명제) 찾기

연결: 해돈로빈슨 · 3주차 묵상맵 → 중심명제 찾기로 이어지는 흐름

중심 명제란?

  • 본문의 심장. 성경저자의 의도와 목적이 드러나는 것
  • 메인 아이디어, 개념(Concept)이라고도 함
  • "개념": 분리되어 있던 사물들이 하나의 통일성을 이룰 때 전에 인식하지 못했던 아이디어가 떠오르기 시작하는 것
  • 설교 = 하나의 단순하고도 모두를 포괄하는 개념을 구현하는 것 → 반드시 하나의 메인 아이디어를 가져야 한다
  • 중심 사상은 개요가 아니며, 이미지나 그림들을 통해 청중들의 마음속에 새겨진다

중심 명제의 구조

중심 명제 = 주요소(주제) + 보충요소(보어/술어)

요소 질문 내용
주요소(주제) "내가 무엇을 말하려 하는가?" 본문의 요점, 성경저자의 의도
보충요소(보어) "내가 말하는 바에 대해서 무슨 말을 하고 있는가?" 대지, 서술, 설명

해돈로빈슨: "하나의 아이디어는 보충요소가 명확한 주요소에 결합될 때 발생한다. 본문의 사상을 주요소와 보충요소로 결합된 하나의 문장으로 표현하라." 라메쉬 리차드: "중심 명제는 본문이 가르치는 핵심적인 내용을 한 문장으로 표현한 것이다."

매튜슨의 3단계 아이디어 — 중심명제 세분화

아이디어 정의 필요 역량
주해적 아이디어 성경저자가 의도한 바를 정확하게 반영 — 주해작업을 통해 얻어진 개념 성경을 잘 알고 관련 지식이 많아야 함
신학적 아이디어 주해적 아이디어를 모든 사람에게 적용되는 보편적 의미로 전환 인류보편적인 인문학적 지식
설교적 아이디어 지금·여기·오늘 우리에게 필요한 의미 — 청중에게 연결 청중들의 구체적인 필요에 관심

주해적 아이디어만 좋으면 → 강의가 될 것이고
설교적 아이디어만 좋으면 → 자신의 말잔치가 될 것이다

흐름: 주해적 아이디어 → 신학적 아이디어 → 설교적 아이디어 → 청중에게 연결

중심 명제 찾는 실제 방법들

  1. 내용별 주제 요약법: 본문 내용별로 주제 요약 → 하나로 묶을 수 있는 주제 찾기 (반드시 본문 전체와 연결) → 그 주제를 설명하는 술어 찾기
  2. 내용 재분류법: 절이 아닌 내용에 따라 새롭게 분류
  3. 하나님-인간 대비법: 본문에서 하나님의 모습 + 타락한 인간의 모습을 찾아 대비 → 타락한 인간의 본성에 임하신 하나님의 은혜로 중심명제 정리
  4. 장르별 접근법: - 내러티브: 스토리 안에 있는 주제로 - 서신서·시가서: 각 문단의 주제 → 전체 중심주제, 또는 내용에 따라 전체 구성 후 연결 주제 찾기
  5. 묵상맵 활용: 모든 등장인물들과 연결되는 공통의 주제 찾기 → "성경저자가 무엇을 말하려 하는가?"를 질문

그래도 찾기 어렵다면: 주제 = 하나님의 모습, 술어 = 타락한 인간의 본성 → 타락한 인간의 본성에 임하신 하나님의 은혜로 정리

TMI 경고 — 단 하나의 주제

설교자가 빠지기 쉬운 TMI(Too Much Information) — 저자의 의도보다 더 많은 이야기를 전달하려는 유혹. 특히 MZ세대 청중은 TMI를 싫어한다.

중심적이고 일원화된 아이디어가 효과적인 설교의 핵심. 이 원리를 무시하면 안 된다.

명확하고 중요한 한 가지 주제를 강력하게 전달하는 것이 설교의 핵심이다.

강해설교준비실습 — 5주차: 설교의 목적

연결: 해돈로빈슨 · 4주차 중심명제 찾기 → 목적 설정으로 이어지는 흐름

WHY로 시작하라

사이먼 시넥 『위대한 리더들의 공통된 패턴』(TED 강의):

구조 일반 광고 애플·나이키
순서 What → How → Why Why → How → What
초점 제품·서비스 설명 신념과 목적
효과 기억에 남기 어려움 브랜드 충성도

설교도 WHY(목적)를 분명히 하면, 청중들이 하나님께 충성을 다하게 된다.

중심명제 vs 설교의 목적

스티븐 D. 매튜슨: - 설교의 중심명제 = 화살 (진리를 진술) - 설교의 목적 = 과녁 (진리가 선포됨으로 달성하려는 것)

학자별 설교의 목적 정의

학자 정의
라메쉬 리차드 "본문의 중심명제에 근거해서 하나님께서는 성도들이 무엇을 이해하며 순종하기를 원하고 계시는가?"
해돈로빈슨 "설교의 결과로서 청중에게 일어나기를 기대하는 것" — 하나님의 필요(저자의 목적) + 청중들의 필요(설교자의 인식)
토마스 롱 목표(focus) = 설교를 통해 무엇에 관해 말할 것인가 / 목적(function) = 설교가 청중과 함께 행하려는 것, 청중에게 일어나기 원하는 변화

토마스 롱의 목표·목적 = 설교의 나침반: 통일성, 명확성, 연결성 확보. 세 가지 기준: ① 본문의 석의에서 나올 것 ② 목표와 목적이 서로 연결될 것 ③ 통일성과 단순성 유지

설교 목적의 분류 (장두만)

  1. 설명: 교리·가르침이 무엇을 의미하는지 분명히 함 (단순 정보제공만으로 끝나면 불충분)
  2. 설득: 성경의 교리·가르침을 사실로 받아들여 믿게 함 (예: 부활, 동정녀 탄생)
  3. 행동촉구: 청중이 기대한 어떤 행동을 하도록 이끎

설교 목적의 필요성 6가지 (장두만)

  1. 설교 서론의 필요성을 분명하게 해준다
  2. 설교를 단순화시켜 포함·제외를 결정해준다
  3. 설교의 결론과 적용에 영향을 미친다
  4. 예화 선택에 도움을 준다
  5. 설교 성공 여부를 측정할 수 있는 객관적 방법을 제공한다
  6. 중심주제가 어떤 형태를 취해야 할지 알게 해준다

설교 목적 찾는 방법

  1. 본문의 저자가 왜 이것을 썼는가? 그는 독자들에게 무슨 효과를 기대하였는가?
  2. 하나님께서 이 설교를 통해 청중들에게 무엇(결과)을 이루시기를 원하시는가?
  3. 저자의 목적을 설교의 목적으로 바꿀 때 두 목적이 서로 조화를 이루는가?
  4. 설교의 목적이 청중의 필요에 부합되는가?

⚠️ 청중의 필요에 의해 본문을 왜곡시키거나, 본문에 없는 것을 억지로 갖다 붙이면 안 된다. 설교의 목적은 본문의 가르침으로부터 자연스럽게 흘러나와야 한다.

그리스도 중심 설교

브라이언 채플 — 그리스도 중심의 설교: 1. 성령이 본문 말씀을 통해 구원을 베풀어 주신 하나님의 본성을 어떻게 보여주는가? 2. 성령은 이 본문 말씀을 통해 구원이 필요한 인간의 본성을 어떻게 보여주고 있는가?

"본문에서 증거하는 그리스도를 드러내지 않은 채 성경적 행동만을 가르친다면 그것은 사람들에게 상처만 입히는 것이다"

팀켈러 — 그리스도 설교: - 매번 그리스도를 설교하라. 성경의 모든 장, 부분, 인물, 이미지, 구원이야기에서. - 설교주제 선택 전 3가지 질문: 1. 본문의 주된 사상 — 무엇에 관해? 2. 목회적 목표 — 원독자·오늘 청중에게 무엇을? 3. 그리스도 질문 — 본문은 어떻게 그리스도를 가리키는가? 그분의 구원은 어떻게 우리를 변화시키는가?

"우리의 설교는 자주 율법적이고 윤리적으로 흘러가기 쉽다. 그런 의미에서 그리스도라는 필터를 써 볼 필요가 있다."

중심명제 → 목적 → 설교 흐름 정리

본문의 중심명제(저자의 의도) ↓ 설교의 목적(설교자의 기대) ↓ 설교의 중심명제(설교자의 메시지)

설교 목적의 효과 — 청중에게 생겨나야 할 3가지 질문

  1. 자기발견 — 하나님이 왜 이 설교를 통해 나에게 무엇을 말씀하시는가?
  2. 자기갱신 — 하나님은 내 삶의 어떤 것을 바꾸기를(깨닫기를) 원하시는가?
  3. 결단 — 이제 나는 어떻게 살 것인가?

1. 연속주해식 설교 (주해설교)

역사적 계보: 학사 에스라(느 8:8, 낭독-해석-적용) → 쿰란 공동체 해석자 → 오리겐·크리소스톰·어거스틴 → 종교개혁자 루터·칼빈 → 존 맥아더·마틴 로이드 존스

정의: 한 구절 다음에 다음 구절을 이어서 해설하며 성경을 읽어가는 설교. 발췌된 단어를 분석하며 성경주해처럼 진행.

장점 약점
본문을 깊게 분석함 성경의 모든 장르에 동일 적용 불가 (시편·역사서는 완전한 사상을 위해 상당한 분량 필요)
설교자 자신의 견해보다 저자의 의도 드러냄 통일성 부족 — 세부는 잘 다루나 전체 그림 실패 가능
본문의 윤곽을 따라 설교 가능 가공되지 않은 연구 자료가 너무 많이 유입, 적용 빈약 가능
진행 속도 느려 청중이 하나님의 전체 경륜을 듣지 못할 수 있음

"강해설교는 통일성과 개요, 전체적인 지향점은 하나도 없이 그저 한 단어에서 다음 단어로, 한 구절에서 다음 구절로 진행되는 주해와는 전혀 다르다." — 리차드 메이휴

효과적인 개선 제안: 1. 사상의 기본 단위를 설교하라 (전체 그림 염두) 2. 단어·구절 집중 말고 연결관계 부각 3. 무엇을 자세히, 무엇은 개략적으로 다룰지 미리 결정 4. 구절 단위 연속주해 + 문단 단위 연속주해 병행

2. 대지설교

정의 (정장복): 설교의 목표는 본문의 교훈과 메시지를 논리적·일목요연하게 전달하여 이해시키고 기억하게 하는 데 있다. 복잡한 내용을 3-4가지 주제로 선명하고 단순하게 요약하며, 연역적 구조.

역사적 기원: 한국 초기 선교사 곽안련(C. Clark, 미국 북장로교, 평양신학교 1901-)이 도입. 1910년 『강도요령』, 1925년 『설교학』 출판.

곽안련이 대지설교를 권한 4가지 이유: 1. 신학교육을 제대로 받지 못한 초보 설교자들이 익히기 쉬움 2. 성경 본문에서 크게 벗어나지 않는 본문 설교 3. 성경 지식이 없는 초기 한국 교회 청중에 맞게 — 전체를 통으로 주면 붙잡을 수 없으니 분할 제공 4. 거짓 교리 만연한 사회에서 논증적인 대지설교가 신앙을 굳건히 세우는 데 적합

강해설교 vs 대지설교 차이:

구분 강해설교 대지설교
의도 성경저자의 의도 본문이 말하는 의도(설교자의 의도)
구조 모든 대지 간 논리적 연결·흐름 각 대지의 주제 부각이 우선
에너지 본문 석의에 집중 전달하려는 내용(대지)에 집중

대지설교 → 강해설교로 전환하는 팁: 1. 본문 속에서 핵심 대지를 찾는다 2. 그 대지를 하나로 묶는 하나의 결론을 도출한다 3. 그 결론을 하나의 개념으로 연결해서 중심명제를 완성한다

예시 (히 11:8, 17-19): - 대지설교: 믿음의 이동 3대지 (공간의 이동 / 대상의 이동 / 삶의 문제의 이동) - 중심명제: "나 중심에서 하나님 중심으로 믿음의 이동이 일어나야 한다"

3. 주제설교

정의: 설교자 자신이 선택한 주제를 입증하기 위해 성경의 여러 본문을 사용하는 설교.

방법 3가지: 성경 인물 주제 / 현대 사회 문제 (동성애·세습·인권·재개발) / 신학적 논제 (교회·제자도·결혼·용서)

장점 (주성호): 원하는 특정 주제 설교 가능 / 광범위한 주제 / 목표 분명 / 문학적 자질 계발 / 창조적 설교 / 논리적으로 확신 / 조직과 구성 용이

단점 (정장복): 1. 주제의 근원이 설교자 개인 생각이나 회중 삶에만 국한됨 2. 설교자의 주관적 사상을 하나님의 말씀과 동일시하는 오류 가능 3. 주제를 풀기 위해 본문을 징검다리로 사용하게 됨

주제설교 → 강해설교로: 설교자가 택한 주제와 성경 저자가 말하려는 부분을 일치시키는 것. 본문에 대한 철저한 주해와 맥락 연구 필요. 오늘날 다양해지는 사회·문화 문제에 '복음적인 해석과 기준' 제시를 위해 귀하게 쓰일 수 있다.

4. 문제해결식 설교 (처방전, 상담적 설교)

시작: Harry Emerson Fosdick — "집단 상담적인 설교". 개인 상담에서 발견한 현대인의 문제·고민에서 출발하여 성경적 해답 제시.

Fosdick의 5단계 구조: 1. 청중들의 개인적 관심과 필요에서 시작 (삶의 긴급한 문제·이슈) 2. 청중들의 삶의 문제를 함께 통찰 — 문제 분석, 성경적·역사적 근거 제시 3. 청중 스스로 자신의 정체성 발견 — 마음과 양심에 도전 4. 설교자의 직관 제시 — 성경적·신학적 해결점과 복음적 대안 5. 성경적 해답으로 청중들이 행동하도록 동기부여

→ 이후 신설교학(크래독 귀납적 설교)으로 이어짐.

연역적 설교 vs 귀납적 설교

연역적 방식: 결론에서 시작 → 구체적 사례로 (전체→부분). 해답이 먼저 나오고 나머지는 이를 분해·분석. - 장점: 설교가 선명해짐 - 단점: 지루해질 수 있음 - 구조: 아이디어 진술(서론) → 설명·증명·적용(본론) → 반복(결론)

아이디어 3가지 전개 방식: 1. 설명 — "이 본문이 무엇을 의미하는가?" (예: 골 1:15-18 맥클라렌) 2. 증명 — "그것이 사실인가?" (예: 고전 15:12-19) 3. 적용 — "이것이 나와 무슨 상관인가?"

귀납적 방식: 구체적 사례에서 시작 → 결론에 도달 (부분→전체). - 서론은 첫 번째 논지만 다루고, 앞의 논지와 연결된 새 논지들이 전개됨 - 결론에 이르러 설교 아이디어가 완전히 제시됨 - 장점: 발견의 기쁨, 청중이 메시지를 받아들이지 않을 수 없게 됨, 긴장감과 집중력

귀납적 방식 핵심 원리: 1. 전환이 중요 — 중심 사상 없이 진행하므로 적절한 전환으로 지금 어디에 있는지 알려줌 2. 설교가 정교하지 않으면 청중이 따라갈 수 없음 3. 마지막 중심사상이 앞의 문제의 처방이 됨

연역+귀납 혼합: 앞부분은 귀납적 → 설교 중간쯤 아이디어 제시 → 이후부터 연역적으로 설명·증명·적용

귀납적 설교 대표 학자들

크래독 (Craddock) — 귀납적 설교: - "설교의 코페르니쿠스적인 혁명" — 논리적·명제 중심 설교 틀을 깨고 새 패러다임 제시 - 구조: 오늘의 삶과 관련된 예 → 본문 전후 상황 실례1 → 본문 접근·문제 제시 실례2 → 공동체 스토리 실례3 → 최종 결론(아하 포인트)·적용

유진 라우리 (Lowry) — 이야기식 설교: - "이야기를 통한, 이야기와 같이, 이야기에 의한 설교의 틀" - 5가지 움직임:

단계 표현 내용
1 Oops 평형 깨뜨리기 — 기존 상식을 깨는 문제 제기
2 Ugh(악!) 모순 분석 — 왜 그런 문제가 발생하는지 설명
3 Aha 실마리 제시 — 본문 해석, 앞의 모순점에 대한 실마리
4 Whee(와아!) 복음 경험 — 핵심 메시지 전달, 의혹 풀림
5 Yeah 결과 기대 — 새로운 미래를 향한 개방적·간접적 결론

11가지 설교자 스타일 (피터 미드)

영국 교회 개척·설교자 양성 사역자 피터 미드 목사 — 설교자의 성격이 설교에 영향을 미친다.

설교 아웃라인의 목적

  1. 설교에 통일성을 주어 횡설수설하지 않게 한다
  2. 설교를 명쾌하게 해주어 흐름을 잘 따를 수 있게 한다
  3. 설교 각 부분 상호간의 관계를 한눈에 보게 한다
  4. 깊이 연구해야 할 부분과 그렇지 않은 부분을 구별하게 한다

아웃라인은 설교자에게 필요한 것 — 청중을 위한 것이 아니다. (스티븐 매튜슨)

효과적인 아웃라인 작성 비결 (매튜슨): - 개요를 청중에게 기억시키려 하지 마라 - 설교의 중심 주제를 시작이 아니라 끝으로 간주하라 → 귀납적으로 제시하라 - 개요의 요점들을 온전한 문장으로 제시하라 (모호함 방지) - 논리적 요점이 아니라 설교의 움직임 관점에서 구성하라

일반적인 설교 아웃라인 5단계: 제목 → 서론 → 대지1·2·3 → 결론

원칙 설명

설교 제목

"10시(텐어클락)"으로 2,000부 팔린 책이 제목만 바꾸니 9,000부 — 제목이 전체를 함축한다.

좋은 제목의 조건: - 설교자가 전하려는 핵심 내용을 담을 것 - 청중들의 호기심을 자극할 것 - 말씀을 기억하는 핵심 수단이 될 것 - 한 문장으로 기억하는 출발점이 될 것

좋은 제목 찾는 법: - 중심명제 + 설교의 목적에 적합해야 함 - 간결·참신·독창·한정적 - 현대적 감각, 기억하기 좋은 것 - 가능하면 동사형보다 명사형이 좋다

서론 작성법

서론은 설교의 마지막에 작성하는 것이 원칙

서론의 3가지 목적: 1. 청중들의 관심과 흥미를 이끌어냄 2. 설교 주제에 대한 관심을 불러일으킴 (전체 주제와 연결) 3. 설교의 목적이 무엇인지 제시

서론 작성 4단계: 1. 관심을 끌라 (설교 주제를 보여주는 예화로 시작) 2. 필요를 제기하라 (본문 배경과 문제 제기) 3. 주제를 제시하라 (설교의 목적 담기) 4. 목적을 선언하라 (대지로 넘어가는 연결)

결론 작성법 (데이비스 『Design for Preaching』)

  • 핵심 메시지가 선명하고 날카롭게 드러나야 함

튜닝의 개념

인생에 튜닝이 필요한 것처럼 설교도 튜닝이 필요하다. 온도·습도·시간의 변화, 의도적·비의도적 터치에 의해 음이 바뀐다.

두 가지 튜닝 방법: 1. 나 자신의 설교를 튜닝하기 (셀프 튜닝) 2. 다른 사람의 설교를 튜닝해봄으로써 자신의 설교 문제점 파악하기

셀프 설교 튜닝 체크리스트 (다니엘 오버도로프 응용)

출처: 다니엘오버도로프 《설교를 적용하기》 응용

  1. 성경의 가르침: 하나님께서 이 본문을 통해 본래 가르치신 것은 무엇인가?
  2. 본래의 목적: 하나님께서 이 본문을 통해 원독자들에게 어떤 영향을 미치려 하셨는가?
  3. 청중의 비교: 원독자들과 비교할 때 나의 청중은 어떠한가?
  4. 청중의 필요: 이 본문은 청중의 어떤 필요에 대해 말하고 있는가?
  5. 설교의 목적: 청중이 이 설교를 듣고 나면 그들의 생각·느낌·행동은 어떻게 달라질까? 믿음·삶·성품에 있어 기대하는 것은 무엇인가?
  6. 설교의 적용: 이 적용이 하나님을 높이는가? 그리스도를 드러내는가? 아니면 단순히 윤리적·도덕적 내용을 전하는가?

설교 준비 후 체크리스트

관련: 해돈로빈슨

내용 평가

중심명제와 목적 - 중심명제가 저자가 말하려는 의도와 내용을 포함하는가? - 중심명제가 본문 전체에 잘 드러나는가? - 중심명제가 기억하기 쉽고 대지 가운데 잘 녹아들었는가? - 설교의 목적이 청중들의 필요와 맞물리는가? - 설교의 목적이 중심명제에서 나왔는가? - 설교의 목적이 본문 전체에 흐르는가?

타인 설교 튜닝 실습 — 김덕수 출14 설교 평가

본문: 출애굽기 14:1-16 / 제목: "신의 한수" / 설교자: 김덕수 (백석대 실천신학교수) / 방송: 그 말씀 (2022.9)

설교 한 문장: "기적은 하나님 백성에게 구원이 되지만 동시에 대적들에게는 심판이 되는 신의 한수다."

주제 및 개요: 하나님께서 기적을 행하시는 목적은 백성에게 구원을 베풀고 대적을 심판하심으로 하나님께서 세상 가운데 영광을 얻으시고, 세상 사람들이 여호와임을 인정하게 만드는 것이다.

대지 내용
1) 기적을 보려면 잠잠히 하나님만 바라야 한다 (10-14절)
2) 기적을 경험하려면 하나님의 말씀에 순종해 전진해야 한다 (15-16절)
3) 기적을 행하심은 하나님의 영광을 세상이 보게 하려 함이다 (4, 17-18절)

평가: - "신의 한수"가 전체를 이끌어가지 않는다 → 제목이 반복·기억되게 해야 함 - 구원과 기적이라는 것보다 "신의 한수"를 계속 써야: 구원과 심판 = 신의 한수 - 어쩔 수 없는 상황에서 건너갔는데, 그것이 누군가에게는 구원, 누군가에게는 심판 - 신의 한수 = 하나님이 여시는 길(홍해) / 우리가 할 일 = 두려움 가운데 그냥 걸어가는 것

8주 수퍼비전: 눅24 설교 자기 평가

8주 토론에서 이상협이 직접 분석한 자기 설교 평가

설교 한 문장: "주님의 '대화 동행'을 부부 대화에 적용함으로 회복과 치유를 통해 인생의 (사명의) 길을 다시 걷게 한다."

핵심 평가: - 설교 제목과 본문의 내용은 연결됨 ✓ - 그러나 성경저자의 목적과 설교자의 목적이 다르게 보인다 ✗ - 본문의 목적: 성경(구약) 전체를 통해 두 제자의 삶에서 선교의 사명을 발견하게 하고, 스스로 되돌아가도록 하는 것 - 설교자의 목적: "대화"라는 수단을 통해 사명을 발견하도록 돕는다는 것 - "대화 동행"이라는 이미지로 전체 맥락을 잡는 것은 유용하나, 본문의 중심명제보다 청중의 필요를 중심으로 주제와 해석이 작용 - 결론: 청중중심·청중의 필요를 중심으로 이루어진 강해설교가 아닌 주제설교로 봐야 함
