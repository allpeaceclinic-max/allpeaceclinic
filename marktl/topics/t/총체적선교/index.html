총체적 선교 (Holistic/Integral Mission)

선교신학 참조 (미씨오데이·로잔 기초 내용). 이 페이지는 함선네 강의를 중심으로 한 선교 패러다임 변화와 총체적 선교의 실천적 함의에 집중한다.

선교 패러다임의 역사적 변화

랄프 윈터 — 선교의 400년 단위 시대 구분

시대 기점 특징
해안선 선교 윌리엄 캐리 (1792~) 서구→ 나머지, 해안선 도시 중심
내지 선교 허드슨 테일러 해안선 넘어 내지로 (2차 산업혁명 인프라 활용)
미전도종족 선교 랄프 윈터 (1974~) 종족·언어 단위 복음화
문화적/숨겨진 영역 현재 지리적→ 종족적+언어적→ 문화적 영역

4차 산업혁명과 선교: 자율주행·AI·플랫폼 시대 → 초연결, 메타버스·SNS = 새로운 신대륙

숨겨진 영역 (Hidden Frontier)

랄프 윈터 개념: 암(癌)처럼 눈에 보이지 않는 영역 — "당신의 암을 낭비하지 마세요" (존 파이퍼) - 총체적 접근 없이는 발견 어려움 - 개인의 관심과 달란트가 진입점

선교 지형 변화 (글로벌 흐름)

흐름 대표자 내용
서구→나머지 윌리엄 캐리 1차 개신교 선교 물결
모라토리엄 선언 존 가투(케냐, 1971) "미셔너리 고 홈" — 서구선교의 종언
모든 곳에서 모든 곳으로 사무엘 에스코바 비서구 교회의 선교 참여
지역중심화 제이 마탕가·폴 벤더 사무엘 도시선교·세계화 (코비드로 가속)

코비드19는 '파송으로서의 선교(Mission-as-sending)' 패러다임에 근본 도전

총체적 선교의 정의

로잔 운동 정의: 온전한 복음 + 온 교회 + 온 세상 - 영혼 구원만이 아닌 창조세계 전체의 회복 - 사회 개발·의료·교육·정의 등 전 분야가 선교 영역

WCC vs 로잔 차이: - WCC: 선교 = 인간화, 사회 변혁 중심 - 로잔: 복음전도 우선성 + 사회적 책임 통합 (긴장 속 유지)

선교적 삶: Being vs Doing

구분 내용
Being (존재적 선교) 하나님의 하나님 되심이 우리를 통해 흘러가는 것
Doing (기능적 선교) 가시적·빠른·경쟁 중심의 결과물 생산

"존재적 선교를 기억해야 한다. 우리는 없고 하나님만 존재하게 되는 것이 선교다." "하나님의 선교를 하려면 존재적 준비에 8~90% 에너지를 써야 한다."

단기선교 오류: 존재적 준비에 시간 쓰지 않고 동선·율동·물품 구매에 에너지 집중

선교사 역할 변화

과거 4P 현재 방향
Pioneer (개척) 퍼실리테이터
Parent (부모) 촉매자
Partnership (협력) 게스트·조력자
Participant (참여) 현지인이 액팅하도록 지원

동원에서 참여로 (정민영·송기태)

  • 동원(Mobilization): 군사적 발상, 상명하달
  • 참여(Participation): 하나님의 선교에, 교회에 위임된 선교에 주체적으로 동참

"그들의 선교도 아닌 나의 선교도 아닌 우리의 선교" (브루스 캠프)

Unlearn → Relearn (토마스 쿤)

앨빈 토플러:

"Evolution attitude: learn, unlearn, relearn. 변화를 주도하는 유일한 방법은 새로운 지식을 얻는 것. 새로운 지식을 얻는 유일한 방법은 다시 배우는 것."

  • 패러다임 쉬프트: 배운 것을 비우지 않으면 새로운 것을 배울 수 없다
  • 코로나가 스탑 기회였으나 — 다시 똑같은 패턴으로 돌아가려는 경향
  • 언런 → 리런: 하나님의 선교, 미씨오 데이로부터 다시 시작

로잔4 서울 2024 — 핵심 주제

주제 내용
대위임령 재해석 "제자 삼으라"(마테튜오) — 1%사역자→99%에게 위임
협력 개인→공동체, 단체→교회, 서구→글로벌 사우스
GenZ 선교 진정성·투명성·개별화·지역화·관계적 방식 요구
공공 이슈 젠더·정의·창조세계·이슬람·난민

"왜 성과가 적을까? 위임을 하지 않았다. I don't need you했다. 1%사역자 → 99%에게 위임." (마이클 오, 로잔 총재)

학습공동체 → 전략공동체

선교 생태계(Missional Ecology) 변화 속에서: - CPM(교회개척운동) + T4T(훈련자 훈련) 전략과 함께 협력 - 당분간 학습공동체로 → 변화에 대응 후 전략공동체로

21세기 선교 5가지 방향

  1. 도시화: 하루 14만명 도시 유입, 2050년 72% 도시 거주
  2. 이주민 선교: 선교지가 우리에게 왔다 — 다문화·다민족
  3. SNS 플랫폼: 40억명, 하루 평균 2시간 29분 접속 — 새로운 신대륙
  4. MZ세대: 스낵컬쳐·유튜브·멀티 — 전혀 다른 선교 방식 필요
  5. 탈육신 시대 대응: 역사성·공공성 회복 (마이클 프로스트)

"그들의 언어로" — 상징 빼앗기

"선교해야 합니다" → 갭 느낌(선교사가 하는 것) / "당신 가정의 미션이 있나?" → 즉시 관심 생김

  • 상징 빼앗기: 그들에게 익숙한 언어(미션·비전·임팩트)로 선교를 재표현하는 전략
  • 사례 1: "당신 가정의 미션" — 선교라는 단어 없이 미션을 얘기했더니 관심을 가짐
  • 사례 2: "생명사역 국제화" = 선교 (동신교회) → 교인 입에서 쉽게 나오고 선교헌금 1억 증가
  • 행 11:20: 흩어진 사람들이 안디옥의 헬라인들에게 그들의 언어로 주 예수를 전함
  • 원리: 그들의 언어, 그들에게 중요한 단어로 들어가야 관심이 생긴다

읽기의 균형 — Text vs Context

구분 내용 위험
Reading text (성경읽기) 말씀을 깊이 연구 환경 무관심 → 사회적 영향력 없음
Reading context (환경읽기) 시대와 사회를 관찰 인본주의화, 말씀 없는 행동주의

"선교는 발명이 아닌 발견이다 — 믿음이 들음에서 나듯, 선교도 들음에서 난다"

  • 균형: 성경읽기 + 환경읽기가 함께 되어야 선교적 방향을 발견할 수 있다

실행역량 vs 개념설계역량 (『축적의 길』)

  • 실행역량: 설계도를 따라 빠르고 정확하게 만드는 능력 → 고도성장기·안정적 환경에 최적
  • 개념설계역량: 원리와 환경을 읽어 설계도 자체를 만드는 능력 → 변화의 시대에 필수
  • 환경이 급변할 때 실행역량만 있는 사람의 두 반응: 1. 어디로 가야 할지 몰라 멈춤 2. 주변과 상관없이 하던 것을 고집스럽게 반복
  • 선교 적용: 미전도종족 시대까지는 "어떻게든 안 가는 곳에 가는" 실행역량으로 충분했으나, 현재는 말씀의 원리와 환경을 읽고 방향을 재설정하는 개념설계역량이 필요

하백선 시리즈 — 의료인 총체적 선교 적용

→ 상세: 하백선시리즈

선택의 논리 (4장 — Healing for the Nations) - 아브라함 선택 = 열방이 복을 받기 위한 선택 (창 18:18) - 복을 받았다 = 사명을 받았다 — 분리 불가 - 의료인의 전문성 = 신탁 자산 → 열방을 위한 수로가 되어야 함

출애굽 구속의 4중 총체성 (6장 — Redemptive Living) - 정치적 + 사회적 + 경제적 + 종교적 해방 - "출애굽 형태의 구속은 출애굽 형태의 선교를 요구한다" - 의료 선교 = 총체적 해방의 현장

창조세계 돌봄 (3장 — Missional Creation Care) - 의료폐기물·기후건강 = 선교적 책임 - 골 1:20 화목의 범위 = 창조세계 전체 - 의료인의 생태적 책임 = 총체적 선교의 한 차원

복음의 우주적 범위 (11장 — Gospel Identity) - 복음 = News(왕의 즉위 선포), not Advice - 골 1:15-20 만물의 화해 = 개인 영혼 + 사회 + 생태 전체 - 의료선교 = 우주적 복음의 부분적 현장 실현
