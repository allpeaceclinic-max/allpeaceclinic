창조세계 돌봄 — 선교의 통합적 지평

"창조된 인간으로서 우리의 사명은 하나님이 창조하신 지구를 돌보는 것이다"

창조명령 — 원어 분석

창 2:15 — 에덴 동산을 "경작하며 지키게" 하셨다

히브리어 뜻 함의
카바스 (정복) 땅 위에 왕권 행사 통치권
라다 (다스리다) 왕권 위임 청지기
아바드 (섬기다·경작하다) 땅을 섬김 봉사
샤마르 (지키다·보호) 보존·보호 수호

→ 창조명령의 본질: 자신의 유익이 아닌 땅의 유익을 위한 섬김과 보호

아다마(땅) → 아담: 동일한 재료를 공유 — 인간은 땅과 결속된 존재

창조세계의 신학적 지위

  • 창조는 구원의 배경이 아니다: 성경은 창조(창1:1)로 시작해서 새창조(계21:1)로 끝남
  • 구속사(타락→십자가)만이 아닌 창조→타락→구속→새창조의 전체 스토리
  • 골 1:20 — 그리스도의 십자가는 전체 창조세계를 위한 좋은 소식
  • 롬 8:19 — 피조물이 기다리는 것도 하나님의 아들들의 나타남
  • 하나님의 창조 대상 = 구속(회복)의 대상 → 동일하다

코이노니아(koinonia): 창조세계와의 코이노니아 — 단순한 환경보호가 아닌 하나님의 선교에 함께 참여하는 것

인카네이션 모델 — 30년+3년

예수님의 선교 방식: - 30년간 나사렛에 뿌리를 내리신 후 → 3년 공생애 - 우리는 3년 훈련 후 30년 파송 — 예수님은 그 반대 - 특정 장소·문화·환경에 뿌리내림 = 선교의 전제

렘 29:7 — "바벨론 성읍의 샬롬을 구하라" → 내가 보내진 그 자리의 샬롬이 나의 선교지

총체적 선교 — 존 스토트 vs 크리스토퍼 라이트

존 스토트 (로잔선언, 1974) - 전도만의 선교(보수) ↔ 샬롬 성취의 선교(에큐메니컬) 양 극단 비판 - 전도와 사회적 책임의 동반자적 역할 — 대위임령 + 대계명 - 단, 전도의 우선성(priority)을 강조

크리스토퍼 라이트 (40년 후 재해석) - 선교 = 대위임령 같은 단일 행동이 아닌 성경 전체 내러티브 속의 사건 - 전도와 사회책임이 동반자 관계가 아닌 통합적 선교 (복음 중심으로 통합) - 복음 = 개인 + 사회 + 창조세계에 대한 하나님의 좋은 소식

로잔 선언: "온 교회가 온전한 복음을 온 세계에" - 온전한 복음 = 창조에서 새창조까지의 전체 스토리 - 온 세계 = 영혼만이 아닌 피조세계 전체

보내심(Missio Dei) — 어원과 신학

언어 단어 의미
히브리어 샬롸흐 / 실로아흐 보내다
헬라어 아포스톨로스 사도 = 보냄받은 자
라틴어 미씨오 하나님의 보내심
한자 선교(宣敎) 전파·가르침 — 보내심보다 목적 중심 번역

요 20:21 — "아버지께서 나를 보내신 것 같이 나도 너희를 보내노라"

이해 안 되는 보내심도 있다: 바벨론 포로도 하나님의 보내심(렘29) — 성도들이 짤리거나 이동하게 되는 것도 하나님의 방식일 수 있다

선교 패러다임의 역사적 변화

시대 대표 특징
지리적 선교 윌리엄 캐리 서구→비서구, 자치·자전·자립 교회
종족·언어 선교 존 가투 (1971, 모라토리엄) 서구 선교 종식 요청
문화적 선교 사무엘 에스코바 모든 곳에서 모든 사람에게
지역중심 선교 마탕가·벤더 사무엘 도시화·세계화 시대

코로나19가 이 전환을 가속화 — 선교사가 추방되는 시대, 선교지가 우리에게 왔다(다문화·다민족)

선교사의 역할 변화: 행동가 → 퍼실리테이터 / 창시자 → 촉매 / 영웅 → 조력자 / 선교사 → 파트너

숨겨진 영역 (Hidden Peoples)

  • 지리적→종족적→문화적→영역(domain) 중심으로 선교 지평 확대
  • 암환자 (랄프 윈터): 암환자는 숨겨진 영역 — 요한 파이퍼 『당신의 암을 낭비하지 마세요』
  • 지금은 지역·분야 전문가·선교 전문가의 총체적 협력 없이는 숨겨진 영역 발견 어려움
  • 개인의 달란트·관심사가 숨겨진 영역의 지도를 그린다

쯔다카 — 창조세계 돌봄의 경제학

레 19장 밭 모퉁이 원리 (페아트 싸데): - 수확의 1/40~1/16 (2.5~6.25%)을 남겨두어 가난한 자가 이삭 줍게 함 - "나의 수입에 남의 몫이 포함되어 있다" = 성경적 정의(Justice) - 적선이 아닌 자립의 기회 (보아스와 룻의 관계) - 의료 적용 — 의원을 찾아오지 못하는 환자를 위한 5% 운동 (시간·재정·봉사)

→ 기독의료인정체성 — 쯔다카 의료 5% 운동 상세

오이코스(Oikos) — 어원의 통일성

단어 어원 의미
에콜로지 (Ecology) 오이코스 + 로고스 집(지구)에서 일어나는 관계의 과학
경제 (Economy) 오이코스 + 노모스 집 안 자원의 사용과 관리
교회 (Ekklesia) 에크 + 칼레오 집에서 불러낸 자들

→ 신앙·경제·생태는 같은 뿌리 — 통전적 삶(Integral Life)

창조적 직업관

전 3:22 — "사람이 자기 일에 즐거워하는 것보다 더 나은 것이 없음"

  • 하나님은 크리에이터이심 → 창조하신 모든 것이 창조적
  • 성령님이 임하면 창조적이 된다 — 그리스도인은 창조적으로, 재밌게 일해야 한다
  • 내 달란트 분야에서 즐겁게 일하는 것 = 하나님처럼 일하는 것

선교적 직업관 (미션스, Missions): - 창조명령(대사명) = 선교(대위임령) — 본래 하나다 - 선교적 직업 = 직업적 거듭남 (삭개오)이 있어야 찐 선교 - 바울 혼자의 선교가 아닌 바울과 동역자 36명의 선교 (롬16) → 모든 사람의 선교 시대

실천 제안

  • 개인: 텀블러, 플라스틱 감소, 걷기, 전기차 고려 — 작은 의도성들
  • 직업: 창조적으로·즐겁게 일하는가?
  • 의료: 5%의 시간·돈·봉사를 사각지대 환자에게
  • 공동체: 숨겨진 영역을 발견하기 위한 달란트·관심사 나눔

영혼구원과 창조세계 돌봄은 이분법이 아닌 통합적 선교의 두 날개다

대위임령현황보고서 — 창조세계 돌봄 데이터 (2024 L4)

대위임령현황보고서는 창조세계 돌봄의 현황을 다음 데이터로 제시한다.

환경 성과 지수 (EPI)

지역 EPI 점수
서유럽 62.5
남아시아 31.5
사하라이남 아프리카 낮음 (남아시아와 유사)
  • 환경 취약성과 기독교 인구 성장 지역이 상당 부분 겹침 → 창조세계 돌봄이 선교적 과업

기후변화 인식

  • 기독교인의 72%가 기후변화를 인류 최대 위협으로 인식
  • 그러나 교회의 실천적 대응은 인식 수준에 크게 못 미침

글로벌 부채

  • 전 세계 부채 = GDP의 350% — 구조적 빈곤과 창조세계 파괴의 경제적 뿌리
  • 외채의 짐: 개발도상국 정부가 국민 복지·환경 보호보다 채무 상환을 우선할 수밖에 없는 구조

루스 파딜라 데볼스트(L4 서울): "기후 문제는 지속가능성이 아닌 정의 문제로 봐야 한다 — 하나님·인간·피조물의 관계 회복이 선교"
