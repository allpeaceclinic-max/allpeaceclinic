성경적 인간 이해

문제 상황에서 시작하기 — 두 가지 케이스

상황 A (의대생 PK): 섬기는 마음으로 발표를 자원했다가, 아무도 나서지 않는 상황이 반복되자 분노가 쌓임 상황 B (약속): 친구들 모임에 아내가 반복적으로 늦자 분노가 치밀고 과거의 서운함이 연달아 올라옴

→ 공통 질문: 무엇이 문제이고, 해답은 무엇인가? 왜 반복되는가?

1. 지식 습득의 4가지 경로

경로 설명 한계
직관 주관적 확신이 들때까지 움직이지 않음 주관적
이성 논리와 사고 화합 불가한 견해차 존재
경험 특히 어린 시절 경험이 이미지로 남음 어린 시절 이미지 → 편견 → 잘못된 신념 → 옛 사람 형성 → 재해석 필요
계시 성경 + 자연 계시 양쪽 모두 오류 가능성 존재

"해답이 무엇인지 모르지만, 성경에 있다는 것만은 확실하다."

  • 자연 계시 목적: 하나님의 존재 인정
  • 성경 목적: 생명 얻는 길 + 창조주가 주려 하셨던 삶을 살도록

2. 세상의 문제 해결 모델 3가지

모델 핵심 전제 접근 방식 한계
역동 모델 내면의 무의식적 과정이 행동을 통제, 뿌리는 과거·어린 시절 "근원의 재구성" — 숨겨진 뿌리 찾아 치료 행동 변화 거의 안 다룸, 내담자 의존·전이
도덕 모델 혼란이 아닌 고집 — 행동 수정이 답 지시적 충고 (제이 아담스) 동기를 거의 살피지 않음
관계 모델 불만족한 관계에서 문제 발생 — 고독→상처→방어→고독 비지시적, 미러링(로저스), 관계 변화를 도구로 행동·동기 모두 과소평가

→ 성경적 모델: 역동처럼 내면을 고찰 + 도덕 모델처럼 행동 변화를 지향 + 관계 모델처럼 사랑의 관점으로 접근

3. 이마고 데이 — 하나님의 형상 4가지 특질

"원본을 복제했다 — 하나님은 전능성·자존성은 주시지 않으셨지만, 그분의 인격적 특질들은 그대로 주셨다." (제임스 패커, 『하나님을 아는 지식』)

하나님이 갖고 있고 인간도 갖고 있는 4가지:

특질 하나님 인간
깊은 갈망 우리가 기쁨을 드리기를 갈망 내면 깊은 곳의 만족을 향한 갈망
사고·평가 세계를 생각하고 평가하심 (노아 시대) 사고하는 존재
의지·선택 목표를 정하고 행동을 선택하심 목표와 행동을 선택할 수 있음
감정 나사로의 죽음(슬픔), 성전 장사꾼(분노), 사역(기쁨) 감정을 의식 속에서 경험

감정(불쾌감·기쁨 모두)은 하나님의 신호다 — 억압이 아닌 해석과 반응이 필요

4. 성경적 변화 모델 — 혼돈을 의뢰의 기회로

문제·혼돈 발생 ↓ 갈망을 점검 (숨은 동기 찾기) ↓ 회개 (메타노이아) ↓ 의뢰·기도 ↓ 섬김으로 반응

상황 B 적용: - 인격적 필요 충족(제시간 도착) → 실망 → 숨은 동기(아내가 약속을 잘 지키는 사람이 되길 바람) → 회개 → 섬김("혼자 준비하느라 고생했겠네") → 마음이 풀림 → 즐거운 여행

  • 어린 시절의 경험은 이미지로 남아 편견과 잘못된 신념을 형성 → 반드시 재해석의 과정이 필요 (윤명숙)

5. 핵심 명제

"인생의 길을 안다는 것은 청사진을 아는 것이 아니다. 하나님을 알고 나를 아는 것이다. 그럴 때 가는 길은 몰라도 상관없다."

  • 변화의 목표: 문제 해결이 아닌 변화·성장·성숙·성화
  • 해답은 성경에 있다 — 그러나 양쪽(성경·과학) 모두 오류 가능성을 인정하는 겸손함 필요
