AP Now (적극적 부모역할 프로그램)

개요

AP Now (Active Parenting Now)는 마이클 펍킨(Michael Popkin)이 개발하고, 한국에서는 홍경자 교수가 도입한 부모교육 프로그램이다.

강의 버전: - 윤명숙 전도사님 — 2025년 동신교회 AP Now 1차 시리즈 (청소년 자녀 부모 대상) - 이상협 의사님 — 교회 AP Now 강사로 직접 강의 진행

핵심 철학: "완벽한 부모가 아니라, Good Enough 부모를 목표로 한다. 방향과 목적을 가지고 일관성 있게 양육하는 것이 핵심이다."

1강 — 적극적인 부모역할 (Active Parenting)

적극적 부모 vs. 반응적 부모

유형 특징
적극적 부모 자녀교육에 대한 목표를 분명히 하고, 일관성 있고 효율적으로 자녀를 지도한다
반응적 부모 자녀가 요구하는 대로 움직임. 치근댐을 한없이 참다가 소리 지르며 폭발. 성적이 안 나오면 학원을 바꾸는 "널뛰기 부모"

부모 역할의 3가지 유형

1) 전제형 (독재자)

  • 지도법: 자녀에게 자유는 없고 제약만 강요
  • 지도방법: 명령, 보상과 처벌 — "숙제해! 알았지?", 학원도 엄마가 결정
  • 효과: 의사표현 불가, 기가 죽거나 포기, 청소년기 반항
  • 초등학교까지는 "착한 아이"로 보이지만, 청소년기에 소극적·적극적 반항으로 전환
  • 보상은 곧 권리가 되어버림. 처벌은 분노를 쌓고, 보복심을 키움
  • 골 3:21 "아비들아 너희 자녀를 노엽게 하지 말지니 낙심할까 함이라"

"마음을 상실시켜 놓고 스스로 공부하라 한다 — 자기가 다 상실시켜 놓고."

2) 자유방임형 (심부름꾼)

  • 지도법: 제약 없이 자유만 허용
  • 지도방법: 무제한의 자유 — 스스로 사랑이 많다고 착각
  • 효과: 안정감과 소속감 결여, 자신감 결여, 책임감을 배우지 못함
  • 규칙이 있어야 안정감을 느낀다. 무제한의 자유 = 높은 자존감이라는 착각
  • 아이가 왕이 되고, 가정질서가 무너짐

"아이에게 잘하고자 하는 것은 본능이다. 이것을 질서를 위해 다스려야 한다."

3) 민주형 (적극적 부모)

  • 지도법: 일정한 한계 안에서 자유가 주어짐 ("한계 안에서의 자유")
  • 비유: 바닷가 수영 — 빨간 깃발 안에서 마음껏 놀기
  • 지도방법: 자녀의 권리·의무·협동심을 가르침
  • 효과: 부모에게 자신의 생각과 감정을 표현, 책임감이 커지면 자유도 확대
  • 독재형의 권위 + 방임형에서 자녀가 생각/감정을 표현할 수 있는 점의 결합
  • 결혼시킬 때 동그라미를 완전히 내려놔야 성인으로 키운 것

순종교육의 두 단계

시기 방식
0~13세 지배의 시기 — 부모가 규칙을 정하고 가르침
청소년기 가르침의 시기 — 자발적 순종으로 전환, 이유를 묻기 시작함. 부모는 대답할 준비가 되어 있어야 한다

"청소년이 '교회 왜 가냐' 물을 때 '덮어놓고 믿어라'는 대답은 준비 안 된 것이다. '오늘부터 같이 연구해보자'만 해도 된다. 자발적인 하나님에 대한 순종이 나와야 하는 시기가 청소년기다."

창조명령과 부모의 목적 (창 2장)

  • 생육·번성·충만: 보호하고 섬기는 것
  • 무한책임이 아닌 유한책임 — 부모를 떠나 하나님의 권위 아래 들어가게 하는 것
  • 엡 6:4 "또 아비들아 너희 자녀를 노엽게 하지 말고 오직 주의 교훈과 훈계로 양육하라"

예수님의 부모 유형

  • 요셉: 의로운 사람 (마 1:19), 마리아: 주의 여종 (눅 1:38) — 둘 다 경건한 사람
  • 유대 교육 4단계: 강보교육(말씀 두루마리), 할례(소유권·내려놓음), 정결예식, 성인식
  • 눅 2:40→52: 신체→지혜→지력→사회성 순으로 성장

자녀의 건강 4영역: | 영역 | 성경 | 핵심 | |---|---|---| | 신체 | 고전 10:31 | 욕심·경쟁심으로 먹지 않아야 | | 마음 | 잠 4:23 | 혼자 먹으려는 마음 지켜줘야 | | 지적 | 잠 1:9, 9:10 | 지식은 쌓는 것이 아니라 심는 것, 경외로부터 | | 영적 | 요 14:6, 4:23 | 집에 있는 아이 축복, 영과 진리로 예배 |

신념과 지혜 (폴트립)

  • 신념: 관계가 탄탄할 때 지켜진다. 야동 보자는 친구 앞에서 버틸 수 있었던 것은 부모와의 관계 때문
  • 4-10계명을 어길 때는 반드시 1-3계명을 먼저 어기게 되어 있다 → 수직관계가 수평관계에 영향
  • 지혜: 성령님의 도우심. 아버지가 하나님께 의뢰하는 것을 자녀가 그림자로 배운다

2강 — 적극적인 의사소통

공동선 (Common Good)

개념 내용
공익(공공성) 너와 우리가 좋은 것 — 99마리를 위한 2자 관계
공동선 너, 우리 뿐 아니라 하나님이 좋아하시는 것 — 잃어버린 한 마리도 포함하는 3자 관계

협동심 = 공동선을 추구하도록 만드는 것. 협동심은 태도와 기술의 산물.

문제는 교육의 기회

  • 보통 문제가 생기면: ① 짜증 ② 배우자 탓 ③ 자녀 원망 ④ 환경 탓
  • AP Now의 관점: 문제 발생 = 최상의 교육적 기회

문제 소유권

  • 부모 소유: 훈육 사용
  • 자녀 소유: 지지, 선택의 자유를 줘야 (잘못된 선택을 알아도 허용해야)
  • 공동소유: 훈육 + 지지, 가족회의로
  • 격려는 언제나

적극적 의사소통 5단계

  1. 적극적 경청 — 가장 어렵다 (배우자에게도 잘 안 됨)
  2. 감정에 귀를 기울임 — 라벨링, 읽어줌
  3. 감정과 이야기 내용 연결 — "~처럼 들리는데, 내가 추측하건대…"
  4. 대안을 찾고 결과 평가 — 자녀가 효율적 문제해결을 하도록 코칭
  5. 추후지도 — 미해결 시 다시 적극적 의사소통

"비강요적인 대안, 자신의 경험을 들려주면서. 실패해도 괜찮아."

비언어 소통의 비중

  • 신체언어 55% | 음조 38% | 말 7%
  • 혼합 메시지의 위험: "핸드폰 그만해라"하면서 자신은 계속 → 하나가 되어야

감정 읽기

  • 엡 4:26 "분을 내어도 죄를 짓지 말라" — 감정은 허용, 행동이 문제
  • 기노트의 정서적 거울: 내용보다 감정 먼저 반영
  • 게임중독 아이의 사례 — 원인은 6개월간 혼자 점심을 먹은 것. 감정을 읽었더라면

3강 — 효과적인 훈육과 문제다루기

왜 책임감이 없을까?

  • 과잉보호, 과잉공급, 구원자 역할, 너무 오랜 부모 노릇
  • 아담과 하와의 후손 — 두려움 때문에 책임을 전가
  • 비난은 책임감을 기르지 못한다 → 비난 → 책임전가·변명·자기합리화

책임감 기르는 법: 선택 → 결과 → 받아들이기

보상 vs. 처벌 vs. 인센티브

구분 내용
처벌 단기 효과처럼 보이나 앙갚음·분노 유발. "상처를 입힐 권리"를 허용하는 것
보상 매수. 자녀가 곧 권리로 여기게 됨
인센티브 논리적으로 연결된 것 (숙제 빨리 하면 놀 시간 증가)

행동 메커니즘

사건 → 생각(의식/무의식) → 감정 → 행동 → 사건에 영향

  • 처음은 의식적, 반복되면 무의식적 반응 — 그래서 처음이 중요
  • 감정은 사고와 가치관의 산물. 생각을 바꾸면 감정도 바뀐다

훈육 도구 (기본 3가지)

  1. 관계 형성: 화나면 타임아웃 후 훈육 — 화만 전달되면 마음이 전달되지 않음
  2. 나 전달법 (I-message): "단호하지만 부드럽게", 말수가 적을수록 좋다
  3. 논리적 결과: 선택권 제시 → 결과 경험 (예: 음식 장난치면 "먹지 말고 나가라")

"아이가 엉뚱한 행동으로 관심 끌기 → 무관심. 보상을 주지 않으면 사라진다. 단 다른 방식으로 전환될 수 있으니 긍정적 방식으로 대체해줘야."

4강 — 자녀의 그릇된 행동 이해하고 교정하기

행동은 목적으로 이해하라 (드라이쿨스 이론)

"왜 이렇게 했니?" (요인 탐색) → "뭘 하려고 했니?" (목적 탐색)

자녀는 유전에도, 환경에도 핑계 댈 수 없다 — 자기결정적 존재

4가지 그릇된 목적

목적 자녀 행동 부모의 실수 대응
부당한 관심 끌기 엉뚱한 행동, 떼쓰기 주의·잔소리·달래기 (관심=보상) 무관심, 긍정적 관심으로 대체
반항 (힘겨루기) "내가 할 거야", 말 안 듣기 통제하려 함, 독재적 경향 대화·선택권·실수 허용, 싸움 피함
앙갚음 고의적으로 상처 주려 함 보복하면 기 꺾임 악순환 끊기, 휴전선언, 차분·단호
극도의 기 꺾임 무시, 무감동, 비행에 빠짐 강압·완벽주의·"아직 멀었어" 무조건적 수용 경험 필요

예수님의 딴청: 요 8:6, 8 — 바리새인들의 힘겨루기에 몸을 굽혀 땅에 쓰심. 말려들지 않는 모범.

기질과 달란트

  • 히포크라테스: 다혈·담즙·우울·점액 기질 (기질과 자녀양육 — 베블리 라헤이)
  • 다중지능이론 (하워드 가드너): 언어지능, 논리수학지능 외 다양한 지능
  • 아들러의 출생순위: 5살 이상 차이나면 별개의 두 가정과 같다

분노 다루기

  • 분노는 fight or flight — 생존을 위한 것. 이성을 잃지 말고 이성을 사용하라
  • "화를 내는 순간 → 힘겨루기로 들어간다. 졌다."
  • 타임아웃: 아이도, 부모도

5강 — 용기, 품성, 자기존중감

세 가지 핵심 자질

자질 어원/출처 내용
용기 (Courage) 불어 꾀흐(심장) 두려움이 있음에도 불구하고 try하는 것
품성 (Character) — 가치관·신념·태도·행동의 총체. 옳은 일을 하려는 내면의 힘
자기존중감 머리(Think, Know) "나는 능력 있고 사랑스러운 인간으로서 성공하게 될 것"이라는 마음

잠 4:23 "모든 지킬 만한 것 중에 더욱 네 마음(레브)을 지키라" — 히브리어 레브(심장)

성공회로 vs. 실패회로

  • 성공회로: 격려 → 용기 → try → 성장
  • 실패회로: 비난 → 기 꺾임 → 낙심 → 그릇된 행동

골 3:21 "낙심할까 함이라" — 기 꺾인 아이는 아무것도 안 하거나 비행한다

기 꺾는 4가지

  1. 부정적 기대 ("니 그럴 줄 알았다")
  2. 실수에만 집중 ("또 40점이야, 네가 잘 하는 게 뭐 있어")
  3. 완벽주의와 과잉기대 (도달할 수 없는 기준)
  4. 과잉보호 ("대신해주는 것은 무시하는 것이다")

격려의 힘

  • 인커리지(encourage) = 용기를 주입하는 것
  • 격려는 자녀에게 용기·자존감·품성과 재능을 길러주는 비료
  • 실패해도 격려, try하는 아이가 결국 얻어 걸린다
  • 칭찬할 만한 짓을 해야 칭찬하는가? → 긍정적 행동을 하는 순간을 포착해서 격려

"내가 할 거야!" — 기회를 놓치지 말 것. 엄마가 내가 저 일을 할 수 있다고 생각하시는구나 라고 인식.

허드렛일의 중요성

  • 선택·위임 = 하나님이 일하시는 방법
  • 창조세계를 인간에게 다스리도록 위임 — 왕 같은 역할, 종된 섬김
  • 선택권 → 자존감 연결: 운동화 선택, 라면 끓이기… 작은 일에 충성

6강 — 적극적인 가정 NOW

민주주의와 가정

민주주의 가정 적용
평등 존중
자유 한계 안에서 책임
투표권 자녀의 의견
발언권 자녀가 이야기할 수 있는 분위기
제한 부모가 지도자

가족회의

목적: 협동심·책임감·용기·자기존중감 → 신뢰/유대감 → 정서적 연결

나이 시간
7세 이상 참가 가능 —
5~7세 약 10~15분
8세 이상 약 30~45분

운영 원칙: 안건은 누구나 포스트잇으로 제의, 사회자는 순번으로, 서기 역할이 논술·필기 능력 키움

3가지 기능: 1. 비형식적 모임 — 일상의 나눔 2. 문제 예방적 — 잠재적 문제 상황 미리 다루기 (백화점, 물놀이, 친척모임, 학교폭력 등) 3. 문제 해결적 — 문제 정의 → 감정 나눔 → 브레인스토밍 → 해결책 → 논리적 결과 → 추후지도

  • 자녀가 설교? → 발상의 전환
  • 구성: 찬양(자녀 기준) → 인터뷰 or 자녀가 가르쳐주기 → 질문으로 적용 → 축복기도(아버지) → 재미있는 놀이/간식

"가정예배를 오래 한 가정도 일방통행이면 실패. 며느리에게 교회 가는 것이 '시아버지가 일방적으로 비난·지적하는 시댁 가는 것'이 되면 안 된다."

하브루타

  • 짝을 지어 질문하고 대화·토론·논쟁하는 유대 교육법
  • 목표: 제자화, 정체성, 시대에 맞는 아이

6강 종합: 성공회로 안에 머물게 하기

  • 부모의 변화 → 자녀의 변화
  • 관계 우선 → 나중에 스스로 변화, 진정한 변화
  • 자신을 먼저 돌보는 것 (부부 에너지 유지)
  • 방치 × 과잉 개입 × → 적절한 개입 (위험·약물·안전은 반드시 개입)
