CMF의 핵심: VPM 철학

Value > People > Ministry
가치가 사람보다 먼저, 사람이 사역보다 먼저다.

개념 정의

VPM이란?

📌 Value (가치) - 왜 우리가 존재하는가? 📌 People (사람) - 누가 그 가치를 실행하는가? 📌 Ministry (사역) - 그 가치로 무엇을 하는가?

순서가 중요하다. - Value 없이는 People이 흩어진다 - People 없이는 Ministry가 무의미하다 - Ministry 없이는 Value가 추상적이다

1. Value (가치가 우선)

왜 가치가 먼저인가?

역사적 사례

순교자들 ``` 기독 순교자 → 신앙이라는 가치에 생명을 걸다 불신자 순교자 → 자유/정의라는 가치에 생명을 걸다

공통점: 가치가 생명보다 우선 ```

나라를 지킨 사람들 ``` 전쟁에서 죽는 군인 → 국가와 자유라는 가치 독재에 저항한 사람들 → 민주주의라는 가치

= 가치에 목숨을 거는 것 ```

성경 근거:

롬14:8 - 주를 위한 삶과 죽음

"우리가 살아도 주를 위하여 살고 죽어도 주를 위하여 죽나니"

행5:29 - 사도들의 결정

"사람보다 하나님께 순종하는 것이 마땅하니라"

막8:34-37 - 영혼의 가치

"온 천하를 얻고도 영혼을 잃으면 무엇이 유익하리요"

CMF 맥락: 가치 없으면 무너진다

``` 예시 1: 사람 때문에만 교회 다니다가 그 사람이 나가니 → 다 나간다

예시 2: CMF에서 어떤 사람이 싫어서 떠나다가 → 공동체의 진정한 가치를 놓친다

예시 3: 돈 때문에 의료인이 되었다가 → 언젠가 공허함에 빠진다 ```

결론: 가치 없이 사람과 사역에만 집중하면
→ 우리는 계속 상처 받고 흔들린다

CMF의 핵심가치는?

5가지 핵심가치 참조 1. 복음을 증거한다 2. 하나님의 사람을 세운다 3. 공동체로 살아간다 4. 약한자를 섬긴다 5. 생명을 존중한다

2. People (사람이 사역보다 소중)

사역 중심의 위험성

``` 📌 현상: 임원 1년 하고 나면 왜 안 나올까?

원인 분석: → 일 중심이 되면 → 사람 관계가 망가지고 → 결국 사람들이 떠난다

결과: "일은 잘 돌아갔는데, 사람은 없다" 🚨 ```

개인 경험담

상황: 나는 "보스" 타입 - 섬에도 혼자 안 갔다 (공미사, 공보의에 미래를 건 사람들이란 모임을 만들고 12명의 동기들이 함께 섬으로 갔다) - "같이 죽자"는 마인드였다

무슨 일이 있었나?: 1. 교회 일을 하면서 후배들이 나를 따랐다 — 가치 때문이 아니라, 사람(나) 때문에 2. 내가 스트레스를 받아서, 관계가 어려워져서 어쩔 수 없이 공동체를 떠났다 3. 나를 따라오던 후배들도 공동체를 같이 떠났다 — 남아 있어야 할 이유를 발견하지 못했기 때문에

깨달음:

나는 후배들에게 미안했다. 가치 때문에 공동체를 하게 했어야 했는데, 나(사람)만 따라오게 했다는 것이 아쉬움으로 남는다.

사람 세우기의 본질

자리가 사람을 만드나?

``` 속담: "자리가 사람을 만든다"

하지만 이건 위험한 생각! ```

역사 사례: ``` ❓ 좋은 사람이 큰 자리에 가면 다르게 변한다? ❓ 국회의원 / 병원 원장이 되면 다른 사람처럼?

원인: 개인적으로 도덕적인 사람도 비도덕적인 사회(제도) 안에서 비도덕적으로 행동할 수 있다 (도덕적 인간과 비도덕적 사회 — 나인홀드 리버) ```

올바른 이해

``` ❌ 자리만으로는 안 된다 ✅ 자리 + 그 자리에서의 성령 역사 + 공동체의 견제

결론: 자리에서 배운다 (하지만 가치관이 있어야) ```

성경: 딤후2:2 - 세대를 잇는 양육

"또 네가 많은 증인 앞에서 내게 들은 것을
신실한 사람들에게 부탁하라"

CMF에서의 People 중심

"공동체로 산다"는 뜻

핵심가치 3번 참조: ✅ 세대별, 직능별, 지역별 다양성 존중 ✅ 적극적 참여 (지역누가회, 사역부서, 동기공동체) ✅ 전국 수련회를 통한 공동체성 회복

성경:

마22:37-40 - 두 계명

"네 이웃을 네 자신같이 사랑하라"
(사람이 사역보다 우선인 이유)

로마서12:15 - 함께하는 삶

"우는 자와 함께 울고 웃는 자와 함께 웃으라"
(공동체 속 인간관계의 깊이)

갈라디아서6:2 - 짐을 나누기

"너희가 그리스도의 법을 이루려거든 짐을 서로 지라"
(사람을 돕는 것 = 사역)

의치한간의 의미

``` 세상에서는: 의사 vs 치과의사 vs 간호사 vs 간사 → 직능간 대립이 심각

CMF에서는: 의치한간이 함께 모인다 → 다양한 직능의 연합

영향: → 믿지 않는 사람들도 보게 된다 → "저 공동체 뭐가 다른데?" → 하나님 나라의 증거! ```

3. Ministry (사역은 성령께서 이루심)

사역의 정의

사역 ≠ "내가 열심히 일하는 것" 사역 = "성령께서 일하시는 것에 내가 참여하는 것"

성경: 고전12:4-11 - 성령의 은사

"여러 은사가 있으나 같은 성령이시고...
성령을 나타내심은 각 사람에게 유익하게 하려 함이라"

사역 위기의 원인

``` ❌ 사역 > 가치 일이 목표가 되고, 가치를 잃는다 ❌ 사역 > 사람 사람을 도구화하고, 관계가 깨진다

✅ 가치 > 사람 > 사역 → 그러면 사역도 가치를 잘 반영하고 → 사람과 공동체가 성장하고 → 성령이 역사한다 ```

통합: VPM의 순환

💎 Value (공유된 가치) ↓ 👥 People (그 가치에 동의하는 사람들) ↓ ✝️ Ministry (성령께서 이루신 일) ↓ 📢 증거 (하나님 나라가 보임)

VPM과 다른 조직의 비교

조직 유형 Ministry 중심 People 중심 Value 중심
일반 기업 ✅ ❌ ❌
사교 단체 ❌ ✅ ❌
종파 집단 ❌ ✅ ❌ (잘못된 가치)
CMF (이상) ✅ ✅ ✅

실천: "같은 마음을 품으라"

성경

고전1:10 & 빌4:2 - 같은 마음

"같은 말을 하고... 같은 마음과 같은 뜻으로 온전히 합하라"

빌2:5 - 그리스도의 마음

"너희 안에 이 마음을 품으라 곧 그리스도 예수의 마음이니"

에베소서4:4-6 - 한 몸의 정신

"한 몸과 한 성령이시고... 한 주, 한 믿음, 한 세례"

현실: 이심이체(二心二體) → 목표: 일심동체(一心同體)

``` 현실: "이심이체(二心二體)" → 서로 다른 마음을 가짐 (당연한 현실) 목표: "일심동체(一心同體)" → 한 마음이 되어가는 것

⚠️ 중요: - 일심동체를 요구할 수는 없다 - 우리의 목표는 "일심동체 되어져 가는 것"이다 - 완성이 아닌 함께 걸어가는 여정(journey) ```

부부관계 적용

``` 나쁜 방식: "넌 왜 내 마음대로 안 하냐!" → 상대를 로봇으로 만드는 것

좋은 방식: "너는 다른 마음이구나. 이해하고 싶어" "그리스도 안에서 같은 마음을 배워가자" → 관계 성장 + 연합 ```

CMF에서의 적용

다양한 세대, 직능, 지역 → 당연히 다른 생각을 가짐 → 그 다름을 수용하고 → 그리스도의 가치를 공유할 때 → 진정한 공동체가 된다

결론

VPM은 CMF의 정체성

CMF = Value(기독적 가치) + People(공동체) + Ministry(성령의 역사)

학생들에게
