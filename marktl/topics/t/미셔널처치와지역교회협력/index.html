지역교회-선교단체 관계 위기

현재 진단

  • 지역교회: 생존의 문제에 허덕임. 미션 잃어버림. 파송 철회 증가. 스스로 문제가 뭔지 모름
  • 선교단체: 필드에만 집중. 지역교회 관심 희박. 전문성 약화. 선교사 관리도 벅참
  • 결과: 상호 약화, 신뢰 부재, 교류 단절

랄프 윈터의 소달리티/모달리티 → 현재 함의

  • 지역교회(모달리티) + 선교단체(소달리티) = 본질적으로 하나
  • 그러나 현실: 분열, 거리, 불신
  • WCC 에큐메니컬 운동: 뜻과 취지는 좋으나 신학적 문제

패러다임 전환 — 지역교회가 선교단체의 선교지

"선교단체(지부)의 선교지는 어디인가? 지역교회가 선교지이다."

레슬리 뉴비긴 보편적 선교 원리 적용: - "선교적 본질은 보편적이어서 서구 세계에도 적용된다" - → 한국교회도 선교지로 보아야 살아남는다 - → 선교단체가 교회를 선교지(섬김) + 파트너(협력) 양면으로 봐야

정민영 선교사 논문 (GBT) — 바람직한 선교 운동의 방향: 1. 동원이 아닌 참여로 유도 2. 지역교회로부터 자원을 뽑아가는 틀 → 교회를 섬기는 틀로 전환 3. 선교단체에 대한 교회 책임 강조 → 교회의 선교적 열정·헌신에 초점 4. 선교단체의 DNA를 지역교회에 이식 → 함께 과업을 완수하는 참여 모델

비전 캠프 외에도 지역교회 선교 세미나 등 다양한 형태의 지역교회 섬김 방안 구상 중.

동원팀 전략 — 정체성과 미션

지역교회 선교 실행위(동원팀)가 반복적으로 묻는 질문: "뭐 해야 돼요? 뭐 하는 데예요?"

이 질문은 두 개의 다른 질문이다: - 무엇을 하는 곳인가 (Being / 정체성) — 교육이 필요한 영역 - 무엇을 하는가 (Doing / 미션) — 실행의 영역

정체성 없이 행동하면 지속이 안 된다. "이익도 없고, 안 해도 되는" 구조에서 정체성이 유일한 동력이다.

정체성을 갖추기 위한 3요소 1. 선교에 대한 이해 — 풀버전의 복음 이해 + 미래 이슈 3가지 2. 자신의 달란트·위치 인식 — 개별화된 직업-선교관. "어디 있나?"를 알아야 할 일이 보인다 3. 창의성 — 하나님이 창조자이시기 때문. 다양하고 모험적인 동원

앨빈 토플러 — Learn · Unlearn · Relearn

"변화를 주도하는 유일한 방법은 새로운 지식을 얻는 것. 새로운 지식을 얻는 유일한 방법은 다시 배우는(relearn) 것."

선교도 마찬가지 — 기존의 편견을 언런(unlearn)해야 리런(relearn)이 가능하다.

엠마오 제자 예화 (눅 24) — 선교 편견을 깨는 법 - 예수님이 구약 전체로 자신을 설명하셨을 때 제자들의 마음이 뜨거워졌다 (눅 24:32) - 제자들이 부활을 믿지 않은 이유 = 편견. 선교를 배우지 못하는 이유도 동일하다 - "선교를 알아가는 것은 하나님을 알아가는 것이고, 성경을 알아가는 것이다"
