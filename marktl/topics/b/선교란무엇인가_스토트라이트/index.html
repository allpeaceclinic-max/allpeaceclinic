선교란 무엇인가

저자: 존 스토트 (John Stott) · 크리스토퍼 라이트 (Christopher Wright)
분류: 선교신학 / 복음주의 선교론 / 로잔운동

"메신저와 메시지는 분리될 수 없다." — 로잔언약

1. 이 책의 위치 — 로잔 선교 개념의 신학적 기반

이 책은 단독으로 읽히지 않는다. 로잔운동 50년의 신학적 진화를 이해해야 이 책의 무게가 보인다.

로잔운동 4대회 흐름

대회 연도 문서 핵심 전환
1차 로잔 1974 로잔언약 전도 + 사회적 책임의 통합 선언. 존 스토트 초안 작성
2차 마닐라 1989 마닐라 선언 전략적 협력, 미전도종족 운동
3차 케이프타운 2010 케이프타운 서약 총체적 선교 공식화. 크리스토퍼 라이트 주도
4차 서울 2024 서울 선언문 선포와 행동, 투게더(Together). 폴리센트릭

핵심 전환: 선교 = 영혼 구원(좁은) → 하나님 나라 전체(총체적)

이재훈 목사: "케이프타운 서약 + 선교란 무엇인가? + 하나님 백성의 선교 — 이 세 권이 함께 읽혀야 한다"

2. 선교 개념의 변화 — 왜 이 책이 중요한가

WCC vs 로잔의 긴장

WCC (에큐메니컬) 로잔 (복음주의)
선교 정의 인간화(Humanization), 사회 변혁 복음 전도 + 사회적 책임
미씨오 데이 적용 교회 밖 세상에서 하나님의 선교 찾기 교회를 통한 하나님의 선교
문제 전도의 약화 사회적 책임 소홀

로잔언약 (1974)의 혁신: WCC에서 강조하는 사회적 책임을 전도의 우선성을 잃지 않으면서 수용

복음주의의 저항과 적용 패턴

  • 근본주의: 저항만 하다가 끝남 (사회적 책임 포기)
  • 자유주의: 저항 없이 끌려들어감 (전도 포기)
  • 복음주의 (로잔): 현대적 개념에 저항하면서 사회적 참여 — "저항 + 적용"

3. 존 스토트 — 회심론

회심에 대한 혐오감, 왜?

현대인이 회심을 거부하는 세 가지 이유: 1. 개종주의(Proselytism): 제국주의·크리스텐돔·강압의 역사 2. 상대주의: 절대 진리 주장 자체가 폭력으로 인식 3. 보편구원론: 아무 것도 안 해도 구원받는다는 신학

회심의 순서 — 은혜가 먼저

``` ❌ 선불행 후구원 패턴: 나는 비참했다 → 회심 → 은혜 → 구원

✅ 바른 순서: 은혜 → 회심(에피스트레포, ἐπιστρέφω) → 구원 "자기 아버지가 답이라는 기억(은혜)이 먼저 → 돌아섬(회심)" ```

→ 부족한것이없는사람에게왜복음이필요한가_윌리몬과 동일 구조: 은혜가 회개보다 앞선다

회심의 차원 — 5가지 방향

회심(에피스트레포 = 방향을 돌림)은 단순히 개인 결신이 아니다:

차원 내용
주되심 회개 → 그리스도의 주되심 인정
교회 공동체로, 하나됨의 요구 (무교회주의 비판)
사회 "오라"가 아니라 "가라" — 세상을 향해
문화 순례자 원리(구별) + 토착화 원리(참여), 보편적 그리스도를 찾아
성령 성령이 회심을 가능하게 하심

회심 vs 거듭남

회심 거듭남
주체 인간 (돌아섬) 하나님 (낳으심)
의식 의식적 무의식 (출생은 기억 못함)
속도 과정 or 즉각 —

4. 크리스토퍼 라이트 — 회심·교회·문화

내부자 운동 (Insider Movement)

  • 그리스도에 대한 반감 ≠ 십자군·서양 종교에 대한 반감 — 구분 필요
  • 내부자 운동에 대한 세 시각: 수용 / 격려 / 배신 논란
  • 라이트의 입장: 공감하되 혼합주의 경계
  • 제도적 교회 없는 상황에서도 제자도는 가능
  • 로잔 IIC: 제자도의 다양성을 존중

판단 원칙 (행15:19 "괴롭게 하지 말고"): - 성급한 새로운 선교전략 ✗ - 비난 ✗ - 주와 함께 머물러 있으라 (행11:20-24, 바나바의 심정) - 다양한 시각 인정 (롬14:1-3)

교회론 = 구원론 = 선교론

"교회 = 복음 전파의 수단이 아니라 자체가 메시지의 구현"

  • 교회 = 존재 자체로 복음의 필수 요소
  • 교회 = 본질상 선교적 (Missional by nature)
  • 교회 = 새 창조의 시작 (역사적으로)
  • 복음 = 하나님과 화해 + 우주적 통합 + 유대인·이방인 통합 + 만물의 통합
  • 바울의 인식: 창세기에 시작된 복음 → 신약에서 교회로 가시화

회심과 문화·상황화

회심은 자신의 문화 전체를 버리는 것이 아니다:

``` 문화의 3가지 층위: ┌──────────────────────────────┐ │ 거부해야 하는 것 │ ├──────────────────────────────┤ │ 유지해도 되는 것 │ ├──────────────────────────────┤ │ 변화가 필요한 것 │ └──────────────────────────────┘

접근 방식: 배척과 금지 → 조건부 관용 → 비판적 인정 ```

5. 선교에 대한 4가지 견해와의 위치

(선교에대한네가지견해 — 스탠리 그렌츠, 크리스토퍼 라이트 외)

견해 대표 핵심

구원론적 선교 (넓은) 존 스토트·크리스토퍼 라이트 전도 + 사회적 책임
참여선교 크리스토퍼 라이트 하나님 선교에 참여, 창조세계 돌봄 포함
상황중심선교 존 프랭키 상황에 따른 다양한 증언
성례전적 선교 피터 라잇하르트 예전과 삶의 조화, 교회의 공적 존재

탐구 중인 질문

Q1. 전도의 우선성을 유지하면서 사회적 책임을 통합하는 실천적 균형은 어디인가? - 부분 답변: 보냄받음_킴햄몬드 — 총체적 선교(차원) + 사도적 우선순위(의도성) - 부분 답변: 로잔언약 — "사회적 행동이 전도에 종속되지 않지만, 전도의 우선성은 있다" - 여전히 열린 질문: 하루 100명 진료하는 의사에게 사도적 우선순위는 어떤 형태인가?

Q2. 내부자 운동과 혼합주의의 경계는 어떻게 그어야 하는가? - 부분 답변: 라이트 — 제자도의 다양성 존중, 전통적 교회 표지로 성급하게 판단 ✗ - 부분 답변: 보냄받음_킴햄몬드 — 다니엘 원리 (언어·학문 참여 but 정체성 지킴) - 여전히 열린 질문: 이슬람 문화권 의료 파트너 사역에서 정체성과 참여의 경계는?
