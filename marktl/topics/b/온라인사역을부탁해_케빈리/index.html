온라인 사역을 부탁해

저자: 케빈 리 (Saddleback Church 온라인 사역 담당)
분류: 교회론 / 디지털 사역 / 선교적 교회

핵심 전제

온라인보다 사역이 목표 — 교회의 목표와 동일하게 세팅

  • 온라인 사역의 이유: 성도와 비신자들이 거기 있기 때문
  • 다음 세대의 삶은 모든 측면이 디지털화되고 있다
  • 영상은 impression (인상), 글은 structure (구조) — 매체 특성을 이해하고 사용

1. 교회 모델의 전환

Centralized Church → Decentralized Church
과거 교회 집중형 모델 → 교회 분산형 모델

  • 확실한 기반 위에서 불확실한 새로운 아이디어를 계속 시도한다
  • Ex) 교회에서 오큘러스 VR을 주고 연구하게 함 → 대담한 시도의 자세
  • 다음 세대 교회의 모델은 전 세대 교회 모델에서 나오기 힘들다 (알 리스)

2. 온라인 소속감

소속감은 교회에 갖게 하기보다 서로에게 갖게 하는 것이 중요하다

  • 어카운터빌리티(accountability) — 말씀이 중심이 되어 서로가 서로에게 책임 의식을 갖고 살아가는 교제
  • 흩어지는 교회의 교제에서 가장 중요한 것: 이 상호 책임감
  • 소그룹 모델: "웬 참견이야" → "참견 좋아, 유익해" — 온라인 제자훈련 커리큘럼 강력 추천

3. 예배와 온라인 참여

예배당의 오프라인 예배를 그대로 송출하는 것은 참여의 태도로 인도하는 데 결정적 걸림돌이 된다

  • 오프라인 예배 → 온라인 참여자는 구경꾼이 될 수밖에 없음
  • 온라인에 맞는 별도 참여 설계가 필요

4. 소그룹 모델의 전환

기존 전환
마스터-티처 모델 퍼실리테이션 모델
한 사람의 역량에 소그룹 건강이 달림 귀납적 성경연구 — 가르침보다 나눔
가르침 중심 하브루타 나눔 중심

5. 온-오프라인 하이브리드 원칙

온라인으로만 교육하면 부족하고, 오프라인으로만 하면 비효과적이다

  • Once for all and once for many — 시간의 확장성
  • 첫 번째 질문(일반적) → 온라인 / 두 번째 질문(개인적) → 오프라인·개별화
  • 온라인: 정보 전달·커리큘럼 / 오프라인: 관계·적용·책임

6. 교회의 외향성

  • 교회 안에서 봉사 → 지역을 섬기는 모습으로 확장 ("당신이 속한 지역의 필요를 채워 주세요")
  • Ex) 미혼모 사역
  • 교회의 빛 → 그리스도의 빛으로 재프레이밍
  • 소그룹 + 선교단체 → 단기선교, 영적 가족

7. 실패 원인

그 변화를 시행하는 팀이 뚜렷하지 않아서

연결 고리

이상협 강의·사역과의 연결

  • 온라인 방송 교육 강의 — ATEM mini pro, OBS, Zoom·유튜브 동시방송 등 실제 구현
  • 온라인 사역툴 회의 — 대구동신교회 앱 개발 논의; "오프라인을 온라인화하는 수준" 비판 = 케빈 리의 경고와 일치
  • 분산형 교회 모델 → 에젤·KCMF·인터서브 등 사역별 온라인 소그룹 적용 가능성
