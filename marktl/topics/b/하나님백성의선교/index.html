하나님 백성의 선교 (하백선)

저자: 크리스토퍼 라이트 (Christopher Wright)
약칭: 하백선
총 과수: 15과
활용: LAMs (Life As Mission School) 핵심 교재 — 단체·교회마다 축약 버전 운영
강사: 이상협 (램스코리아 코디네이터), 송기태 선교사 (인터서브) 외

1장 — Life Is Mission: 당신의 삶 자체가 하나님의 선교

핵심 선언

"당신의 삶 자체가 하나님의 선교입니다"

선교를 '특별한 사람이 특별한 장소에서 하는 특별한 사역'이라는 편견에서 벗어나,
모든 백성이 모든 장소에서 모든 영역에서 감당하는 부르심으로 재정의.

Missio Dei — 선교의 주체는 하나님

  • 요 5:17 "아버지께서 이제까지 일하시니 나도 일한다"
  • 선교를 만들어내는 주역이 아닌, 하나님이 이미 행하시는 구원 역사에 동참하도록 초대받은 자들
  • 성패가 우리의 능력이 아닌 하나님의 신실하심에 달림

로잔 언약의 3중 구조

구조 내용
온 세계 지리적(땅끝) + 영역적(정치·경제·의료 등 모든 삶의 영역) + 생태적(피조 세계)
온 교회 목회자·평신도·선교사·전문인 구분 없는 원 팀 — 교회는 세상으로 파송된 공동체
온전한 복음 영혼 구원 + 사회적 책임 — 분리 불가, 독수리의 두 날개

Being > Doing

  • 무엇을 하기(Doing) 전에 누구인지(Being)가 우선
  • 하나님의 성품을 닮은 존재 자체가 가장 강력한 선교 전략
  • 거룩함 = 도피주의가 아닌, 세상 한가운데서 하나님 나라의 '다름'을 증명

이원론 극복 + 보냄 받은 자의 의식

  • 성소(교회 안) vs 광장(직장) 구분 타파
  • 요 20:21 "아버지께서 나를 보내신 것 같이 나도 너희를 보내노라"
  • 직장·병원 = 하나님이 파송한 선교지
  • Unlearn + Relearn: 왜곡된 선교관 비우고 총체적 선교관으로 채우기

의료 현장 적용 (HAMs 연계)

  • 진료실 = 최전방 선교지
  • 환자의 전인적 고통에 귀 기울이는 태도
  • 의학적 전문성을 갈고닦는 수고 = 하나님께 드리는 선교적 제사
  • "내가 고치는 것이 아니라 하나님의 은혜가 환자를 돌보신다"

2장 — The Missional Narrative: 자신이 속한 그 이야기를 아는 백성

핵심 질문

"당신은 어떤 이야기의 일부입니까?"

알래스데어 매킨타이어: "내가 무엇을 해야 하는가?"에 답하려면 먼저 "나는 어떤 이야기의 일부인가?"를 알아야 한다.
→ 세상의 서사(성공·권력·생존) vs 하나님의 서사(우주적 회복)

엠마오 도상 (눅 24)

  • 제자들: 파편화된 지식은 있었으나 전체 이야기 연결 못함 → 절망
  • 예수님: 성경 전체를 하나의 흐름으로 설명 → 마음이 뜨거워짐 → 도망치던 발걸음을 선교적 증언으로

성경 읽기 두 기둥

  1. 메시아 중심: 모든 서사 → 그리스도의 고난과 부활로 수렴
  2. 선교 중심: 성경 전체 = 모든 민족을 향한 하나님의 선교 이야기

4막 드라마

막 내용 선교적 의미
1막 창조 샬롬, 인간 = 하나님의 대리 통치자 선교 = 샬롬 회복
2막 타락 우주적 단절 — 하나님·인간·자연 모두 파괴 세상은 병들어 있다 (Disease)
3막 구속 아브라함 → 출애굽 → 십자가·부활 우리는 이미 결정적 승리 안에 있다
4막 새 창조 만물의 완벽한 회복, 샬롬의 완전한 성취 선교의 종착역 — 포기하지 않는 이유

아브라함 언약 (창 12:1-3)

  • 복을 받는 것과 세상에 전하는 것은 분리 불가
  • 축복 = 저수지(나만)가 아닌 파문(세상을 향해 뻗어나감)

선교적 성경 읽기

  • 나 중심: "이 말씀이 오늘 나에게 어떤 위로를?"
  • 하나님 중심: "이 본문은 하나님의 어떤 선교적 계획을?"
  • 미션 중심: "오늘 내가 속한 세상에서 어떻게 살아야?"

의료 현장 적용 (HAMs 연계)

  • 병리학 = 2막(타락의 결과)을 다룸
  • 임상 치료 = 3막(구속)의 사역 — 회복을 향한 투쟁
  • 치유 = 4막(새 창조)의 예표 — 샬롬을 미리 맛보게 함
  • 진료실 선교 3단계: 경청(고통을 들음) → 위로(회복 이야기 투영) → 기도(중보)

3장 — Missional Creation Care: 창조 세계를 돌보는 백성

핵심 질문

"환경 보호가 복음 전파와 무슨 상관이 있는가?"

창조 세계 돌봄 = 선교의 본질적 일부 (부수 활동이 아님)

선교의 출발점: 창세기 1~2장

  • 창세기 3장(타락)이 아닌 1~2장(창조)에서 시작해야 함
  • "보시기에 좋았더라" — 물질 세계는 그 자체로 선함
  • 창조 → 선교는 '사고 수습'이 아닌 원래 선한 세계의 회복

두 기둥: 창조 + 새 창조

  • 골 1:20 — 그리스도 안에서 만물이 새롭게 회복됨 (폐기 아님)
  • 선교 = 인간만이 아닌 모든 피조물과의 관계 회복 (우주적 프로젝트)

인간의 이중 신분 (거룩한 역설)

신분 히브리어 의미
왕 에바드 하나님의 성품으로 피조 세계를 다스림
종 샤마르 정원을 원래의 아름다움대로 지키고 보존

문화 명령 + 대위임령

  • 창조 명령(생육·번성·정복·다스림) = 대위임령(제자 삼기)의 전제이자 토대
  • 구속 = 창조를 완성해 가는 과정
  • 인간 중심주의 탈피: 피조물도 독자적으로 하나님께 찬양 (시 148편)

십자가의 우주적 구원 (골 1:20 / 롬 8장)

  • 그리스도의 보혈 = 인간 영혼 + 만물 전체를 화목케 함
  • 탄식하는 피조물(롬 8) — 하나님 자녀들의 출현을 기다림
  • 창조 세계 돌봄 = 그리스도의 구속 사역에 동참

생태학적 실천 = 선교적 영성

  • 재활용, 탄소 배출 감소 = 창조 신앙의 일상적 표현
  • 기후 위기: 타락의 결과 → 가난한 나라의 소외된 이웃에게 먼저 피해 → 환경 보호 = 선교적 정의의 적극적 형태
  • 단순한 삶: 소비주의 거절, 자족, 여백 — "적게 소유하지만 더 풍성하게 존재"

종말 이해 교정

  • 불 심판(벧후 3) = 파괴가 아닌 정제(Purification) — 죄의 흔적만 소멸
  • 이 땅에서 행한 선한 일과 창조 돌봄의 흔적 → 새 하늘과 새 땅으로 이어짐

의료 현장 적용 (HAMs 연계)

  • 의술 = 창조주의 창조물을 수리하는 거룩한 일
  • 의료인의 3가지 태도: 겸손(하나님이 고치심) + 경외(생명의 신비) + 예배(진료가 예배)
  • 병원 실천: 전자 차트 활용, 안전한 의료 소모품, 에너지 절약
  • 환자에게 생태적 건강관 교육 (자연환경 = 건강의 토대)

4장 — Healing for the Nations: 열방에 복이 되는 백성

핵심 선언

"축복받은 축복의 통로 (Blessed to be a Blessing)"

선교의 대헌장 (창 12:1-3)

  • "땅의 모든 족속이 너로 말미암아 복을 얻을 것이라"
  • 한 가문을 부르셨지만 시선은 처음부터 전 인류를 향함

선택의 논리

  • 선택 = 독점 특권 X → 열방을 복 주시기 위한 파송의 전 단계
  • 복을 가두는 '저수지' = 선민의식의 타락
  • '너로 말미암아': 하나님의 복은 반드시 자기 백성을 경유하심 → 내 순종에 이웃의 복이 달림

복의 총체적 의미

물질적 풍요 + 관계적 샬롬 + 영적 생명 = 하나님의 통치 아래 누리는 총체적 '잘 됨'

갈라디아서 3장

  • 갈 3:8 — 복음의 기원 = 아브라함 언약 (기독교는 뿌리부터 이방인을 위한 선교적 종교)
  • 갈 3:29 — 우리 = 아브라함의 자손 → 선교적 사명을 유산으로 이어받음

수평적 축복

  • 적대자(라반·보디발·바로)에게까지 흘러감
  • 중보기도의 범위: 내 적대자와 경쟁자까지 확장
  • 요셉 모델 (창 39): 보디발의 집이 요셉 때문에 복을 받음 → 자기 점검: "나 때문에 우리 병원과 팀원들이 덕을 보는가?"

실천 3가지

  1. 말의 축복 — 만나는 이에게 "하나님의 복이 임하기를" 선포
  2. 친절의 선교 — 낯선 이와 약자에게 베푸는 작은 환대
  3. 재정적 유통 — 소득 중 열방의 가난한 이들을 위해 의도적으로 흘려보냄

의료 현장 + 의사 누가

  • 전문 지식 = 신탁 자산 (사유화하면 아브라함 언약 배반)
  • 진료 = 샬롬의 전달 — 환자가 "존중받고 사랑받았다" 느끼도록
  • 의사 누가: 의술 + 기록의 은사 → 열방의 영원한 복

5장 — Integrity as Mission: 하나님의 도를 행하는 백성

핵심 주제

"윤리 없는 선교는 없다"

Ethics as Mission — 삶의 방식이 복음의 신뢰도를 결정함

창 18:19의 선교 공식 (세 톱니바퀴)

선택(Election) → 윤리적 순종(Ethics) → 선교적 약속 성취(Mission)

윤리(가운데 톱니바퀴)가 빠지면 선교는 논리적으로 불가능

여호와의 도: 두 개념

개념 히브리어 의미
의 쩨다카 관계적 올바름 — 하나님·이웃·피조물과 마땅한 관계 실천
공도 미쉬파트 불의를 적극적으로 바로잡고 억눌린 자의 권리 회복
  • 소돔의 도 (겔 16:49): 풍요·탐욕·약자 무관심
  • 야웨의 길: 나눔·환대·정직

제아카 (고통 소리)

  • 하나님은 불의에 짓눌린 자들의 제아카에 민감하심 — 직접 내려와 확인하심
  • 사각지대의 고통 소리에 귀 기울이고 그곳에 쩨다카와 미쉬파트를 가져가는 것이 선교

윤리 = 전도의 문

  • 세상은 우리의 삶(신뢰성)을 먼저 본 뒤 메시지(복음)를 듣는다
  • 한국 교회 반성: 57,000개 교회 vs 낮은 사회 신뢰도 → 쩨다카 실종
  • 크리스토퍼 라이트 인용: 인도 선교 최대 장벽 = 영국 그리스도인의 위선
  • 코람 데오: 보이지 않는 곳에서의 정직함이 진짜 영성
  • 구속적 윤리: 보복 대신 용서 — "용서의 매력"이 불신자를 이끄는 힘

배려의 경제학 (레 19)

  • 밭 모퉁이 = 가난한 자와 나그네를 위한 '하나님의 몫'
  • 나눔 = 선행이 아닌 공의(쩨다카)의 실천

의료 현장 적용 (HAMs 연계)

  • 쩨다카: 과잉 진료 거부, 정직한 진료
  • 미쉬파트: 차별 없는 환대, 소외 계층을 위한 목소리
  • 다니엘 모델: 바빌론(세상 논리)에서 뜻을 정한 정직한 소수가 역사를 바꿈
  • 사회적 공의: 공적 광장(Public Square)으로 나아가는 정의로운 선교

6장 — Redemptive Living: 구속적 삶을 살도록 구속받은 백성

핵심 질문

"당신은 구원받은 자로서 누군가를 '해방'시키는 삶을 살고 있습니까?"

구속(Redemption)의 성경적 의미

  • 고엘(Goel): 친족의 기업을 무르고 억울함을 풀어주는 보호자
  • 하나님 = 위대한 고엘 — 사탄의 압제 아래 신음하는 소유를 피 값 치르고 되찾아 오심

출애굽 — 총체적 구속의 원형

  • 단순한 종교 변화 X → 정치·사회·경제적 압제로부터의 총체적 해방
  • 출 3장: "내가 고통을 보고, 비명을 듣고, 내려가서, 건져내려 하노라"
  • → 하나님의 구속은 보고·듣고·내려가고·건져내는 4단계 행동

십자가 — 궁극적 새 출애굽

  • 죄와 사망의 권세에서 완전 해방 + 우주적 재창조
  • 개인적(신분 변화) + 공동체적(사회 관계 회복, 불의한 시스템 재편)

희년 (레 25) — 구속적 경제

  • 50년마다: 부채 탕감 + 토지 반환 + 종 해방
  • 토지와 인생의 진짜 주인 = 하나님
  • 선교적 실천: 가난의 대물림 끊기, 다시 시작할 기회 제공

구속적 삶(Redemptive Living)

  • 소극적(죄 안 짓기) → 능동적(고통받는 자를 찾아 건져내기)
  • "거저 받았으니 거저 주라" (마 10장) — 받은 구속의 감격이 동력
  • 과거 은혜를 매일 기억 = 타인을 향한 긍휼의 원천

현장 적용

  • 현대판 노예: 중독(알코올·마약·스마트폰)에서 경청과 기도로 해방
  • 사회적 약자 옹호: 태아·장애인·이주노동자의 목소리가 됨
  • 구속적 재정: "내 것은 내 것" 사슬에서 자유 → 나눔
  • 구속적 리더십: 실수한 동료에게 다시 일어설 기회 제공
  • 빌레몬·오네시모: 노예를 형제로 받아들인 파격

의료 현장 적용 (HAMs 연계)

  • 선한 사마리아인 태도 = 진료 태도의 모델
  • 경제적 이유로 치료 포기하는 환자 → 구속적 무료 진료·봉사
  • 육체 뒤의 가정·영적 갈급함까지 돌보는 전인적 케어(Holistic Care)
  • Already / Not Yet: 인내하며 작은 승리의 벽돌을 쌓아가는 선교

7장 — God's Medical Ambassadors: 세상을 향해 하나님을 대표하는 백성

핵심 질문

"세상 사람들은 당신을 보고 하나님의 성품을 어떻게 '광고'받고 있습니까?"

출 19:4-6 — 시내산 언약의 3중 정체성

정체성 히브리어 의미
보배로운 소유 세굴라 왕의 가장 소중한 보물 — 성과 이전에 이미 완전한 존재적 가치
제사장 나라 맘레케트 코하님 공동체 전체의 소명 — 하나님↔세상의 중간 다리
거룩한 백성 고이 카도쉬 카도쉬(구별됨) — 장벽이 아닌 증거를 위한 차별성

전제: "세계가 다 내게 속하였나니" — 하나님은 온 우주의 주인

제사장의 두 방향

  1. 중보 (세상 → 하나님): 환자·이웃의 고통을 기도로 하나님께 올림
  2. 계시 (하나님 → 세상): 하나님의 정직·사랑·평강을 삶으로 번역해 보여줌

거룩 = 침투

  • 분리·도피 X → 가장 세속적인 곳에 침투하여 변혁
  • 벧전 2:9 "왕 같은 제사장들" → 브랜드 앰배서더(Brand Ambassador)
  • 벧전 2:12 — 세상은 입술의 교리보다 '선한 행실'을 먼저 목격함

하나님 나라 대사의 3특징

  • 권위(Authority): 자기 말이 아닌 왕의 말을 전함
  • 책임(Responsibility): 내 불친절이 하나님의 이미지를 훼손할 수 있음
  • 소속감(Belonging): 빌 3:20 — 시민권은 하늘에 있음

살아있는 광고판 — 3가지 성품

  • 정직 → 하나님의 진실하심 광고
  • 인내 → 하나님의 자비로우심 광고
  • 탁월함 → 하나님의 창조적 지혜 광고

성경 모델

  • 다니엘·에스더: 권력의 심장부에서 하나님을 대표 — 이방 왕들이 하나님을 찬양
  • 에파프로디토: 스포트라이트 없이 묵묵히 헌신하는 충성스러운 대사
  • 이스라엘의 실패: 거룩을 잃으면 하나님 이름이 모독받음 → 경고

윤리적 대조 사회 (Contrast Society)

  • 군림 X → 섬김 / 소유 X → 나눔 / 보복 X → 용서
  • 세상이 흉내 낼 수 없는 '다름' = 자석처럼 사람을 하나님께 끌어당김
  • "나의 거룩이 곧 선교다"

의료 현장 적용 (HAMs 연계)

  • 흰 가운 = 제사장의 의복 / 진료실 = 성소(Sanctuary)
  • 과잉 진료 거부 → 하나님의 진실함 광고
  • 임종 환자 끝까지 돌봄 → 하나님의 신의 광고
  • 가난한 환자 배려 → 하나님의 긍휼 광고
  • 환자 질문: "왜 저를 이렇게 대하나요?" → "하나님이 당신을 사랑하시기 때문입니다"

8장 — Magnetic Medical Mission: 다른 사람들을 하나님께로 끌어들이는 백성

핵심 질문

"당신의 진료실, 당신의 삶에는 믿지 않는 사람들을 하나님께로 끌어당기는 '매력'이 있습니까?"

두 가지 선교 방향

방향 유형 내용
밖으로 원심적 선교 (Go) 땅끝으로 나아가 복음 선포 (신약 대위임령)
안으로 구심적 선교 (Come) 아름답게 살아 사람들이 스스로 찾아오게 함 (구약 제사장 나라 비전)

선교적 자기력 (Magnetic Mission)

  • 우리가 스스로 빛을 내는 발광체 X
  • 내주하시는 하나님의 영광이 일상을 통해 투과·굴절되어 나타나는 현상
  • 억지로 설득하지 않아도 "저 사람은 왜 저렇게 살까?" 자연스럽게 끌어당김

구심적 선교의 근거: 신 4장

  • "이 큰 나라 사람들은 과연 지혜와 지식이 있는 백성이로다"
  • 신앙의 가시성(Visibility) — 우리의 삶의 방식이 외부에서 관찰 가능해야 함

3가지 필수 요소

  1. 가시성 — 위기의 순간에 정체성과 가치관이 자연스럽게 드러남
  2. 독특성 — 세상의 유행을 거슬러 하나님 백성만의 거룩함 유지
  3. 호기심 유발 — "당신의 그 다름의 이유가 무엇입니까?" 질문을 낳는 삶

거룩한 멋 (SWAG)

  • 종교적 의무감으로 찌든 피곤함 X
  • 주님과 동행하는 자의 당당함과 자유함
  • 위기 앞에서도 여유와 배짱 → 진짜 전도지

선교 순서 (벧전 3:15)

  1. 삶이 먼저 질문을 유발 — "어떻게 그렇게 평안하십니까?" 자발적 질문 생성
  2. 준비된 대답 — 온유함과 존중으로 소망의 이유(예수 그리스도) 고백

두 공간의 자기력

  • Front of House (진료실·대기실): 환대와 평강의 공기 / 1분의 진실한 여유
  • Back of House (동료 관계): "우리 원장님이 믿는 하나님이라면 나도 믿어보고 싶다"

소금과 빛

  • 소금: 스며들어 희생함 — 갈등 중재, 분위기를 살맛 나게
  • 빛: 진리의 기준을 명확히 세움 — 불의한 관행에 방향 제시

추문의 종교 경계

  • 탐욕·위선·무례함 = 척력(Repulsion) — 사람들을 하나님에게서 멀리 밀어냄
  • 진짜 매력의 원천: 예수님 — 죄인과 세리가 스스로 찾아온 분

구심적 선교의 궁극적 목적: 예배

매력 → 호기심 → 대화 → 하나님께 무릎 꿇고 경배 → 진료실이 거룩한 예배의 처소가 되는 것

9장 — Truth, Loyalty and Mission: 살아계신 한 분 하나님을 아는 백성

핵심 전환: Showing → Telling

단계 내용
전반부 (1-8장) Being + Ethics + Attraction = 보여주는 선교 (Showing)
후반부 (9-15장) Message + Witness + Proclamation = 들려주는 선교 (Telling)

행함 없는 말 = 허구 / 말 없는 행함 = 모호함 → 둘이 함께해야 완성

핵심 구절: 신 4:35

"야웨는 하나님이시요 그 외에는 다른 신이 없느니라"

다원주의 사회에서 타협 없이 유일하신 하나님의 독보성을 선포해야 함

유일신 신학의 발전

  1. 단일신론 (출 15:11): 많은 신들 중 최강자
  2. 혼합주의 (가나안 정착): 여호와 + 바알 병행 섬김
  3. 윤리적 유일신론 (엘리야 시대): 공의와 정의를 요구하시는 유일하신 분
  4. 절대적 유일신론 (바벨론 포로기): 다른 신은 존재하지 않음
  5. 삼위일체 유일신론 (니케아 325): 진리의 변개가 아닌 인식의 확장

지식(Knowing) + 충성(Loyalty) = Mission

  • 알아야 전할 수 있음 — 모르면 공허한 메아리
  • 성경의 지식 = 단순한 정보 X → 인격적 신뢰와 관계
  • 유일하신 하나님을 알면 → 완전한 충성이 자연스럽게 요구됨

"알려지기 원하시는 하나님의 확고부동한 결심이 있기 때문" — 크리스토퍼 라이트

기독교 신앙의 사유화 문제

  • "무슨 상관이 있느냐" = 하나님을 아는 것을 개인 영역에 가두는 오류
  • 구약에서 하나님을 아는 것 = 공동체 전체가 받은 포괄적 사회적 의제

라이프 에즈 미션 패러다임 전환

Life and Mission → Life for Mission → Life as Mission → 가정·직장·사회에서의 모든 삶 = 선교적 삶 (Missional Life)

10장 — Living as God's Witnesses: 살아계신 하나님을 증거하는 백성

  • 핵심 선언: "너희는 나의 증인이라!" (사 43:10)
  • 1-9장 전환: Showing(존재·행동) → Telling(말씀 증언)의 통합

증거(Witness)의 본질

  • 법정 용어: 직접 보고 듣고 경험한 사실을 있는 그대로 발언하는 자
  • 총체성: 입술의 말(Speaking) + 증인의 삶(Living)이 통합되어야 온전한 증언
  • 복음을 새롭게 발명하는 것이 아닌, 이미 일어난 사건을 증언하는 것

하나님의 이중 문제 (Double Problem)

  1. 무지한 열방: 창조주 하나님이 누구인지 전혀 알지 못함
  2. 눈먼 이스라엘: 선택받았음에도 우상 숭배에 빠짐 → 해결책: 눈먼 이스라엘을 먼저 고치셔서 종과 증인으로 삼고, 그들을 통해 열방을 깨우침

증인의 두 목적

  1. 증인 자신을 위해: 복음을 입술로 전할 때 성령이 내면에 더 강력히 인치심
  2. 하나님을 위해: 야웨만이 참 하나님이심을 선포하여 하나님의 명예를 회복

증인의 삼중 구조

  • 존재(Being): 거룩한 성품과 정체성 — "우리는 누구인가?"
  • 행동(Doing): 의와 공도, 구속적 삶 — "우리는 무엇을 하는가?"
  • 말씀(Telling): 그리스도의 유일성과 복음 선포 — "우리는 무엇을 말하는가?" → 셋 중 하나라도 빠지면 안 됨: 삶 없는 말 = 위선, 말 없는 삶 = 도덕주의

내적 변혁과 지상대위임령

  • 복음으로 변혁된 삶에서만 증언이 생명력을 가짐
  • 예수님의 방식: 3년 동안 함께 살며 '하나님 나라의 방식'을 먼저 보여주심
  • 지상대위임령(마 28장) 핵심 동사: "제자를 삼으라(Make Disciples)" — 교인 확장이 아닌 생명 전수
  • 증인은 내 결단이 아닌 성령의 권능으로 '되어지는 것(Be)' (행 1:8)

마르투스(Martus)의 의미

  • 증인 → 순교자: 진정한 증거의 가치는 고난과 손해를 감수할 때 가장 선명하게 드러남

현대판 우상들

  • 과학만능주의, 맘몬(돈), 극단적 안락함과 자기 결정권
  • 삶의 양식(Lifestyle)으로 우상들이 인간을 구원할 수 없음을 선포

성경 속 증인 모델

  • 나아만 장군의 여종 (왕하 5장): 전쟁 포로 신분 → 담대하게 증거 → 이방 대국의 2인자가 하나님을 찬양
  • 증거의 능력은 신분·지위가 아닌 살아계신 하나님에 대한 절대적 확신에서 나옴

HAMs 의료 적용

  • Being(인격·따뜻함) + Doing(탁월한 의술) + Telling("진정한 치유는 생명의 주인이신 하나님께 있습니다")
  • 진료실 = 인간이 겸비해지는 곳 → 하나님을 증거할 최고의 골든 타임
  • 리베이트 거절, 진상 환자에게도 예수의 친절 유지 = 삶으로 드리는 증언

11장 — Gospel Identity and Kingdom Mission: 그리스도의 복음을 선포하는 백성

  • 핵심 질문: "당신이 환자와 이웃에게 전하는 '좋은 소식'의 실체는 과연 무엇입니까?"

복음(Gospel)이란 무엇인가

  • 유앙겔리온(εὐαγγέλιον): "좋은 소식" — 고대 로마의 승전보·황제 즉위 공포
  • 기독교적 전환: 하나님이 그리스도를 통해 세상을 구원하시고 만유의 왕으로 즉위하셨다는 우주적 승전보
  • 핵심 구별: 복음은 조언(Advice)이 아닌 뉴스(News) — 우리가 해야 할 일이 아니라 하나님이 이미 이루신 일

복음의 신구약적 뿌리

  • 구약 뿌리 (사 52:7): "네 하나님이 통치하신다(Your God reigns)!" — 포로기 흑암 속 해방 선포
  • 예수님의 선포 (막 1:15): "때가 찼고 하나님의 나라가 가까이 왔으니" — 죽어서 천국이 아닌 지금 여기서 시작된 하나님의 통치
  • 사도들의 선포 (고전 15장): 그리스도의 죽음·장사·부활 — 십자가 + 부활 = 하나님의 즉위식

복음의 총체적 범위

  • 영혼 구원 + 사회 구조의 정의와 관계 회복 + 문화의 성화 + 자연 생태계의 해방
  • 골 1장: 만물이 그리스도의 피로 화목하게 되기를 기다림

다원주의 시대의 복음 선포

  • 복음은 철학적 의견이 아니라 역사 속에서 성취된 절대적 사건
  • "생명을 살리는 유일한 치료제는 배타적인 것이 아니라 생존에 필수적인 것"
  • 복음의 능력: 롬 1:16 "모든 믿는 자에게 구원을 주시는 하나님의 능력"

말(Telling)과 행동(Doing)의 통합

  • 말만 하고 행동 없음 = 오류 / 익명의 자선만 함 = 선교적 오류
  • 예수님의 방식: 말씀(선포) + 기적·치유(입증)
  • 탁월하고 진실한 진료(Doing) = 입술의 복음(Telling) 신뢰도를 높이는 확성기

복음 선포 4단계 (진료실 적용)

  1. 라포(Rapport) 형성: 공감과 탁월한 진료로 신뢰의 토양 구축
  2. 적절한 타이밍: 환자가 인생의 근본적 질문을 던질 때 영적으로 포착
  3. 간결한 선포: "제가 믿는 예수님은 죽음을 이기신 분입니다. 그분께 참 소망이 있습니다"
  4. 사후 관리: 계속 중보하며 삶의 모델이 되어줌

선포의 도구

  • 성령의 나타나심과 능력 + 하나님의 말씀 + 기도(엡 6:19) — 인간의 유창한 말이 아님

성경 속 모델: 빌립과 에티오피아 내시 (행 8장)

  • 성경 근거로 명확하게 예수의 복음을 가르침
  • 진료실의 환자 한 명이 하나님이 예비하신 '에티오피아 내시'일 수 있음

HAMs 의료 적용

  • 질병 = 하나님의 통치가 거부된 타락의 흔적
  • 의술을 통한 치료 = "하나님의 나라가 지금 이곳에 임하고 있다"는 복음적 행위
  • 의술(Medical Art): 육신의 생명 연장 / 복음(Gospel): 죽은 영혼에 영원한 생명 부여

12장 — Sent Identity: 보내고 보냄받는 백성

  • 핵심 질문: "나는 보냄 받은 자입니까, 보내는 자입니까, 아니면 둘 다입니까?"

선교의 위대한 사슬 (롬 10장)

보내심(Sending) → 전파 → 들음 → 믿음 → 부름 → 모든 사역의 기초는 파송: "보내심을 받지 아니하였으면 어찌 전파하리요"

보내시는 하나님

  • 아브라함 → 모세 → 예레미야 → 요나 → 사도들 → 독생자 예수(성육신 = 파송의 절정)
  • 히 3:1: 예수님은 "우리가 믿는 도리의 사도" — 최고의 선교사이자 보냄 받은 자

파송의 대헌장: 요 20:21

"아버지께서 나를 보내신 것 같이 나도 너희를 보내노라" - '같이(As)'의 무게: 낮아짐·성육신·섬김의 방식 그대로 세상으로 보냄받음 - 예수님의 파송은 오늘날 우리에게까지 연속성을 가짐

모든 그리스도인은 보냄 받은 자 — 이원론 타파

  • 타문화권 선교사만 보냄받은 것이 아님
  • 의사 면허증·졸업장 = 하나님 나라의 공식 파송장
  • "나는 생계를 위해 일하는 자가 아니라, 하나님이 이 현장에 심으신 전략적 선교사다"

보내심의 두 방향

유형 내용
지리적 보내심 (Geographic) 국경을 넘어 타문화권 복음 전파
영역적 보내심 (Spheres) 의료·교육·비즈니스 등 공적 광장으로 파고듦

사도적 우선순위 (Apostolic Priority)

  • 선교를 우연한 현상으로 두지 않고 의도적으로 복음 전파를 목표 삼는 것
  • 목회적 돌봄(내부)과 사도적 전파(외부) 사이의 건강한 균형

위임의 신비

  • "하나님은 우리 없이도 하실 수 있지만, 굳이 우리와 함께하시기를 기뻐하심" (마이클 오)
  • 99%의 평신도 전문가들이 공적 광장에서 수행해야 할 몫
  • 하나님의 전능하심 + 우리의 작은 순종 = 위임의 역설

보내는 자의 역할

  • 나가는 자와 보내는 자는 동일한 선교의 동역자, 동일한 하늘의 상급 — 원팀(One Team)
  • "하나님께 합당하게 그들을 전송하라" (요삼 1:6)
  • 입체적 지원: 기도 + 재정 + 정서적 지지 + 재입국 시 환대

의료 선교 팀워크

  • 나가는 누가(Going Luke): 선교지로 직접 들어가 의술과 복음을 베품
  • 보내는 누가(Sending Luke): 재정 후원, 의학적 자문, 중보 기도
  • "우리는 각자의 자리에서 보냄 받은 하나님의 한 팀"

사도적 우선순위 상실의 경고

  • 자기 유지와 내부 행사에만 소진 → 선교적 야성 소멸
  • 고인 물은 부패하듯, 외부로 자원을 보내지 않는 교회는 성경적 정체성을 잃음

성경 속 모델: 안디옥 교회 (행 13장)

  • 가장 핵심 지도자 바나바와 사울을 기꺼이 파송
  • "가장 좋은 것"을 내어드릴 때 온 세계가 복을 받는 원리

13장 — Public Square Mission: 공적 광장에서 살아가는 백성

  • 최종 선언: "당신의 삶, 당신의 직업 자체가 하나님의 선교입니다" — 램스스쿨 13주 여정의 완성
  • 핵심 질문: "당신의 신앙은 주일 예배당 안에 갇혀 있습니까, 아니면 월요일 광장으로 흐르고 있습니까?"

공적 광장 = 성문(The Gate)

  • 구약의 성문: 시장·재판·의회가 열리던 도시의 심장부
  • 현대의 광장: 병원·학교·기업·정치·언론 — 가치관이 충돌하고 시대의 방향이 결정되는 공간

최대의 적: 이원론(Dualism)

  • 교회 일만 거룩(Sacred), 직장은 세속적·소모적(Secular)이라는 거짓말
  • 영혼 구원만 가치있고 사회 정의·신체 치유는 부차적이라는 오해
  • 결과: 그리스도인의 삶이 파편화 → 세상에서 선교적 영향력 전무

광장의 진짜 주인 (Lord of the Square)

  • 하나님은 교회 건물 안에서만 다스리는 반쪽짜리 왕이 아니심
  • 진료실·수술실·시장 바닥의 유일한 주권자 — 십자가에서 이미 우주적 승리 거두심

성경 속 광장 모델

인물 현장 핵심
요셉 이집트 총리 (경제·행정) "하나님이 생명을 구원하려고 나를 먼저 보내셨나이다" — 행정 자체가 구속 사역
다니엘 바빌론·페르시아 요직 (정치·교육) 우상 제물 거절·매일 기도 → 이방 왕들이 하나님을 찬양
느헤미야 성벽 재건 (건축·정치) 묵도 + 세속의 왕에게 계산된 지원 방안 제시 — 탁월함과 거룩함의 균형

의료 현장 = 공적 광장 중의 광장

  • 삶과 죽음이라는 인간의 가장 본질적 이슈가 다루어지는 공간
  • 환자는 의학적 처치를 넘어, 하나님의 긍휼과 위로를 갈구함
  • 진료실 = 하나님 나라의 생명법이 집행되는 거룩한 성문

광장 진입의 입장권: 탁월함(Excellence)

  • 실력 없는 그리스도인은 광장에서 무시당하고 복음의 문을 스스로 닫음
  • 최고의 의학적 전문성(Skill) = 하나님께 올려드리는 선교적 제사
  • 탁월함 + 거룩함 → 세상의 귀를 여는 폭발력

진짜 영적 전장 = 의사 결정 현장

  • 영적 전쟁은 기도원이 아닌 매일의 결정 순간에 벌어짐
  • 무기: 리베이트 단호한 거절, 동료 존중, 윤리적 절대성

소금과 빛의 공적 기능

  • 소금 (방부성·Preservation): 직장 내 부패를 방지, 공의와 윤리 기준 유지
  • 빛 (가이드·Illumination): 혼란한 가치관 속에서 성경적 세계관으로 방향 제시
  • 동화(Surrender) vs 선교(Mission) — 타협이 아닌 광장을 주도적으로 변화시킴

변증적 삶 3단계

  1. 태도의 매력: 성실함과 평안의 신비로움으로 동료의 호기심을 자극
  2. 언어의 번역: 보편적·전문적 언어로 하나님의 진리를 번역
  3. 인내로 증명: 충분히 삶으로 보여준 후, 소망의 이유를 온유하게 고백

비즈니스 애즈 미션 (Business as Mission)

  • 이윤 추구를 넘어 병원 경영 전체가 복음의 통로
  • 환자의 전인적 치유와 존엄성, 직원에 대한 공정한 대우, 투명한 재무 관리
  • 모든 의사 결정에 "하나님이라면 어떻게 하실까?"

고난 = 영광의 무대

  • 공의를 고수할 때 세상의 배척과 조롱이 따름
  • 다니엘의 사자 굴 = 하나님의 살아계심이 가장 극적으로 증명되는 공간
  • "죽으면 죽으리이다"의 기개로 광장의 진실을 지켜냄

새 창조의 재료

  • 고전 15:58: 진료실에서의 선한 일들은 사라지지 않음
  • 환자를 살리고, 정직하게 진단하고, 위로하는 행위 = 새 하늘과 새 땅을 짓는 영원한 재료

14장 — Missional Medicine: 찬송하고 기도하는 백성

  • 핵심 통찰 (존 파이퍼): "선교는 예배가 존재하지 않기 때문에 존재한다"
  • 예배의 이중 역할: 선교의 연료(Fuel) + 선교의 궁극적 목적지(Goal)
  • 진정한 예배자가 될 때 비로소 진정한 선교사가 될 수 있음

찬송하도록 창조된 백성

  • 사 43:21 "이 백성은 내가 나를 위하여 지었나니 나를 찬송하게 하려 함이니라"
  • 인간에게 부여된 가장 고귀한 기능 = 창조주의 영광을 소리 높여 노래하는 것

찬송하도록 구속받은 백성

  • 엡 1장: "그의 영광을 찬송하게 하려 하심이라" — 세 번 반복 강조
  • 찬송 = "우리는 더 이상 죄의 종이 아닌 하나님의 자녀다!"라는 우주적 선포

찬송 = 복음 전파의 수단

  • 벧전 2:9 "그의 기이한 빛에 들어가게 하신 이의 아름다운 덕을 선포"
  • 환난 중에도 기쁨을 잃지 않는 성도의 찬송 = 복음의 매력을 전달하는 매개체

삶의 찬송 (진료실 적용)

  • 환자를 대하는 온유한 태도, 정직한 진료 기록, 지친 동료에 대한 진심 어린 격려
  • 이 모든 것이 진료실에 울려 퍼지는 '삶의 찬송' — 환자들이 하나님의 임재를 느끼게 함

기도의 선교적 의미

  • 살아계신 하나님의 증거: 신 4:7 "기도할 때마다 우리에게 가까이 하신다" — 응답하지 않는 우상들과의 대조
  • 열방 축복의 통로: 시 67편 — "주의 구원을 모든 나라에 알리소서"
  • 우상 숭배 전복의 무기: 세상을 장악한 돈·권력·안락함에 대항하는 영적 다이너마이트
  • 현장 선교사의 생명줄: 엡 6장 — 기도로 선교사를 파송하고 지탱함

제사장적 중보 기도 (진료실 적용)

  • 처방전을 쓰기 전 1초 마음의 기도
  • 위독한 환자의 손을 잡고 조용히 하나님의 긍휼을 구함
  • 인간 의술의 한계에서 "위대한 치료자이신 하나님의 만지심"을 요청하는 겸손

선교의 궁극적 목표: 예배

  • 계 7장: "각 나라와 족속과 백성에서 나온 무리가 보좌 앞에서 구원하심이 하나님께 있도다"
  • 진료실에서 드리는 작은 예배 = 영원한 하늘 예배의 예고편

예배 = 선교사의 생명줄

  • 영적 탈진의 유일한 회복: 하나님과의 깊은 사귐
  • 예배가 멈추면 선교도 멈춤 — 방전된 배터리 비유

기도로 경영하는 병원

  • 모든 위기 앞에서 인간의 계산기보다 하나님의 지혜를 먼저 구함
  • 아침마다 스태프와 함께 기도하는 '기도 공동체' 구축

성경 모델: 다윗의 선교적 예배

  • 시 57편: "만민 중에서 주께 감사하며 뭇 나라 중에서 주를 찬송하리이다"
  • 왕궁 한복판(공적 광장)에서도 열방을 향해 하나님의 영광을 선포한 예배자

15장 — Biblical Medical Mission: 이제까지의 여정과 앞으로의 여정

  • 최종 선언: "당신의 삶 전체가 곧 하나님의 선교입니다! (Life As Mission)"
  • 핵심 질문: "15주간의 공부가 세상을 보는 당신의 '안경'을 어떻게 바꾸어 놓았습니까?"
  • 램스스쿨 수료: 배움의 끝이 아닌 현장의 진짜 시작

성경 이야기의 중심축 = 선교

아브라함의 복 → 이스라엘의 윤리 → 예수님의 구원 복음 → 교회의 증언 = 하나의 선교적 열망(Mission)으로 통합 → 성경은 처음부터 끝까지 철저히 선교적인 책

선교의 세 원리

  1. 급진적 윤리: 하나님의 약속은 '의와 공도'를 실천하는 백성을 통해서만 성취
  2. 단호한 충성: 다원주의 속에서도 그리스도의 유일성 타협 없이 고수
  3. 삶의 증거: 입술의 복음(Proclamation) + 삶의 양식(Witness)이 완전히 일치할 때 설득력 발생

하나님의 '살아있는 광고판'

  • 탁월함과 거룩함의 가시성으로 보이지 않는 하나님을 시각화
  • "내 전문성이 타락한 세상 시스템을 공고히 하는가, 아니면 회복하는가?"

복음의 확신 회복

  • 상대주의·포스트모더니즘 속에서 복음의 진리성을 더욱 확고히
  • 복음은 개인 구원 + 타락한 세상을 변화시킬 유일한 대안

교회의 추문(Scandal)과 회복

  • 추문: 지위 탐욕, 권력 탐닉, 이념 포로, 윤리 타락 = 복음 전진의 장벽
  • "나는 복음의 통로인가, 아니면 복음을 가로막는 벽인가?"
  • 회복의 길: 회개 → 거룩함 우선 회복 → 야웨의 도로 돌아옴

지배 문화에 대한 거룩한 저항

  • 맘몬과 무한 경쟁에 무비판적 동화 거절
  • 정의·긍휼·정직 = 하나님 나라의 대항 문화
  • 역설: 거룩한 저항이 세상에는 가장 압도적인 '매력'으로 다가감

제자도의 회복

  • 복음 전파 + 삶의 변화가 완전히 통합된 제자도
  • 훈련생(Trainee) → 실전가(Practitioner)로 전환
  • 누군가를 제자로 만들려면, 먼저 내가 따르고 싶은 본보기가 되어야 함

땅끝까지의 긴급성

  • 미전도 지역과 소외된 영역을 향한 거룩한 조급함 유지
  • 직접 가거나 / 기도로 보내거나 / 재정으로 후원하는 세 가지 통로

의료인 정체성 재확인

  • "의술을 강력한 도구로 사용하는 선교사"
  • "매일 아침 진료실로 파송되는 하나님의 대사"
  • 의사 면허증 = 하늘 나라가 발급한 영광스러운 파송권

실제 삶의 모델 (램스 수료자 사례)

  • 봉사형: 진료 시간을 줄여 외국인 근로자 무상 진료
  • 경영형: 병원 수익 일부를 선교지 병원 건립에 헌신
  • 연구형: 창조 질서를 세상에 밝혀내고 믿음의 후배 양성

약속: 수고는 헛되지 않음
