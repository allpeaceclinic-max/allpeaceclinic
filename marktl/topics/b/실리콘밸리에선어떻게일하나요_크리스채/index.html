실리콘밸리에선 어떻게 일하나요 — 크리스 채

저자: 크리스 채 (Chris Chae) — 실리콘밸리 프로덕트 매니저
분류: 조직문화 / 리더십 / 업무방식
적용 관심: 누가회(KCMF)·인터서브·선교단체 조직문화 혁신

조직문화란 자연스럽게 생겨난 것도 있지만 목적 달성을 위해 의도적으로 설계될 수 있다.

7가지 실리콘밸리 업무 문화

1. Bottom-up Culture (바틈업 문화)

  • 탑다운 아이디어 = 작은 삼각형 / 바틈업 아이디어 = 넓은 사각형 → 면적 차이
  • 조직의 비전은 탑다운, 그러나 바틈업으로 해결하지 못할 문제는 없다
  • 핵심 조건: 실력과 책임감이 강한 직원을 고용하는 것
  • 메타(Facebook) — 비전에 맞게 조직문화를 세우고 업무 방식을 의도적으로 설계

2. Feedback Culture (피드백 문화)

  • 철학: 모든 생각과 사상은 일단 논의되어야만 발전한다 — 자유로운 발언권
  • 반대의견이 표출될 수 있어야 하고, 과정이 있어야 가치관이 일치
  • 360도 피드백: 직원↔회사, 직원↔제품, 직원↔직원(상사·부하·동료)

피드백 5원칙:

원칙 핵심
투명성 숨기지 않는다
시의성 제때 한다
공정성 편향 없이
친절함 빠지면 받기 어렵다
실행가능성 실제로 바꿀 수 있는 것
  • 피드백은 제안이다 — "약점" 대신 "성장 영역(growth area)"
  • Trust building → Open feedback → Growth
  • "Feedback is a gift, not all feedback is a gift"
  • "직원은 회사를 떠나는 것이 아니라 상사를 떠난다"

✏️ 적용 메모: "누가회를 떠나는 것이 아니라 선배들을 떠난다"

3. Flat Culture (수평 문화)

  • 직책을 세분화하지 않고, 레벨을 공개하지 않는다
  • 리더십 = "매니지한다"보다 "서포트한다"
  • 장점: 사내정치 감소 / 단점: 목소리 큰 사람이 이길 수 있음

⭐ 4. Manage UP — 핵심 인사이트

주도권이 부하직원에게 있다 — 실리콘밸리 문화의 가장 반직관적인 원리

  • 정보 제공 / 도움 요청 / 피드백 요청과 제공
  • 업무 3유형: 미래전략업무 / 주기적인 업무 / 돌발성 업무 → 돌발성 업무가 발생하지 않도록 아래에서 위를 제때 돕는 것
  • 핵심 역량: 자기인식능력
  • 상사가 먼저 Manage UP을 소개 → 기대치 공유 / 1:1 / 상사를 깨운다
  • 더블 매니지업도 좋다 / 상사도 칭찬이 필요하다

교회·선교단체 적용 가능성: - 한국 교회·누가회는 전형적 탑다운 위계 — 아랫사람이 위를 관리한다는 개념 자체가 낯설다 - 그러나 성경적으로: 바울도 디모데·디도에게 "나를 도우라"고 요청(딤후4:9-13) - 예수님도 제자들에게 기도를 요청하셨다(겟세마네, 마26:38) — 리더가 도움을 요청하는 것은 약함이 아니라 공동체의 본질 - Manage UP이 작동하려면: 리더가 먼저 "나를 관리해 달라"고 초대해야 한다 → 이것 자체가 서번트 리더십

5. Parallel Track (병렬 트랙)

트랙 강점
관리자(Manager) 리더십, 팀빌딩
IC 리더(Individual Contributor) 전문성, 일 자체
  • 둘은 파트너십 — 왔다갔다 가능
  • 후행식 승진(promotion is lagging): 실행해보고 결정
  • 경력 많은 부하직원을 둘 수 있게 됨

6. Strength-based Culture (강점 기반 문화)

  • 강점 = 기술 / 문제접근법 / 행위 / 가치관 / 업무스타일
  • 약점: 무시할 수 있는 약점 vs 개선해야 할 약점(경고→해고)
  • 강점 6유형: 비저너리 / 시스템 싱커 / 장인 / PM하이브리드 / 제너럴리스트 / 스페셜리스트
  • 채용 기준: 자기인식이 강한 사람

7. Impact Driven Culture (임팩트 주도 문화)

  • 책임제도: 기대치와 임팩트 → 평가시스템 → 승진/경고/해고
  • 핵심 역량: 의욕 / 주도성 / 객관적 자기인식
  • 기대치가 높으면 번아웃 — 균형 필요
  • 공정한 평가 시스템 + 공정성을 지적할 수 있는 문화

아인슈타인: "나는 55분 동안 문제에 대해 생각하고, 5분 동안 해결책에 대해 생각할 것이다."

⭐ 성경적 바틈업의 패러다임 — 예루살렘 공의회 (행15장)

이 책의 바틈업 문화 + 피드백 문화 + Manage UP이 통합된 성경적 원형:

행15장 예루살렘 공의회의 구조: 1. 현장의 바틈업 피드백이 결정의 핵심 근거가 되었다 - 바울과 바나바가 선교 현장에서 올라와서 "하나님이 이방인 가운데 행하신 일"을 보고 (행15:4,12) - 이 현장 보고가 공의회 결정의 결정적 증거가 됨 — 신학적 논쟁만으로는 결론이 나지 않았다 2. 다양한 계층의 변론이 가능했다 — 바리새파 신자, 사도들, 장로들, 현장 선교사 모두 발언 3. 다툼과 변론이 있었지만 결국 합의에 이름 (행15:7 "많은 변론이 있은 후에") 4. 야고보의 종합과 결론 — 탑다운 리더십이 바틈업 정보를 종합하여 결정

→ 비전(이방인 선교의 방향)은 탑다운이지만, 결정의 근거는 현장의 바틈업 피드백

어떤 조직이든 바틈업을 가능하게 하려면: - 현장의 목소리가 올라갈 통로가 있어야 한다 (행15 — 바울·바나바의 보고 기회) - 반대 의견이 안전하게 표출될 수 있어야 한다 (행15 — 바리새파 신자도 발언) - 리더가 바틈업 정보를 종합할 의지와 역량이 있어야 한다 (야고보의 역할) - 결론은 공동체적 합의로 — "성령과 우리가 결정하였으니" (행15:28)

탐구 중인 질문

Q1. Manage UP 문화를 누가회·교회 조직에 이식할 수 있는가? - 부분 답변: 한국 교회·의료계의 위계문화에서 "상사를 관리한다"는 개념은 도전적 - 부분 답변: 성경적 근거 — 바울-디모데 관계(딤후4), 겟세마네의 예수님(마26:38) - 부분 답변: CMF_VPM철학 — People을 세우는 것 = 아래로부터의 역량 강화 - 부분 답변: "직원은 회사를 떠나는 것이 아니라 상사를 떠난다" → 누가회 젊은 의사 이탈 문제와 직결 - 여전히 열린 질문: 리더가 먼저 "나를 관리해 달라"고 초대하는 것이 가능한가? 첫 단계는?

Q2. 예루살렘 공의회 모델을 현대 선교단체 의사결정에 어떻게 구조화하는가? - 부분 답변: 행15 — 현장(바울·바나바)의 바틈업 보고가 공의회의 결정적 근거 - 부분 답변: 거버넌스강의 — 이사회가 현장 목소리를 듣는 구조 - 부분 답변: 피드백 5원칙(투명성·시의성·공정성·친절함·실행가능성) 적용 가능 - 여전히 열린 질문: 인터서브·누가회 이사회에서 현장 선교사·젊은 의사의 바틈업 피드백이 결정에 반영되는 공식 통로가 있는가?

Q3. 실리콘밸리의 피드백·강점 문화를 선교단체 평가에 적용하면? - 부분 답변: 360도 피드백 → "약점" 대신 "성장 영역"이라는 언어 전환 - 여전히 열린 질문: 누가회·인터서브 정기 평가에서 360도 피드백이 실행 가능한가?
