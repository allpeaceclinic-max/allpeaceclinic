니고데모의 안경

책/강의 개요

항목 내용
핵심 프레임 기독교 세계관 = 창조-타락-구속 (창타구)
강의 시리즈 1강 니고데모의 안경 → 심화편(밤의 질문→낮의 순종) → 2강 니고데모의 왕진가방 → 3강 바울의 안경
대상 CMF 누가생, 의료인
핵심 인물 니고데모 (요3 밤의 질문자 → 요19 낮의 행동가)
핵심 성구 전 3:11 — 영원을 사모하는 마음 / 창 45:5 — 하나님이 보내셨다

1. 세계관이란 무엇인가

두 가지 방향의 정의

방향 설명
View of the world 세계를 바라보는 렌즈 — 현실을 어떻게 해석하는가
View for the world 세계를 향해 나아가는 방향 — 어떻게 살 것인가

자연 → 세계관 → 문화 세계관은 자연(현실)을 해석하여 문화를 만들어낸다

폴 고갱의 질문 (전 3:11)

"우리는 어디서 왔는가? 우리는 무엇인가? 우리는 어디로 가는가?"

  • 전 3:11 — "하나님이 모든 것을 지으시되 때를 따라 아름답게 하셨고 영원을 사모하는 마음을 그들에게 주셨다"
  • 인간 깊은 곳의 질문 = 세계관이 없으면 답할 수 없다

2. ⭐ 창조-타락-구속 (창타구) 프레임

창조 (Creation)

  • 하나님은 Creator → 인간은 Creative (창조성을 위임받음)
  • 문화 명령의 히브리어:
  • 카바쉬 (정복하라): 하나님 형상으로 세상을 채움
  • 라다 (다스리라): 청지기 리더십
  • 아바드 (경작하라): 섬기는 노동
  • 샤마르 (지키라): 보존, 돌봄
  • 관계 구조: 2자 관계(공익, 너와 나) → 3자 관계(공동선, 너와 나 + 하나님)
  • 공익(Public Interest): 너와 나의 이익 합산
  • 공동선(Common Good): 하나님 관점에서의 선

타락 (Fall)

  • 구조(Structure) vs 방향(Direction)
  • 구조는 선하게 창조되었다 (의료, 정치, 경제 — 그 자체는 선하다)
  • 방향이 문제다 — 타락은 구조를 비틀어 하나님이 아닌 곳을 향하게 함
  • 일반은총: 타락 이후에도 구조가 완전히 부서지지 않도록 붙들어 주심
  • 전적 타락: 인간이 전적으로 나쁘다는 것이 아니라, "선한 것을 바르게 행사하는 능력을 상실"했다는 것

구속 (Redemption)

카이퍼: "구속이란 삶의 바퀴를 바로 잡는 것"

  • 이미(Already): 그리스도 안에서 시작된 회복
  • 아직(Not Yet): 완성을 향해 나아가는 긴장
  • Sent Church: 교회는 세상으로 보내진 공동체
  • CMF = 의료사회에 보내진 sent church
  • 요 5:17 "내 아버지께서 이제까지 일하시니 나도 일한다"

3. 니고데모의 변화 — 밤의 질문에서 낮의 순종으로

시점 모습 본문
요 3장 밤에 찾아온 질문자 — "어떻게 거듭날 수 있습니까?" 세계관의 전환 시작
요 7장 바리새인 앞에서 소극적으로 변호 중간 단계
요 19장 낮에 공개적으로 장례 준비 — 100리트라의 몰약 세계관이 삶이 된 순간

니고데모는 죽은 예수를 위해 할 수 있는 가장 비싼 것을 드렸다 — 지식이 아니라 몸으로 드리는 헌신이 세계관 완성의 증거

쯔다카 (Tzedakah)

"내 몫에는 타인의 지분이 포함되어 있다" - 단순한 자선이 아니라, 정의(Justice)로서의 나눔 - 100리트라 = 물질이 세계관의 통제를 받는 구체적 사례

4. 4인물 안경 — 세계관을 산 사람들

요셉 — 소명은 찾는 것이 아니라 알게 되는 것

창 45:5 — "형들이 나를 이곳에 팔았다 하여 근심하지 마소서. 하나님이 생명을 구하시려고 나를 당신들보다 먼저 보내셨나이다"

  • 22년의 침묵과 고통 → 세계관 전환의 긴 터널
  • "형들이 팔았다" (인간 해석) → "하나님이 보내셨다" (세계관 해석)
  • 소명(vocation)은 스스로 찾는 것이 아니라, 하나님이 뒤에서 알게 하시는 것

삭개오 — 총체적 구원

  • 영적 거듭남(예수를 만남) + 직업적·재정적 거듭남(절반을 내놓음) = 총체적 구원
  • 세계관이 직업에 침투한 증거

다니엘 — 거룩한 경계선

"바벨론의 기술(학문)은 배우되, 바벨론의 가치에 동화되지 않는다"

  • 지식과 기술 → 수용
  • 왕의 음식(가치와 윤리) → 거절
  • 거룩한 경계선: 세상 안에 있되, 세상에 속하지 않음

바울 — 거룩의 방향 전환

정체 거룩의 이해 선교 방식
사울 거룩 = 분리 "Come & Be like us"
바울 거룩 = 성육신 "Go & Serve"
  • 아브라함의 소명 구조 (창 18:19): 선택 → 윤리 → 약속
  • 선택받은 이유가 있다 → 그 이유(윤리, 공의와 정의)를 살아야 → 그래야 약속이 이뤄진다

바울의 전도여행 — 막힘을 통한 방향 전환

단계 내용
1차 전도여행 익숙한 땅(갈라디아), 익숙한 언어, 유대 회당 중심
2차 전도여행 — 아시아 성령이 막으심
2차 전도여행 — 비두니아 예수의 영이 허락하지 않으심
2차 전도여행 — 무시아·드로아 지나쳐 드로아에 이름
마게도냐 환상 "건너와서 우리를 도우라" → 유럽 선교의 문 열림

"막힘 자체가 안내판이다" — 길이 닫힐 때 새 방향을 보여주시는 하나님

5. 의료 현장 적용 — 니고데모의 왕진가방

소명의 두 층위 (오스 기니스)

  • 1차 부르심: 하나님께로의 부르심 — 누가 불렀는가
  • 2차 부르심: 직업적 부르심 — 무엇으로 불렀는가
  • 2차가 1차를 대체할 때 소명이 왜곡된다

77 패러다임 — 구조적 문제 진단

한국 의료: 저부담·저수가·저보장 = 77 패러다임

  • 이것은 개인의 도덕 문제가 아니라 구조(방향)의 문제 — 창타구 프레임으로 진단
  • 방향 전환 없이 기술 혁신만으로는 고칠 수 없다

선교 패러다임의 전환

패러다임 동사 설명
오이코도메오 세우다 교회 중심, 안으로 모으는 구조
아포스텔로 보내다 세상 중심, 밖으로 보내는 구조
  • CMF = 아포스텔로 공동체 — 의료 현장으로 보내진 교회
  • 거룩한 스파이 / 방향 전환자: 시스템 안에 들어가 방향을 바꾸는 사람

리미라리티와 커뮤니타스

  • 리미라리티(liminality): 문지방, 경계의 공간
  • 커뮤니타스(communitas): 경계에서 태어나는 진짜 공동체
  • 의료 현장의 위기(의정갈등, 번아웃)가 리미라리티가 될 수 있다

강의 시리즈 구조

``` 1강: 니고데모의 안경 (기본) → 세계관 정의 + 창타구 프레임 + 4인물 안경

심화편: 밤의 질문에서 낮의 순종으로 → 니고데모 변화 + 쯔다카 + 아브라함의 소명 구조 + 5인물 안경

2강: 니고데모의 왕진가방 (의료생태계 편) → 소명의 두 층위 + 77패러다임 + sent church

3강: 바울의 안경 (선교 패러다임 편) → 사울→바울 거룩의 전환 + 전도여행 막힘 내러티브 + 마게도냐 환상 ```

미해결 질문

  • 구조와 방향의 구분은 실제 의료 윤리 딜레마에서 어떻게 적용되는가?
  • 거룩한 경계선은 어떻게 협력과 저항의 균형을 유지하는가?
  • 다니엘의 원칙이 오늘날 의대생/전공의에게는 어떤 구체적 모습인가?
  • 니고데모가 100리트라를 가져온 것이 왜 구속 이후의 삶인가? (쯔다카와의 관계)
