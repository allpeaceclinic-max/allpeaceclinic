잊혀진 교회의 길 (The Forgotten Ways)

저자: 앨런 허쉬 (Alan Hirsch, 2006)
분류: 교회론 / 선교학 / 사도적 운동론

★★ 핵심 명제: 기독론 → 선교학 → 교회론

기독론이 선교학을 결정하고, 선교학이 교회론을 결정한다.

이 한 문장이 책 전체를 관통한다.

그리스도가 누구인가? ↓ 어떻게 선교하는가? ↓ 교회는 어떤 형태여야 하는가?

  • 교회론은 핵심 교리 중 가장 유동적: 역사적으로 유한하고, 문화적으로 특정됨
  • 예배 방법, 사회적 역동성, 예전은 복음을 상황화하는 과정의 결과물
  • 교회는 반드시 선교적 사명을 뒤따라야 한다 — 형태는 사명에서 파생됨
  • 교회 형태를 신성불가침한 것으로 만들면 → 우상숭배

칼 바르트: "선교는 교회가 속해 있는 하나님의 속성이다."
→ 미씨오데이: 선교의 주체는 하나님, 교회는 도구

1. 사도적 천재성 (Apostolic Genius) — mDNA 6가지

초대교회(성경도·전문가도·건물도 없이 성장)와 중국 지하교회(박해 속에 증가)의 공통 비결:
→ mDNA(선교적 DNA) — 구조가 아니라 DNA

# 요소 핵심 내용
1 J is L 예수님이 주님 — 만물의 주, 그리스도 중심의 유일신 (고전8:16)
2 제자도 모든 영역을 주님께 돌려드리는 것; 세상에서 이루어지는 제자도; 교회사역의 문턱↓ 제자 되는 문턱↑
3 선교적-성육신적 추진력 복음은 끊임없이 주소를 바꿔야 한다 (헬무트 틸리케); 파송(원심력)+뿌리내리기(성육신); presence·proximity·prevenience·powerlessness·passion·proclamation
4 사도적 환경 APEST 문화 조성 — 협동심·방향감각·목적의식
5 유기적 조직 교회 개척 X → 운동 O; DNA 기반 재생산 (씨앗→나무→숲)
6 경계성과 커뮤니타스 경계성(안전구역 밖) → 커뮤니타스(확장된 공동체); 빅터 터너

2. 구조 vs 제도주의 — 결정적 구분

구조는 필수다. 제도주의는 병이다.

  • 구조: 협동·사회적 결속 유지에 절대 필요 (생명 시스템 유지)
  • 제도주의: 시간이 지나면서 구조가 목적을 대체 → 하나님 백성의 역할·권한을 중앙기관이 독점

비전·사명 → 프로그램·경영기법(제도화) → 쇠퇴

  • "당신이 체제와 사랑에 빠져버리면, 그것을 변화시킬 능력을 잃는다"
  • 운동저격범(사냥꾼) → 목동이 되려는 방향으로 이동해야
  • 신학교→권위→목사안수 기관→목회자 권위: 이 구조 자체가 제도주의의 산물

네트워크 vs 구조(교단): - 구조(교단): 힘의 중앙집중 → 종교적 제도주의 - 네트워크(로잔): 힘과 책임이 두루 미침 → 참된 예수 운동

3. ★ APEST — 사라진 APE를 찾아라

엡4:11 — 사도·선지자·전도자·목사·교사

은사 역할 주요 사역
Apostolic 사도 운동 확장·개척, DNA 파종과 순수성 보호 교회가 당면한 초지역적·전략적 사안
Prophetic 선지자 하나님 뜻 선포, 사회정의, 분별 음성 경청, 의문 제기
Evangelistic 전도자 복음 확장, 아웃리치 복음 전파
Shepherding 목자 공동체 돌봄 셀 모임, 상담, 예배
Teaching 교사 말씀 교육, 지혜 개발 성경공부, 신학·철학 토론

서구 교회는 ST(목사-교사)만 남기고 APE를 추방했다.

APE가 사라진 결과 → 파라처치 기관으로 흘러나왔다:
네비게이토, 소저너스, 월드비전, 인터서브, 누가회…

이것이 허쉬의 깊은 통찰이다. APE 은사를 가진 사람들이 교회 구조 안에서 자리를 찾지 못하자 파라처치(선교단체·NGO)라는 새로운 그릇을 만들어낸 것이다. 파라처치는 교회의 실패가 아니라 억압된 사도성의 출구였다.

  • 사도적 리더십은 공식 직분이 아닌 기능
  • 목사적 개념이 교회를 선교적 기관으로 재인식하는 데 가장 큰 방해요소 (록스버그)
  • 엡2:20 — 사도와 선지자의 터: 환경과 판단기준을 제공하는 기초적 은사

4. ★ 경계성과 커뮤니타스 (Liminality & Communitas) — 공동체성의 탁월한 설명

커뮤니타스는 안락함에서 태어나지 않는다. 경계에서 태어난다.

빅터 터너(Victor Turner)의 인류학 개념을 교회론에 적용:

중산층 + 소비주의 = 안락함 → 공동체가 아닌 '나를 위한 공동체'로 전락

  • 경계성(Liminality): 안전구역 밖, 기적이 쓰이는 장소, 고난과 위험의 장소
  • 커뮤니타스: 경계성에서 탄생하는 확장된 공동체 (하나님·교회·세상의 공동구역)

△ → ○ → ⓞ 수직구조 → 원 → 원의 경계에서 활발

  • 원의 중심에 있으려 하나, 하나님은 경계(변두리)에서 일하신다
  • 교회가 경계에서 사역할 때 항상 중앙까지 살렸다
  • 성전 → 광장

공동체성 적용: - 프로그램·예배 중심 교회는 안락한 중심을 지향 → 커뮤니타스 불가 - 순례자들의 규범: 경계성이 먼저, 커뮤니타스는 그 결과 - KCMF 누가회, 인터서브 선교 공동체: 본질상 경계성의 공동체 — 커뮤니타스를 지향해야

5. 크리스텐덤 모드 → 선교적 모드

  • 크리스텐덤: 문화가 교회 편 → 끌어모으는 전도법 가능
  • 탈크리스텐덤: 문화의 장벽을 넘어 나가야 — 초대교회·현대 동일

Ecclesia semper reformanda — "교회는 항상 개혁해야 한다"

역사적 선교적 운동 사례: 신약교회, 속사도 시대, 켈트족의 운동, 모라비안 운동, 초기 감리교, 중국 교회, 인디언 하우스 처치 운동

6. 유기적 교회론

교회를 개척하지 말고, 운동을 시작하라!

  • 교회 = 운동의 은유 (건물과 운동은 상반된다)
  • 씨앗 안에 나무, 나무 안에 숲 — DNA 기반의 자기 재생산
  • 세상 인구는 곱하기, 교회는 더하기 (월터 헨릭슨) — 지상대위임령이 어려운 이유
  • Pay it forward ← → Pay it back

강의와의 연결

이상협 강의 — 직접 연결

강의 연결 지점
공동체, 구조와 문화 제도적(딤전3)+유기체적(요15), 구조 변화만으로 부족 → 문화 변혁 필요 = 허쉬의 mDNA 개념
선교적 교회 세미나-삼덕교회 선교적 교회의 정의·필요, 타문화·도시문화·지역사회 선교 = 선교적-성육신적 추진력
교회와 선교, 경운교회 미셔널 패밀리 + 선교적 교회 공동체(확장된 가정) = 커뮤니타스
MBTS 조직신학II/교회행정학 포스트 코로나 교회론, 기업교회 비판 = 제도주의 비판

연결 고리

교회론과의 연결

  • 기독론→선교학→교회론 명제: 교회론은 선교학 이후에 온다
  • APEST 섹션: 서구 교회의 APE 추방 진단
  • 파라처치 기관 출현의 신학적 해석

선교신학과의 연결

  • 성육신적 선교(missional-incarnational): "복음은 끊임없이 주소를 바꿔야"
  • 미씨오데이와 공명: 선교의 주체는 하나님, 교회는 참여

제자훈련과의 연결

  • mDNA 2번 = 제자도
  • "교회사역의 문턱은 낮추고, 제자 되는 문턱은 높이자"
  • 세상에서 이루어지는 제자도 (탈크리스텐덤 맥락)
