비영리단체의 경영

저자: 피터 드러커 (Peter F. Drucker)
분류: 비영리 경영 / 리더십 / 거버넌스

"비영리단체는 사명감으로써 시작된 단체다."

1부. 사명보다 더 중요한 것은 없다

  • 사명 선언문 = 실행 가능한 것이어야 한다
  • Ex. "응급실은 상처 입은 사람들에게 응급질환이 아니라는 확신을 주기 위해 설치된 곳"
  • 사명도 피봇팅 가능하다 — 시대가 변하면 사명의 표현도 변해야
  • 사명 수행의 3가지 요건: 기회의 포착 + 전문지식 + 전심전력의 각오
  • 영리기업: 이익이 성과 기준 / 비영리: 단일한 결정적 성과 기준이 없다 — 종합예술작품 평가
  • 다수의 유권자(선교사·교인·교역자·위원회·헌신자) → 이들을 묶을 수 있는 사명

2부. 지도자의 역할

선출 기준: 무엇을 성취했는가 / 사람의 장점 / 정직 및 됨됨이

지도자의 4가지 채무: 1. 조직의 사람의 질 — 새로운 과제에 철저히 책임 부과 2. 의욕, 능력, 경청 3. 되풀이 소통 — 객관적, 초연하게 4. 가장 나쁜 지도자: 물러났을 때 조직을 파멸 상태에 놓이게 하는 사람

소통의 모델 (맥아더): 참모회의마다 가장 계급이 낮은 장교가 먼저 발언 → 그 누구도 방해 못하도록

하지 말아야 할 4가지: 1. 결정 후 "시간이 없다"며 과정 생략 2. 조직 내의 힘을 두려워하는 것 3. 후계자를 혼자서 지목 4. 칭찬에 인색한 것

3부. 기회에서 실행까지

기회 포착(통계·변화) → 내부 자원 교육 → 실행 (1/3만 따라와도 시작) → 시간이 지나 전체로 확산

선교 적용: 선교의 변화·기회 인식 → 내부 자원(청년) 교육 → 어떤 교육을, 어떤 단체와 → 주변 지역교회로 확장

  • 변화는 위협이 아니라 기회라는 인식이 출발점
  • 행동으로 옮기는 작업은 항상 단기적이어야 한다 (긴 안목 + 단기 행동)
  • 성과평가와 피드백 지속

4부. 기금 개발과 성과 측정

  • 기금 개발: 사람의 가슴에 호소하되, 머리에도 부단히 호소해야
  • "동정하는 마음에서 오는 피곤" → 머리의 납득이 없으면 지속 불가
  • 성과 지표: 출석 인원이냐, 지역사회에 미친 영향이냐
  • Ex. 박물관을 교육기관으로 간주 → 성과 지표가 달라짐
  • 성과는 계획되어야 한다 — 동기가 선하다고 결과가 보장되지 않는다

⭐ 거버넌스 강의와의 연결

이 책의 핵심 개념들이 이사님의 거버넌스 강의안(거버넌스강의)과 직접 대응한다.

드러커 거버넌스 강의안 연결
사명이 최고 권위 "진정한 주인은 하나님의 뜻(목적과 가치)" — 거버넌스의 본질: 권력 아닌 위탁(Trusteeship)
조직의 힘을 두려워하지 말 것 3가지 함정: 이사회의 주인 노릇 / 영적 리더의 독주 / 회원의 이익단체화
후계자 혼자 지목 금지 누가회 거버넌스: 대표가 이사회 투표권 갖는 것이 맞는가?
지도자 선출: 됨됨이와 성취 리처드 포스터: 돈·성을 넘어 파워의 문제
다수의 유권자를 묶는 사명 "나를 행복하게 해주시오" 문화 → 소비자 집단 전락 방지
성과는 계획되어야 한다 거버넌스 목표: 배임 방지 + 효율적 동역 + 구성원 참여 + 그리스도의 다스림
기금 개발: 머리에도 호소 재산권 세미나: 듀딜리전스, 공정한 절차, 회의록 정기 정리

"좁은 문" 전략 (이사님 강의): 이사회 인원보다 가치 이해자 중심으로 구성 → 드러커의 "지도자 선출 기준"과 동일선상

⭐ CMF 적용 (원본 메모)

"(CMF, 사명이 없느냐? 있는데, 사명을 위한 추구·교육·노력·행동들이 없나?
아니면 사명에 대한 인식이 없나? = 사명에 동의 안 하나?)"

드러커의 진단을 CMF에 적용하면: - 사명이 없는 것이 문제가 아니라 → 사명 인식과 동의, 사명을 향한 행동이 문제 - 처방: 사명을 재조명·개정하는 것이 첫 번째 과제 → 그 다음 우선순위 결정

탐구 중인 질문

Q1. 비영리의 "종합예술작품 평가" — 한국 선교단체에 맞는 성과 지표는 무엇인가? - 부분 답변: 거버넌스강의 — 거버넌스 목표: 배임 방지 + 구성원 참여 + 그리스도의 다스림 - 여전히 열린 질문: 인터서브·KCMF·LAMs 각 단체에 맞는 핵심 성과 지표(KPI)는 무엇인가?

Q2. "1/3이 따라와도 시작하라"는 원칙과 공동체적 합의는 어떻게 균형을 이루는가? - 부분 답변: 거버넌스강의 — 하나님과 함께하는 공동체의 의사결정 - 여전히 열린 질문: 선교단체 리더십 전환 시 소수의 선도와 다수의 참여를 어떻게 조율하는가?

관련 페이지

  • 리더십 — 지도자의 역할, 소통, 됨됨이
  • 소명 — 사명감으로 시작된 단체
  • 거버넌스강의 — 재산권 세미나, 이사회 운영, 의사결정 (28개 강의)
  • 인터서브 — 거버넌스 실제 적용 사례 (대표 독점권 문제)
