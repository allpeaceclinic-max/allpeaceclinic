개요

  • 핵심 성경: 시 78:72 — "마음의 완전함으로 기르고, 손의 능숙함으로 지도하였도다"

★ 핵심 명제

바울은 선택받은 리더였다. 디모데는 길러진 리더였다.
하나님은 다양한 경험을 통해 리더를 만들어 가신다.
리더십 = 인격(됨됨이) + 은사(행실)

1. 영적 리더의 조건 (딤전 3장)

딤전 3:1 — 감독의 직분을 얻으려 함 = 선한 일을 사모하는 것

갖추어야 할 것: 1. 가정 (가정을 잘 다스림) 2. 양심적인 인격 3. 절제 4. 교회 밖의 존경 5. 탁월한 교육 능력

빠져야 할 함정: - 자기연민 → 한계 인정 후 멘토를 찾는 것이 옳다 - 리더라는 자의식 → 팀·민주적 의사결정이 필요 - 쓰리G의 함정: Girls(성적 유혹), Gold(돈), Glory(명예)

D.E. 호스트: "그는 그리스도를 기억하게 하기 위해서 잊히는 삶을 살았다"
— 페트릭 펑의 요약

2. 리더십 형성의 4가지 변수

변수 내용
기질 (Temperament) 타고난 성향
환경 (Environment) 선택할 수 없는 상황
경험 (Experience) 다양한 사역·삶의 경험
반응 (Reaction) 경험에 어떻게 반응하느냐

→ 요셉의 예: 사랑받는 아들 → 형들 관리 → 가정총무 → 간수장 조수 → 총리
→ 같은 환경도 반응에 따라 리더를 만들기도, 파괴하기도 한다

3. 클린턴 박사의 리더십 타임라인

성경 인물들의 평균값으로 추출한 리더 형성 경로

1단계: 사역의 기초 (Ministry Foundations)

소단계 기간 내용
주권적 기초 13-20년 인격·성품 형성 — 선택 불가한 환경
리더십 이동 3-6년 사역으로 첫 걸음, 부르심과 순종

2단계: 일반적 사역 (General Ministry)

소단계 기간 내용
잠정적 사역 2-6년 작은 일에 충성, 성품 개발, 진실성 검증
성장하게 되는 사역 6-8년 인격적 성장, 권위 문제 발생, 은사 개발
유능한 사역 2-6년 지속적인 열매

3단계: 집중된 사역 (Focused Ministry)

  • 은사와 숙명 확인 후 집중 → "아니요"를 말할 수 있는 단계
  • 직접 사역 → 간접 사역, 섬기는 자 → 청지기

4단계: 수렴 사역 (Convergent Ministry)

  • 은사·숙명·역할·효과·영향력의 극대화
  • 잔광(Afterglow): 한 걸음 물러나지만 오히려 더 많은 것을 이룬다

4. 팀 리더십

팀 vs 네트워크 vs 그룹

유형 특징
네트워크 비공식 연결 — 협력하나 강력한 공동 목표 없음
그룹 공동 관심사 — 관계 중심 (동문, 운동)
팀 소수, 상호보완, 공동 목표, 헌신, 책임, 정체성 분명

리더가 영향력을 발휘하는 단위는 팀이다

팀에 대한 3가지 오해

  1. 훌륭한 인격이나 성령의 힘으로 자동 된다 (바울과 바나바, 마가 요한 문제)
  2. 언제나 좋다 (고도의 전문 기술·예술가 영역은 예외)
  3. 누구나 팀으로 일할 수 있다

팀 형성 4단계

형성기 (기대) → 폭풍기 (갈등) → 규범기 (규칙) → 완성기

역기능 팀의 5가지 증상

신뢰 부족 / 갈등에 대한 두려움 / 헌신 부족 / 책임 회피 / 결과에 대한 무관심

5. 비전·사명·가치·전략

비전 vs 사명

개념 내용
사명 우리가 해야 할 일 (대위임령 → 계 7:9)
비전 보여주는 최종적 그림 (마틴 루터 킹, 세종대왕)

허드슨 테일러: "여러분이 하려는 일을 이야기하지 말고 여러분이 한 일을 말해 주세요"

핵심가치

가치는 목표도 아니고 과업도 아니다. 가치가 없으면 방식이 혼란에 빠진다.

  • 가치 = 희생을 치르더라도 지키는 일하는 방식·태도
  • 기능: 개인과 조직 조정 / 변화 가능하게 / 과업 지속하게 / 결속력 강화
  • 인터서브의 핵심가치: 둘째 되기, 파트너십

문화의 4층 구조 (로이드 콰스트): 세계관 → 신조 → 가치 → 태도(행동)

HIS — 리더의 3가지 자질 (로잔, 크리스토퍼 라이트)

H I S
Humility (겸손) Integrity (성실) Simplicity (단순)

6. 조직의 생명주기 — PAEI 프레임워크

조직은 기계가 아닌 유기체 시스템이다 (고전 12:12, 27)

역할 의미
P roducer 제작자 — 단기 성과
A dministrator 행정가 — 시스템·효율
E ntrepreneur 기업가 — 비전·변화
I ntegrator 통합가 — 사람·관계

조직 생애 흐름: paEi(구애기) → Paei(유아기) → PaEi(소년기) → pAEi(사춘기) → PAEi(전성기) → PAeI(안정기) → pAeI(귀족기) → _A__(관료기) → 사망

조직이 안정기라고 생각할 때 E(기업가)를 영입하라 — 변화 추구가 리더의 역할

7. 전환기 (Transition) — 변화와 인간 사이

변화는 외부적이지만, 전환기는 내부적 현상이다. 변화 + 인간 = 전환기

전환기의 3단계: 시작 (하던 일을 멈춘다) → 진행 (혼돈의 시간) → 끝 (새로운 일을 시작)

성경적 유형: 애굽 생활 끝 → 광야 40년 → 가나안
→ 인터서브 PIT (Partner in Transit) 프로그램

리더의 목표: 교육·정보·지원을 통해 가능한 한 효과적이고 고통 없는 전환을 돕는 것

8. 거버넌스 구조

구조 내용

이사회 최종 의사결정 기구 — 대표 위의 구조

10. ★ 미해결 질문 (나중에 디벨럽 필요)

  1. 타임라인과 조급함: 클린턴의 타임라인이 보여주듯 리더 형성에는 수십 년이 걸린다. 조급한 선교 단체·교회 환경에서 이 긴 과정을 어떻게 지킬 것인가? → 크라이더의 "끈기 있는 발효" 초기교회의인내와발효_크라이더 → 조급함(긴급성·효율성)이 아비투스 형성의 가장 큰 적 — 티모데 형성의 가장 큰 적이기도 하다

  2. 실패한 후계자 문제: 바울-마가 요한, 사무엘-사울처럼 실패한 계승은 어떻게 다룰 것인가? → 반면 바울-디모데, 모세-여호수아, 엘리야-엘리사는 성공했다 — 차이는 무엇인가?

  3. PAEI와 인터서브: 인터서브 코리아는 현재 어느 생애주기 단계에 있는가? E 영입이 필요한 시점인가?
