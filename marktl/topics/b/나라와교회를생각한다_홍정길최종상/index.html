나라와 교회를 생각한다 — 홍정길·최종상

책 개요

항목 내용
저자 홍정길 목사 (남서울은혜교회), 최종상 선교사
형식 인터뷰·대화 방식
핵심 주제 한국교회의 역사적 기여와 미래, 가정사역, 사회·정치 참여

핵심 인사이트

1. 교회의 시초는 가정 — 가정사역이 모든 사역의 시작

"하나님이 인간을 창조하실 때 어린아이가 아닌 결혼 적령기의 아담과 하와로 만드셨다"

  • 창조 질서의 시작이 가정이다
  • 교회는 가정들의 공동체로 출발했다
  • 따라서 모든 사역의 토대는 가정 회복에 있다

이 명제에서 인생 12단계 전 생애 사역이 파생된다:

결혼예비학교 → 신혼커플학교 → 부부태교학교 → 애착부모학교 → 아동기부모학교 → 사춘기부모학교 → 청년기부모학교(자녀 떠나보내기) → 중년 → 어머니학교 → 아버지학교 → 후반기 → 호스피스학교

→ 교회가 가정의 전 생애 여정을 품을 때, 그것이 곧 선교적 교회의 실체가 된다

2. 자유 — 하나님의 이름이자 선물

"출 3:11-14 — 자유가 있을 때 비로소 창의적이 되며 온전한 믿음을 향유할 수 있다" C.S. 루이스: "하나님마저도 반역할 수 있어야 자유다"

  • 구원 = 자유 (눅 4:18 — 포로에게 자유를)
  • 다른 종교는 복종만 있지만, 기독교에는 자유가 있다
  • 오늘날 평등을 앞세우며 자유를 헌법에서 지우려는 시도는 옳지 않다

→ 자유는 방종이 아니라 자기 결정, 하나님 앞에서의 인격적 응답 능력

3. 개혁의 현장은 나 자신

"개혁하려는 사람들 자신이 먼저 개혁되지 않고서는 그들이 말하는 개혁은 불가능"

  • 좌파·우파 모두에게, 그리고 교회에게 던지는 메시지
  • 내가 개혁되는 만큼 교회도 개혁된다
  • 개혁의 시작점: 나

한국교회의 역사적 기여

  • 사립학교·의료·음악·외교·선교 — 근대화에 대한 교회의 공헌
  • 특별한 은총을 받은 교회 = 동시에 책임이 따른다
  • 부흥 경험을 다음 세대에 전수하는 것의 필요성

사회주의 두 가지 구분

유형 근거 특성
왕정국가 기반 사회주의 유신론적 공동체 책임, 공동선 지향
공산주의 기반 사회주의 무신론적 강제적 평등, 자유 부재

→ 동유럽은 민주화, 동양 국가들은 전체주의로 — 자유의 기반이 달랐기 때문

  • 학생사역·선교·북한사역·구제·장애인사역까지 폭넓은 균형

CMF·의료 선교 연결

가정사역: - 가정이 모든 사역의 시작 → 기독 의료인에게도 가정 건강이 선교의 기반 - 인생 12단계 = 의료인이 커버할 수 있는 전 생애 돌봄 범위와 일치

교회와사회참여: - 자유의 신학 → 구조적 자유 억압에 목소리 낼 근거 - 개혁의 현장은 나 → 사회 비판 전에 내부 개혁 선행
