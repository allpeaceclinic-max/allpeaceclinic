선교지에 어떤 교회를 세울 것인가

분류: 선교 재정 정책 / 교회 개척 전략 / 선교지 재산권
연결 강의: raw/lectures/my강의안/필드 재산권 관리 정책 공청회.md

"선교사가 시작할 때 재정 문제, 선교사 생애주기 끝에 재산권 문제 — 처음과 끝이 돈 문제다." — 헤셀 그레이브

1. 한국 선교 역사와 재정 정책의 변화

6시기 한국 선교 역사

  • 역사적 시기 분류 요소 4가지 → 6시기로 구분
  • 흐름: 협력 → 독자적 사역으로 전환
  • 그 속에서 선교지 예배당 건축 급증 → 독자적·비협력적·물량주의

현지인의 평가

  • 예배당 건축은 반기면서도 간섭은 싫어함
  • → 의존성 조장 + 선교사 군림의 구조

미래 과제

  • 은퇴 선교사 문제 (한국 선교 1주기 마감 → 은퇴자 급증)
  • 선교 자원 발굴 문제 (누구를 보낼 것인가?)
  • BAM을 통한 정직(integrity) — 수단이자 목적
  • 선교 방향 전환 필요: 예배당 건축 → 선교 자원 발굴·은퇴 준비

2. 재산권 문제의 핵심 — 동상이몽

재산권 논쟁의 뿌리는 3자의 서로 다른 인식에 있다:

주체 인식
선교단체 본부 파송 단체에 귀속된 재산
파송(후원)교회 후원교회의 자산
선교사 남다른 애정과 소유 의식 내포

"겉으로는 하나님의 것이라고 말하지만 동상이몽이다." — 김종성

→ 3자간 창조적 긴장: 책무(accountability)를 정립할 때 긴장이 완화됨 (마25:14-30)

3. 원칙의 역사 — 헨리 벤에서 홀러까지

원칙 내용
헨리 벤 3자 원칙 자치(self-governing)·자급(self-supporting)·자전(self-propagating) — "선교의 안락사"
네비우스 정책 한국 교회에 적용된 3자 원칙의 변형
홀러의 4P 원리 Pioneer → Parent → Partner → Participant (우산이론)

이 주제는 항상 불편함을 유발 — 어떤 선교사는 모욕적으로 받아들이기도 함.
그러나 더 좋은 선교를 위해 반드시 짚어야 할 주제.

4. 선교지 재산권 — 개념 정의

선교지 재산: 선교사가 재임 중에 취득한 유·무형의 모든 권리와 재산

구분 내용
유형 재산 동산 + 부동산 (토지·건물)
무형 재산 지적 재산권, 소유권, 이권

청지기 의식: 재산의 소유권은 없고 사용권만 있다는 신분 의식
현실적 형태: 개인 명의 / 법인 명의 / 장기 임대(50-99년 사용권)

5. 선교단체별 정책 비교

단체 핵심 원칙
PCK(총회 선교회) 법인체 명의 / 불가능시 자제 / 부득이 개인이면 각서 제출 / 필리핀 교회와 재산권 포기 협약
GMS 현지 선교부가 주체 / 현지 법인체 / 본부가 최종 정책 입안자
성결교 해외선교 외국인 불가능 → 현지인 명의 or 공동 명의 or 법인체(현지인 51% 이상)
SIM 14개 원칙 / 해당 국가 등록 선교부 명의 / 5만불 이상 국제 총재 승인 / 아프리카 99년 임대
인터서브 국제본부 가이드라인 없음 → NO(국가 사무소) 자체 정책 권고

6. 부정 사례 — 방콕 선교 포럼 보고

  1. 사역 보고와 현지 사역의 차이 → 목사 제명
  2. 현지인 명의 주택 가격 상승 → 현지인이 대금 일부 가로챔
  3. 부동산 사유화
  4. 목적 헌금 개인 사용
  5. 파송 교회와 관계 악화 → 선교사가 관계 단절
  6. 재산권이 현지인 이사들 손에 넘어감

부정행위 주체: 선교사 / 파송단체 리더 / 지역교회 목회자 / 선교 브로커

7. 정직(Integrity)과 선교의 통합

"비자 문제 = 정직의 문제 (인디언 선교의 실패 사례)"

  • BAM(Business as Mission): 정직을 수단이자 목적으로
  • 정직(정의·윤리) = 선교 그 자체 — 총체적 관점으로 봐야
  • 선한영향력-김진수 — ISI(정직·나눔·자립), 소유권 이전이 자립의 핵심

재산권 관리 정책의 핵심 가치:

명확·단순 / 투명 / 융통성(초문화권 적용) / 효율성

청지기 의식의 실천: 사유화 예방 → 공적 선교 실천 → 공공성 유지 → 효율성 극대화

8. 건강한 출구 전략 — 이양(Transfer)

"현지의 의존 문제, 이양의 시점, 사역의 지속성 확보 — 이것이 핵심이다"

  • 현지인 동역자 역량 강화(capacity building) 없이는 이양 불가
  • 자발적·의도적 이양: 172년간 자산을 현지에 이양한 SIM 사례
  • 이양 이후 재산 싸움이 사라짐 — DNA로 정착
  • 이양 후 사유화 문제도 존재 → 역량 강화가 선행되어야

탐구 중인 질문

Q1. 인터서브 코리아의 필드 재산권 정책은 어떻게 수립되어야 하는가? - 부분 답변: NO 자체 정책 권고(국제본부 방침) — 핵심 가치: 청지기·투명성·공공성 - 부분 답변: SIM 14개 원칙 참고 / PCK 필리핀 교회와의 협약 모델 - 여전히 열린 질문: 인터서브 코리아의 단계별 이양 로드맵은?

Q2. "선교 재정 정책 교육"은 선교사 생애주기 어느 시점에서 반복되어야 하는가? - 부분 답변: 훈련 시에 배우지만 현장에서 잊어버림 → 정기적 외부 입력 교육 필요 - 부분 답변: 선교적리더십의양성 — 콜브 순환으로 재정 교육 설계 - 여전히 열린 질문: 인터서브 연례 모임에 재정 책무 교육을 어떻게 통합할 것인가?

9. 함선네 세미나 — 교회론 적용과 실천 토론 (2025)

발제자: 김한성 | 2025년 대구 함선네 세미나

실천의 신학 체계

성경 → 신학 → 선교학 → 전략 → 실천

  • 예배당 건축·교회 개척은 실천 영역
  • 생각과 실천이 일치하는가가 핵심 질문

교회론의 역사적 변화

시대 핵심
313년 공인 이후 예배당 건축 — 왕·성주가 지음
종교개혁 참된 교회 추구, 설교+성례전 강조
아나뱁티스트(재세례파) 자유교회 운동, 믿는 자들의 공동체
칼빈-웨슬리-밀라드 에릭슨 믿는 사람 강조, 현실은 여전히 교회=건물

→ 현실: 전도 = 예수님 전하는 것보다 교회 구성원 만들기로 축소

한국교회 압축성장의 문제

  • 압축 성장 경험을 무비판적으로 선교지에 적용
  • 온정주의: 사랑이지만 의존성 조장
  • 자문화우월주의: "다른 나라도 가난해서 못 짓는다?" — 이중 잣대
  • 통계: 한국 장로교 1500개 교회 중 선교부 도움 받은 교회 20개 미만 — 스스로 건축
