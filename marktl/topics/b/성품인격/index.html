성품, 인격

분류: 개념 정리 / 기독교 인간론
형태: 메모 (출처 미확인)

1. 어원 — 단어의 역사

단어 어원 의미
Character (성격) 라틴어 — 금속 인장(도장) 낙인 찍힌 것, 지워지지 않음 → 개인의 고유 특성
Personality (인격·인성) 라틴어 페르조나 (persona) = 가면 남에게 주는 인상으로서의 성격
  • Character: 내면에 새겨진 것, 본질적·지속적
  • Personality: 외면에 드러나는 것, 관계적·인상적

2. 성품 vs 품성 — 신학적 구분

한국어에서 혼용되지만 구분해서 사용해야 한다:

용어 범위 내용
성품 양면 포함 의의 성품 + 죄의 성품 (엡2:3 — "본질상" 진노의 자녀, 헬: 휘시스)
품성 성장적 측면 성품의 성장·개발 가능한 부분 / 훈련되고 다듬어질 수 있음

품성: 품격과 성질을 아울러 이르는 말
일반적으로 사람과의 관계 속에서 구체적인 말·행동·태도를 통해 표현됨

신의 성품에 참여 (벧후1:4): - 본질상 최고의 품성 → "성품"이라고 표현해도 무방 - 그리스도인의 목표: 죄의 성품을 버리고 신의 성품에 참여하는 것

3. 헬라어 — 휘시스 (φύσις)

  • 의미: 본래, 본성, 자연적 특성
  • 성경 사용: 성품(nature) = 18회
  • soul / spirit / nature (벧전1:4)
  • 엡2:3: "본질상(휘시스) 진노의 자녀" → 타락한 본성
  • 벧후1:4: "신의 성품(휘시스)에 참여" → 회복된 본성

타락한 휘시스 (엡2:3) ← 성화의 과정 → 신적 휘시스 참여 (벧후1:4) 죄의 성품(품성) 하나님의 성품(품성)

⭐ vault 내 연결 지점

이 책의 개념 연결 자료
품성 = 훈련되고 다듬어지는 것 사람은어떻게변화되는가_레인트립 — 복음의 3시제, 외면주의 극복
죄의 성품(휘시스) → 신의 성품 참여 구원론 — 성화, 칭의·성화·영화
관계 속 품성 표현 산상수훈 — 팔복: 심령의 가난·온유·긍휼·청결

탐구 중인 질문

Q1. "성품은 타고나는가, 만들어지는가?" - 부분 답변: 어원상 Character = 새겨진 것 (타고남의 함의), 하지만 품성은 훈련 가능 - 부분 답변: 사람은어떻게변화되는가_레인트립 — 복음이 변화의 능력, 아비투스는 형성됨 - 부분 답변: 신의 성품 참여(벧후1:4) — 은혜로 주어지나 믿음·덕·지식…(벧후1:5-7)으로 더함 - 여전히 열린 질문: 의료 현장에서 기독 의사의 품성 형성을 어떻게 훈련할 것인가?

Q2. Personality(페르소나=가면)와 Character(본질)의 불일치는 어떻게 극복되는가? - 부분 답변: 산상수훈 — "마음의 청결" = 외식의 반대인 순수, 두 마음을 하나로 - 부분 답변: 사람은어떻게변화되는가_레인트립 — 외면주의 = 가면 문제, 마음의 싸움 - 여전히 열린 질문: 선교사·목회자의 공적 페르소나와 사적 품성의 통합은?
