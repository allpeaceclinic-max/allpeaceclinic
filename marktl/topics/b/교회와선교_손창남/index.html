교회와 선교

건강한 선교를 고민하는 교회들을 위한 가이드북 — 손창남

책 개요

항목 내용
저자 손창남 선교사 (인터서브)
핵심 주장 선교는 소달리티-모달리티의 건강한 협력으로, 책무 구조 위에서 이루어진다
실용 목적 선교위원회 운영 매뉴얼 + 선교적 교회 로드맵

핵심 인사이트

★ 개신교가 잃어버린 소달리티

"개신교는 종교개혁 과정에서 수도회에 해당하는 소달리티 구조를 가져오지 않았다."

  • 가톨릭: 교구(모달리티) + 수도회(소달리티) → 두 구조의 협력으로 선교
  • 개신교: 종교개혁 때 수도회를 해체 → 소달리티 없이 300년
  • 18세기 말이 되어서야 자발적 선교 구조가 등장:
  • 윌리엄 캐리 → 해안 선교
  • 허드슨 테일러 → 내지 선교
  • 카메룬 타운젠드 → 종족 선교 (위클리프)

한국 교회가 선교를 '교회가 직접 하는 것'으로만 보는 경향도
이 소달리티 부재의 역사적 맥락에서 이해할 수 있다.

1. 선교사의 두 가지 책무

"타문화 사역을 하는 사람으로서 사역적·재정적 책무를 이행하는 사람"

사역적 책무

  • 필드 사역 4년 + 본국 사역(Home Assignment) 1년
  • "안식년(furlough)"은 군대 용어 → 더 이상 맞지 않는 표현
  • 첫 달: 아무 사역 없이 쉼 / 귀임 전 한 달: 준비
  • 선교사 자신이 사역지를 결정하지 않음 → 필드 리더의 전략에 따라 배치

재정적 책무

  • 개인 통장 후원 금지 → 전부 선교 본부 경유
  • 강사료·직업 소득도 보고 대상

"요리사와 요리하는 사람" — 책임의 소재가 다르다

2. 소달리티(Sodality)와 모달리티(Modality)

안디옥 교회의 두 모델 (행 11-13장)

구조 설명 현대
모달리티 목양 중심, 지역 교회 파송 교회
소달리티 복음 확산 중심, 선교 구조 선교 단체
  • 행 11:20 — 흩어진 사람들: 자발적 증인 (모달리티적 흩어짐)
  • 행 13장 — 바나바·바울 파송: 조직적 구조 (소달리티)
  • 바나바 — 구브로 출신 해외 동포 / 바울 — 다소 출신 디아스포라

2000년 교회 역사에서 소달리티-모달리티가 시너지를 낼 때 선교가 가장 활발했다

3자 관계의 건강한 구조

불완전: 파송교회 ← 선교단체(홈/필드) ← 선교사 → 현지교회 건강한: 파송교회 ← 선교단체(홈/필드 ∪ 선교사) → 현지교회

협력 관계 4분면

건강한 교회 건강하지 않은 교회
건강한 단체 이상적 파트너십 갈등적 파트너십
건강하지 않은 단체 피해의식 파트너십 없어야 할 파트너십

3. 필드 구조

  • 인사·행정 / 언어·문화 / 사역 코디네이터 / MK 코디네이터 / 메디컬 어드바이저
  • 처음부터 출구 전략을 계획
  • 감독과 돌봄의 구조 없이 필드 사역 시작 금지

4. 신임 선교사 허입 4단계

  1. 홈 클리어런스 — 됨됨이·소명 확인, 추천서, 지역 교회 역할
  2. 필드 웰컴 — 필드의 요청(매치메이킹)이 있어야 파송
  3. 재정 확인 — 예산 85% 확보 시 파송 준비 완료
  4. 의료 확인 — 정신건강 포함

"선교사 허입 과정은 하나님의 뜻을 알아가는 과정이다"

5. 선교적 교회 로드맵

Doing Missions → Being Missional

단계 내용
문제 인식 통합적 선교 이해 부족, 이원론, 개교회주의, 삶과 선교의 괴리
결정적 성공 요소 리더 그룹의 선교 자각, 이원론 극복, 디아스포라 인식, 파트너십
주요 결과 영역 설교가 선교적으로 / 직업 속 증거 / 균형 잡힌 재정 / 타민족 교회 협력

6. 관점의 변화

  • 성경적: 시 67편, 출 19:5-6 — 온 세계가 하나님께 속함
  • 역사적: 헬라어 신약·로마 도로망·유대인 디아스포라 → 모라비안 → 개신교 200년
  • 문화적: 자문화 중심주의 → 수신자 중심 커뮤니케이션 → 상황화
  • 전략적: 2차 대전 이후 비자 제한 → 총체적 복음 → 교회 개척 운동

종족 무지

"동방박사들이 서쪽으로 온 것은 유대인들이 동쪽으로 가지 않았기 때문이다"

  • 대구 근처 인도네시아인 300명 — 내 눈 앞에 펼쳐진 추수할 밭
  • 하나님은 타문화 사람들을 우리 이웃으로 허락하셨다

핵심 결론

  1. 개신교가 종교개혁에서 소달리티를 잃었고, 선교 회복에 300년이 걸렸다
  2. 소달리티-모달리티 협력이 역사적으로 선교를 가장 활성화시킴
  3. 선교사의 책무 구조(사역·재정)가 건강한 선교의 기초
  4. 선교적 교회는 Doing → Being의 전환을 요구한다
  5. 종족 무지를 깨야 한다 — 먼 곳이 아니라 지금 이웃
  6. "선교적 삶이란 불편하게 사는 삶이지만 가장 즐거운 삶이다"

CMF·의료 선교 연결

CMF_VPM철학 — CMF = 의료계 안의 소달리티: - CMF는 지역 교회(모달리티)가 하지 못하는 의료인 특화 선교 구조 - VPM의 책무 구조(사역·재정 책무)와 소달리티 구조는 같은 원리 - CMF가 건강하려면 지역 교회와 건강한 4분면 파트너십을 맺어야 한다

기독의료인정체성: - 필드 구조의 메디컬 어드바이저 — CMF 의료인이 역할을 맡을 수 있는 자리 - 허입 의료 확인(정신건강 포함) → CMF 멘탈 헬스 사역과 연결 - 의사 자신도 선교사 정체성 5타입 중 하나에 속한다
