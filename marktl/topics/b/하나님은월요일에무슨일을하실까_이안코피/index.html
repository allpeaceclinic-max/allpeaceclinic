하나님은 월요일에 무슨 일을 하실까?

개요

  • 저자: 이안 코피 (Ian Coffey)
  • 구성: 9개 챕터
  • 출처: raw 파일 2개 통합 — 독서모임 토론 노트 + 책소개 영상 스크립트 (이상협)
  • 핵심 물음: 주일 오전 11시와 월요일 오전 11시가 같은 사람인가?

★ 핵심 명제

신앙과 일의 이원론에서 벗어나라.
일은 하나님의 창조 선물이다.
제자도는 교회 안에서만이 아니라 월요일 일터에서 완성된다.

1. 월요일 오전 11시 테스트 (마크 그린)

마크 그린 (런던 현대기독교 연구소 소장):
"주일 오전 11시에 교회에서 배우는 내용이
월요일 오전 11시에 자신이 하는 일과 상관이 없다면, 뭔가 잘못되고 있다."

바꾸어 말하면:

"당신은 주일 오전 11시만큼, 월요일 오전 11시가 즐겁습니까?"

→ 이 질문에 "No"라면 성속 이원론이 작동하고 있는 것이다.

2. 9개 챕터 핵심 요약

1장 — 일: 저주인가 축복인가?

  • 일이 형벌 → 빨리 끝내고 싶다
  • 일이 생존 → 적당히 하고 싶다
  • 일이 축복 → 잘하고 싶다, 더 하고 싶다

성경적 근거: - 하나님은 창조의 일을 하셨고, 지금도 세상을 운행하신다 - 우리는 하나님의 형상을 따라 지음받은 존재 → 일은 창조 위임 - 창 1:31 — "보시기에 심히 좋았더라" → 만족감·성취감이 하나님의 형상 - 전 3:22 — "사람이 자기 일에 즐거워하는 것보다 더 나은 것이 없음을 보았나니"

"번성하라, 쓸모 있는 존재가 되라" — 창세기의 명령
일을 저주가 아닌 축복으로 대하면 거기에 계신 하나님을 보게 된다

2장 — 하나님은 월요일에 무슨 일을 하실까?

일터에서의 3가지 질문:

질문 내용
상관이 누구인가? 최종 보스는 예수님 — 그러나 현실 상관에게도 주께 하듯 (골3:23)
직무가 무엇인가? 직업은 부르심 — 직종과 무관하게 자녀·제자로 부르심
어떻게 수행하는가? 순종 · 정직 · 성실 (상관일 경우: + 공평)

일하러 갈 때 하나님과 따로 출근하는 것이 문제다.
제자도는 교회에서만 지켜지는 것이 아니다 — 삶의 전 영역이다

3장 — 내일이 너무 싫은 걸 어떡해!

다니엘 모델 — 마음지킴 + 습관지킴: - 원치 않는 장소에 살았다 - 섬기고 싶지 않은 사람을 위해 일했다 - 원하지 않는 일을 수행했다

→ 확신의 근거: 하나님께서 직접 나를 보내셨다는 확신

실천 본문 내용
마음지킴 잠 4:23 마음을 지키는 것이 생명의 근원
습관지킴 단 1:8 뜻을 정하여 — 경계선 설정, 수시로 점검

일하기 싫을 때 어려움을 글로 적어보라 →
인격적 문제인지 / 관계 문제인지 / 욕구불만인지 분별 가능
→ 근본적으로 다른 부르심을 찾아야 하는지, 내가 돌파해야 할 문제인지 알게 된다

4장 — 자전거에 올라타라

  • 게으름 ↔ 일중독: 균형이 필요하다
  • 일을 제대로 하는 본보기(멘토) 가 필요하다
  • 꼰대 vs 멘토: 물어볼 때 이야기하면 멘토, 묻지 않는데 이야기하면 꼰대

5장 — 인테그리티

인테그리티(진정성·정직성)가 처녀성과 같다 — 일단 잃으면 되찾을 수 없다

  • 탁월함이 먼저가 아니라 인테그리티가 탁월함의 근거
  • 창 39:2 — "여호와께서 요셉과 함께 하시므로 그가 형통한 자가 되었고"
  • 하나님이 함께 하시기 위해서는 거룩하고 정직할 수밖에 없다
  • 성공 = 여호와께서 함께 하심 — 인테그리티가 그 통로

요셉 모델 — 복의 통로:

요셉이 있는 곳 복 받은 자 본문
보디발의 집 보디발 창 39:5
감옥 간수장 창 39:23
애굽 바로 창 47:7,10
라반의 집 라반 창 30:27-30

보디발의 감옥 = 경제학교 + 정치학교
타락한 세상에서 공의롭게 살려 할 때 안전을 보장해주지 않는다
중간 결말이 어렵다 — 그러나 결국 하나님의 뜻 (창 45:5)

인테그리티를 잃었다면 — 리셋의 신학

인테그리티 = 처녀성이라면, 직장을 옮기는 것이 리셋인가?
새술은 새부대 (마 9:17) — 언제 적용되는가?

핵심 구분:

상황 의미 처방
내가 인테그리티를 잃음 성품의 훼손 직장을 옮겨도 가져간다 — 리셋 불가
인테그리티를 지킬 수 없는 환경 구조적 압박 "새술은 새부대" 적용 가능

→ 직장 이동은 인테그리티 회복이 아니다.
→ 그러나 인테그리티를 구조적으로 불가능하게 만드는 환경이라면, 이동은 선택지가 된다.

"새술은 새부대"의 적용 기준 — 언제 떠날 것인가?

  1. 이 환경이 나를 구조적으로 타협하게 만드는가? (단순 어려움 vs 구조적 강제)
  2. 나에게 아직 요셉처럼 "복의 통로"가 될 가능성이 남아 있는가?
  3. 하나님이 나를 여기 보내셨다는 확신이 남아 있는가?
  4. 내가 떠나는 것이 FROM(도피)인가, TO(소명)인가?

다니엘은 선택권이 없었다. 우리는 있다.
선택권이 있다는 것이 오히려 더 어렵다.

성경적 모델 비교:

모델 선택 의미
다니엘 원치 않는 곳에서 마음·습관으로 버팀 머물기의 모델
아모스 목자·농부로 살다 소명 받아 이동 새부대의 모델
요셉 선택권 없이 감옥까지 중간 결말을 견디는 모델

6장 — 전환기 (내 나이 예순넷이 되어도)

  • 직업 전환기는 있어도 그리스도 안에서 은퇴는 없다
  • 전환기마다 하나님 나라의 관점에서 결정하고 예비하신 하나님을 쫓아가는 것

7장 — 친구의 도움을 받아

  • 같은 직종의 그리스도인 공동체 필요 → 조찬모임
  • 계명누가회 조찬모임 15년 = 이 챕터의 실천 모델
  • 아쉬운 점: 피상적인 나눔에 머물기 쉽다 → 무슨 일을 하고, 무엇을 즐기고, 무엇이 힘든지 실질적으로 나눠야 한다

허드슨 테일러 — 개와 고양이 테스트:
"만일 당신의 아버지, 어머니, 형제, 자매, 집에 있는 개와 고양이가
당신이 그리스도인이기 때문에 더 행복하지 않다면,
당신이 진정한 신자인지가 의심스럽다."

8장 — 돈의 유혹

  • 재물 얻을 능력은 하나님이 주셨다 — 그러나 자족을 배워야 한다
  • 하나님 없이 일 → 괴로움 / 하나님과 함께 일 → 성취감·만족감
  • 전 2:21 — 수고하여 얻은 것을 수고하지 않은 자에게 넘겨줌 = 헛됨

9장 — 은사와 일상

  • 은사는 주일에만 사용하지 말라 — 직장에서도 작동해야 한다
  • 아모스의 모델: "나는 목자요 뽕나무를 재배하는 자로서" → 일상에서 부르심을 받음
  • 열린 자세 + 기회 포착 + 순종 = 은사가 일상에서 작동하는 방식

3. 임마누엘 신학 — 일터 영성의 핵심

"창세기부터 임마누엘이다. 성탄절에만 임마누엘이 아니다."

  • 보디발의 집, 감옥, 바로의 궁 — 어느 일터에서나 하나님이 함께 하셨다
  • 일터에서의 임마누엘이 곧 영성
  • 상관 = 하나님의 현존 / 아내 = 하나님의 현존

4. 독서모임 나눔 — 의료인의 일터 적용

(계명누가회 독서모임 실제 나눔 발췌)

공통 어려움: - 월요일 오전이 가장 힘들다 (통제 안 되는 일들에 대한 불안) - 까다로운 상관에게 "주께 하듯" 하기 어렵다 (벧전 2:18 / 골 2:23) - 봉직의 마음과 오너의 마음 차이 → 갈등

실천 사례: - 월요일을 "선교지에 가는 마음"으로 출근 (박문호, 소아과) - 직원 전도 — 삶으로 보여주는 것 (남계윤, 요양병원 80명) - 어려운 수술 전 기도하는 시간 (김대동, 입원 전담의)

핵심 통찰:

순종했을 때 성취가 연결된다.
점점 "주님과 함께 한다"는 것을, 순종할 때 주님께서 일하신다는 것을 경험한다.
일터가 훈련터다 — 요셉과 다니엘처럼

5. LAMs·직장선교 연결

LAMs

LAMs (Life as Mission)의 선언과 이 책의 핵심 명제가 일치한다:
"당신의 삶 자체가 하나님의 선교입니다"

이 책의 언어 LAMs의 언어
월요일 제자도 Life as Mission
성속 이원론 극복 Being before Doing
일터 = 하나님 나라 현장 공적 광장 (13과)
인테그리티 = 성공의 통로 쩨다카·미쉬파트 (5과)
은사가 일상에서 작동 각자의 직능으로 선교

이상협 원장의 제안: "LAMs → HAMs" — 의료인 일과 신앙의 통합을 후배들과 함께,
러닝커뮤니티 형성으로 구체화

7. ★ 미해결 질문

  1. LAMs → HAMs: 의료인 맥락에서 일과 신앙의 통합을 후배들과 함께 훈련하는 구체적 플랫폼이 될 수 있는가? 계명누가회 조찬모임이 피상성을 넘어 실질적 무브먼트가 되려면 무엇이 필요한가?

  2. 중간 결말의 신학: 요셉이 감옥에 갔듯, 인테그리티로 살아도 불이익을 당할 수 있다. 그것을 어떻게 신학적으로 품을 것인가?
    → "당신들보다 먼저 보내셨다" (창 45:5) = 섭리론

  3. 인테그리티 리셋의 신학: 인테그리티는 처녀성과 같아 잃으면 되찾을 수 없다. 그렇다면 직장을 옮기는 것(새술은 새부대)은 의미 있는 선택인가?
    → 단순 어려움과 구조적 강제를 어떻게 구분하는가?
    → 나는 FROM(도피)으로 떠나는가, TO(소명)으로 가는가?
    → 다니엘(머물기) / 아모스(새부대) / 요셉(견디기) — 나는 지금 어느 모델인가?

  4. 개인 vs 공동체: "개인적 핸들링이 아니라 공동체적 무브먼트"가 필요하다. 일터 신앙을 혼자 감당하지 않으려면 어떤 공동체 구조가 필요한가?
