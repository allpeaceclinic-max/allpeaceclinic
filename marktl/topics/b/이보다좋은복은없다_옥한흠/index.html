이보다 좋은 복은 없다 — 옥한흠

2013.5.23 목요제자반 독후감 — 이상협 집사

때를 따라 돕는 은혜, 마음과 생각을 지키시는 하나님의 평강 — 이것이 이 땅에서 행복자로 살아갈 힘이다.

에베소서 1장을 중심으로 한 구원의 복 묵상.

→ 같은 날 제출: 예수가선택한십자가_맥스루케이도 (맥스 루케이도) 루케이도 = 십자가에 담긴 사랑 묵상 / 옥한흠 = 하늘의 복을 누리는 행복자

구원의 4중 복 (엡 1장)

1. 예정과 선택 — 창세전의 계획

"내 형질이 이루어지기 전에 주께서 하신 일"

  • 내가 조금 잘해서 택하신 것이 아니다
  • 하나님의 완벽한 계획 속에서, 기뻐하시며 계획하셨다
  • "나의 무엇이 좋으셨나요?" — 감히 여쭙지 않겠다는 고백

2. 예수님의 보증 — 나의 어떠함은 문제 없다

예수님께서 보증하셨기에 나의 어떠함이 전혀 문제 되지 않는다.

  • 연약함·의심·걱정이 여전히 있지만 — 보증자가 계신다

3. 성령의 인치심 — 이보다 더 큰 약속은 없다

  • 예정(성부) + 보증(성자) + 인치심(성령) = 삼위일체의 구원 보장

4. 영광스러운 기업 — 상속자의 위치

  • 영원한 기업·상급·영화 → 믿음의 느보산에서 바라보기

★ 점진적 변화 + 상호협력의 원칙

성화는 나의 역할·책임에만 의존하는 것이 아니라, 상호협력의 원칙을 통해 성령님께서 내 안에 이루어 가시는 일이다.

  • 연약함으로 인한 의심·걱정을 아시는 하나님
  • 점진적 변화 → 성령님의 사역 + 나의 책임의 시너지

자리바꿈 — 의인 되기의 근거

예수님의 자리바꿈 → 나를 의인으로, 하나님의 의가 되게 하셨다.

  • 이 자리바꿈에 대한 정당한 반응 = 늘 감사

★ 핵심 통찰 (이상협, 2013)

하늘의 복 = 이 땅의 행복의 기초. 세상의 기준이 아닌 하나님의 기준으로 살 때 진정한 은혜와 평강이 있다.

  • 바울은 감옥에서 에베소서를 썼다 → 하늘의 복을 아는 자는 상황이 행복을 결정하지 않는다
  • 그리스도인으로서의 자의식 — "나는 성도이며 신실한 자"
  • 하나님의 사랑에 사로잡히는 것 = 삶의 목표

연결된 페이지

  • 구원론 — 예정·선택·칭의·성화
  • 성화 — 점진적 변화, 상호협력 원칙
  • 예수가선택한십자가_맥스루케이도 — 같은 날, 사랑 묵상의 쌍
  • 옥한흠 — 제자훈련 사역, 에베소서 해설
