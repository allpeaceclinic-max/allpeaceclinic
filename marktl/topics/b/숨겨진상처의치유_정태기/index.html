숨겨진 상처의 치유 — 정태기

저자: 정태기 목사 (충현교회 원로목사, 기독교 상담·내면치유)
분류: 내면치유 / 기독교 상담 / 영성
성경 틀: 씨 뿌리는 자 비유 (마13:3-9)

네 가지 토양 = 네 가지 마음 상태 = 내면 상처의 네 가지 유형

1. 성경 틀 — 씨 뿌리는 자 비유

길가 → 폐쇄된 자아중심성 (딱딱하고 굳은 마음) 돌밭 → 억압된 내면의 분노 (한이 박혀 있는 마음) 가시떨기 → 허위적 체면의식 (돈·명예·권력에 빼앗긴 마음) 옥토 → 가면을 벗은 어린아이의 심성 (치유된 마음)

2. 폐쇄된 자아중심성 (길가)

  • 길바닥처럼 굳은 마음 — "바늘로 찔러도 피 한 방울 안 나올 인간"
  • 편견, 자기 틀에 맞추려 함, 융통성·여유 전혀 없음
  • 자라온 문화의 틀 안에서 벗어나지 못하는 것이 근본 원인
  • 바리새인처럼 "나사렛에서 무슨 선한 것이 날 수 있느냐" — 지역·계층·문화 편견

3. 억압된 내면의 분노 (돌밭) — 한(恨)

  • 돌덩어리가 박혀 있는 마음 = 한(恨) = 분노의 앙금
  • 한의 신체적 결과: 암 발병(큰 상처 3~8년 후), 각종 심신 질환
  • 잘못된 문화 전통은 머리에 돌덩어리를 이고 가는 것 — 허리가 휨

삭개오 해석 (한승렬 교수): - 뽕나무에 오른 이유 = 키가 작아서가 아니라 마음의 병 때문 - 예수님을 가까이 대면할 용기가 없어서 → 소극적 행동

한의 증상: 심리 불안 / 죄책감 / 불면증 / 무기력 / 식욕 극단 / 일중독 / 쾌락 추구 / 성격 급함 / 병적 공포

해결: "내게로 와서 짐을 벗어버리라" — 예수님이 한의 짐을 받으신다

4. 허위적 체면의식 (가시떨기)

  • 가시덤불 = 내 신경을 온통 빼앗아가는 것 — 돈·명예·권력
  • 근심·걱정·불안·시기·질투로 에너지 낭비
  • "사촌이 논을 사면 배가 아프다" — 비교와 경쟁의 심리
  • 결국 씨(복음·변화)가 자랄 공간이 없어짐

5. 가면을 벗은 어린아이의 심성 (옥토) — 치유의 목표

어린아이의 마음 = 상처가 치유된 상태의 특징:

특성 내용
염려 없음 내일 일을 염려하지 않음
선입견 없음 폐쇄된 자아중심성이 없음
티 없는 웃음 웃음이 치유 — 놀만 커슨 교수(불치병 → 웃음으로 회복, UCLA·스탠포드·존스홉킨스 연구)
티 없는 울음 정화·카타르시스
티 없이 논다 놀이와 휴식
가면 없음 진정한 자아로 삶

옥토의 심성 특징: 악의 없는 유머 / 사생활을 즐길 줄 앎 / 터놓을 수 있는 친구 / 인간의 현실을 직시 / 유행에 휘말리지 않음 / 민주적 사고·도덕성·윤리감·사회공동체 의식

탐구 중인 질문

Q1. 한(恨)은 한국 문화 특유의 상처 개념인가, 보편적인가? - 부분 답변: 몸이아니라고말할때_게이버메이트 — 억압된 감정이 신체 질환을 낳는다는 서구 심신의학의 동일한 발견 - 부분 답변: 분노의 앙금 = 억압된 내면 갈등 → 문화를 초월한 현상 - 여전히 열린 질문: 한국인 의료 환자 상담에서 "한"을 어떻게 다루는 것이 복음적인가?

Q2. "가면을 벗은 어린아이의 심성"은 어떻게 훈련·형성되는가? - 부분 답변: 사람은어떻게변화되는가_레인트립 — 복음의 3시제, 성령의 조명으로 마음 내부에서 변화 - 부분 답변: 웃음·울음·놀이 = 치유의 실천 → 결혼과가족상담 카타르시스 개념 - 여전히 열린 질문: 클리닉 환자·의료인 모두에게 "옥토의 심성"을 회복하게 하는 영성 훈련은?
