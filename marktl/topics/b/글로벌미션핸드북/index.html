부르심을 받는 네 가지 길

  1. 신비를 통한 부르심 — 베드로-고넬료, 바울-마게도니아, 이사야, 요나 → 통상적이지 않으므로 주의
  2. 위임을 통한 부르심 — 행 13장 안디옥 교회 → 장로나 교회 리더로부터 은사가 있다는 말을 가볍게 여기지 말라
  3. 상식을 통한 부르심 — 빌립이 에티오피아 내시를 만났을 때 (행 8장) → 지속적으로 만나는 사람, 독서, 메일을 통해서도
  4. 순종을 통한 부르심 — 룻/나오미 스타일

바울의 프론티어 정신 (롬 15:19-24)

"내가 예루살렘으로부터 두루 행하여 일루리곤까지 그리스도의 복음을 편만하게 전하였노라" "이 지방에 일할 곳이 없고" (롬 15:23)

납득이 안 가는 이 말의 뜻: - 에베소·빌립보 등에 불신자가 아직 많은데 왜 일할 곳이 없다고 하는가? - 작더라도 교회가 세워졌기 때문 — 이제 그것은 교회의 사역 - 바울은 교두보가 세워지면 다음 프론티어로 나간다

선교사 마인드 vs. 목회자 마인드 - 목회자: 세워진 교회를 든든히 하는 것 - 선교사: 부르지 않는 곳, 터가 없는 곳을 향하는 것

"선교는 초청받아 가는 곳이 아니다. 환영받는 곳은 선교지가 아니다."

9과. 평생학습

"정체기에 빠지는 것을 막는 가장 중요한 한 가지 해결책은 잘 개발된 학습 자세다" — 로버트 클린터 (풀러 신학교)

FAT 원칙: - Flexible / Adaptable / Teachable

약함과 치유의 여정: Stripping (벗겨내기) → Wrestling (씨름) → Transformation (변화) → Expansion (확장)

약함을 직시하고 받아들일 때 하나님이 치유하신다 치유된 그 상처(Scar)가 다른 사람에게 도움이 된다

  • 리더의 90%가 고립을 경험한다
  • 사색하는 사역자 (Reflective Practitioners)
  • 연간 독서 목표: 10-15권

10과. 강건하게 잘 마치기

"측정할 수 있는 결과가 아니라 궁극적인 목표에서 눈을 떼지 않도록 하는 계속적인 격려가 필요하다"

잘 마치지 못하는 이유

잘못된 재정관리 / 성적인 죄 / 가정 문제 / 권력 남용 / 자만과 야망 / 이양하지 못하는 무능력

예방 가능한 중도 포기의 원인

  1. 영성 형성과 성품의 부적절함
  2. 섬김의 희생에 대한 오해
  3. 대인관계 갈등 (가족·동료·현지인)
  4. 사역에 맞는 지식과 목적의식의 부재

잘 마치는 리더의 6가지 특징

  1. 하나님과 생명력 넘치는 관계를 끝까지 유지
  2. 배우려는 자세를 유지하고 여러 곳에서 배움
  3. 예수님을 닮은 성품
  4. 진리의 삶으로 하나님의 약속이 실재함을 보임
  5. 하나 이상의 궁극적인 공헌
  6. 운명 의식을 갖고 살며, 그것이 이루어지는 것을 봄

CMF·의료 선교 연결

기독의료인정체성: - 의료인도 세 가지 소명(mission/missionary/missions) 중 어디에 있는지 물어야 한다 - 바울의 프론티어 정신 → 의료 기술이 닿지 않는 곳으로 나가는 의료 선교사 - Stripping→Transformation 여정 = 번아웃 의료인의 회복 구조 - 잘 마치지 못하는 이유 중 재정·가정 문제는 의료인에게도 동일
