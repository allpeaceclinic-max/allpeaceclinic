5무(無)란?

없는 것 의미
로고에 십자가 없음 복음의 본질은 더 깊이, 형식적 상징은 내려놓음
새벽예배 없음 일상과 연결된 영성으로 전환
전통적 성경공부 없음 취향 기반 인격적 소그룹으로 대체
구역(셀) 제도 없음 자발적 공동체 형성
장로 직분 없음 (아직) 수평적 팀 리더십

5무 → 5UP 전환 프레임: - 십자가 없음 → 인스타그램 + 교회 공간으로 - 새벽기도 없음 → 온오프 더 연결 (일상으로) - 성경공부·구역 없음 → 성도 중심 소그룹 - 장로 없음 → 어디선가 세워짐

4부 구조 요약

1부. 배경 (Background)

Mobile 혁명 - 농업→산업→인쇄→전기→디지털→모바일 - 인쇄 = 지식의 민주화 / 모바일 = 관계와 감각의 민주화 - 일상과 관계를 실시간으로 재구성한 최초의 혁명 - 쇼츠·릴스 소비량이 미국의 2배인 한국 → 디지털 환경에 맞는 교회 소통 필요

MZ세대 = Me Generation - 노동인구의 50%, 이미 기성세대 - 도덕 규범 약화가 아니라 개인의 선택과 자유가 우선되는 가치관 - 순차적 인생 모형(학교→취업→결혼→은퇴)이 더 이상 유효하지 않다 (마우로 기옌) - 한달살기, 디지털 노마드, 웨케이션 → 삶 자체가 이동(Mobile)

K (한국적 컨텍스트) - 책은 올드파워, 모바일은 뉴파워 - 미네르바 스쿨(7개 도시 한 학기씩) — K가 오히려 선두주자 - 아직도 서구의 이야기인가? 오히려 K의 이야기다

2부. 현상 (Phenomenon)

Lifestyle — 페르소나 공간 - 청년들은 페르소나 공간에만 간다 - 모바일 혁명 → 무한한 온라인 공간 → 오프라인은 더욱 특별해야 - 여의도 더현대 사례: 9가지 페르소나, MZ세대의 놀이터 - 팬데믹 이전의 교회 = 구약 성전 모델 = 낯설고 두려운 공간 - 그러나 성전 휘장이 찢어졌다 (마 27:51) — 교회는 두려운 공간이 아니어야

Ritual — 예배의 재발견 - 종교에 관심 없는 게 아니라 제도화된 종교 형태에 관심 없는 것 - 사주·타로·MBTI → "나는 누구인가?"라는 질문의 다른 표현 - 러닝크루, 드라이 제뉴어리, 오운완, 저속노화 → 자기돌봄이 의식(ritual)화 - 영국 파크런이 새로운 교회인가? - 필사·저널·묵상 → 앱 기반 '초원', 느슨한 공동체

Local — 동네의 가능성 - 퀵턴투어: 경주 황리단길, 전주 행궁동, stay type travel - 구미라면축제, 김천김밥, 농부시장마르쉐 → 로컬이 컨텐츠가 된다 - 교회 뒷산의 변화?

3부. 적응 (Adaptation)

No Cross — Fresh Expressions - 영국 프레쉬 익스프레션스: missional · contextual · ecclesial · formational - 6단계 순환: 경청 → 섬김 → 공동체 → 제자도 → 교회 → 재생산 - Contextual church formation: 교회 구조에 사람을 끼워 맞추는 것이 아니라, 사람과 맥락에 따라 교회가 새롭게 형성 - 십자가가 사라지는 것이 아니라 더 깊은 곳에서 더욱 강력하게 이야기됨 - 디처치드(Dechurched) — 가나안 성도보다 정확한 표현 - 한국교회는 '첫째 아들'이 다니기에 좋은 교회가 되었다

Team — 수평적 리더십 - Leadership에서 Team으로 - 디지털 네이티브 + 진정성 + 사회적 책임 + 수평적 협업 - 아트 디렉터가 있는 교회 (포드 교회, 원유경 목사) - 문턱을 낮추되 영적 여정에 관심은 오히려 많다 - MZ세대에게 수평적 리더십을 이양하지 않는 교회는 성장하지 못할 것

Popup — 새로운 공간 문법 - 낯설고 새로운 공간 / 높은 밀도 / 스토리텔링 / 환대 / 미션과 참여

4부. 여정 (Journey)

  • 퍼널(깔때기, 구매 전환 중심)이 아닌 여정(경험·감정·관계 중심)
  • 포토덤프 = 블로그, 일상여정, 아카이빙, 공유 → 인스타 1장과 다르다
  • 성경은 '떠남'으로 시작 → 여정이 본질
  • 둘째 아들이 다닐 수 있는 교회, 돌아올 수 있는 교회

핵심 인사이트

  1. 그릇과 내용물의 구분 — 형식(그릇)을 바꾸는 것이 본질(내용물)을 버리는 게 아니다
  2. 이동이 본질 — 모바일 혁명은 단순한 기술 변화가 아니라 인간 존재 방식의 변화
  3. 리추얼은 죽지 않았다 — 사주·타로·러닝 모두 의미 추구의 다른 표현이다
  4. 디처치드는 영적 포기가 아니다 — 형식에 지친 것이지 복음에 지친 게 아니다
  5. 경청이 먼저다 — Fresh Expressions 6단계의 첫 단계는 항상 경청(Listening)
  6. K가 선두다 — 서구 이야기가 아니라 한국이 이 변화의 최전선

개인 성찰 및 구상

이 책이 던진 질문: CMF는 첫째 아들이 다니기 좋은 공동체인가, 아니면 둘째 아들도 돌아올 수 있는 공동체인가?

새로운 모임 형태에 대한 구상 (아이디어 수준) - 커뮤니케이션 공간: 온·오프 모두 가능, 삶을 나눌 수 있는 공간 - 사주·타로·MBTI보다 좋은 것: 복음 안에서의 자기 발견 - 제도화되지 않은 영성, 낯설고 두려운 공간이 아닌 곳 - 나이로 묶지 않는 다양한 커뮤니티 - 네이밍 아이디어: Meet Me In Him, Bridge, 먹튀 환영

CMF 사역 적용 포인트 (→ 5무교회가 온다, 황인권 참고) 1. 구역·정기모임 해체 → '저부담 고연결' 유연한 소모임(Fluid Network) 2. 권위주의 탈피 → 수평적 팀 사역 + 공감의 소통 3. 형식적 예배 대신 → 일상의 리추얼 + 멘탈 헬스 케어 4. 모바일 혁명 대응 → 소통 채널 일원화 + 숏폼 미디어 5. 구조에 사람 맞추지 않기 → Contextual Formation
