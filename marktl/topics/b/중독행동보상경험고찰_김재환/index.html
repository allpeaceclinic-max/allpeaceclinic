중독행동이 보상경험과 심리적 기능에 대한 고찰

말러의 공생적 융합개념과 코헛의 자기심리학을 중심으로 — 김재환

핵심 주장

중독 행동의 보상경험 = 말러의 공생기 융합경험의 병리적 재현 자기 발달의 결함 → 온전한 융합경험을 강박적으로 추구 → 중독

말러의 발달 단계와 융합경험

유아 자아발달 4단계: 자폐기 → 공생기 → 분리개별화 → 대상항상성(독립된 자아)

공생기 (생후 2~6개월)

  • 어머니와 자신이 동일한 시스템을 공유하는 것처럼 경계선 없는 융합 상태
  • omnipotent fusion (전능한 융합) 경험
  • 욕구 충족 대상에 대한 희미한 자각 — 전능감, 즉각적 충족

재접근기 (18개월경)

  • 독립하고 싶음 vs 여전히 함께 있고 싶음 → 양가감정
  • 양육자의 적절한 정서적 가용성이 이 긴장을 건강하게 통과시킴
  • 실패 → 파편화·분리의 두려움 → 병리적 융합 추구

코헛의 자기심리학과 중독

  • 공감적 자기대상(selfobject)의 부재 = 심각한 자기 발달 결함
  • 자기애적 외상 → 중독행동은 이 결함에 대한 방어로 작동
  • 중독 대상이 거울대상 또는 이상화된 자기대상으로 기능
  • 중독 대상과의 퇴행적 융합에서 일시적 따뜻함·전능감·안정감 경험

정상적 융합 vs 병리적 융합

정상적 융합 병리적 융합(중독)
음악·기도·명상·성관계·데이트 알코올·게임·도박·쇼핑
일시적, 자기로 돌아옴 강박적, 통제 불가, 부정적 결과에도 지속
변연계 보상회로 정상 활성화 신경생리학적 손상 야기

중독의 3요소

갈망 + 통제 어려움 + 부정적 결과임에도 지속

★ 정상적 융합의 신학적 해석

엡 5:18 — "술 취하지 말라… 오직 성령으로 충만함을 받으라"

  • 성령충만한 사람이 술 취한 사람처럼 보일 수 있음 = 융합의 모습
  • 에덴 = 공간적 융합 (신과 연결, 배우자와 한몸 → 전능감·자율감) → 타락으로 박탈

융합 = 충만 (성경적 언어): - 유아-부모 → 융합 → 자기의 탄생 - 인간-신 → 성령충만 → 인간이 되어감

교회의 융합기능 — 기독교적 개입

영적 측면

  • 교회 = 신과의 영적 융합경험을 제공하는 매개체
  • 성령충만 = 신의 전능감을 내재화 + 온전한 관계성 경험
  • 에덴의 회복을 가정에서 갈구

심리적 측면

  • 사랑이라는 매개를 통한 일시적 심리적 융합경험 제공
  • 교회 = 유사가족 경험 (초대교회 모델)
  • "짐진 자들아 다 내게로" — 존재 수용의 공간

상담적 개입

  • 인지행동치료만으로는 한계 ("마른 주정" 현상)
  • 중독 = 대상관계적 문제 + 성격구조적 결함
  • 공감적 태도의 지속적 제공 → 스트레스 내성 증가 → 자아발달 결핍 보완

★ 개인 코멘트 (이상협)

창 2장 — 부모를 떠나 아내와 연합, 한몸 분리-개별화의 과제가 건강하게 이루어져야 할 때 박탈되면 중독 위험이 높아진다.

  • 완전히 박탈된 경우에도 예수님을 통해 공급받은 예들이 있다
  • 교회 공동체의 융합경험보다 부부관계를 통한 회복이 선행되어야 하지 않을까?
  • 초대교회의 융합경험, 은혜가 넘치는 공동체의 유익에는 동의

연결된 페이지

  • 내면치유 — 자기대상, 공감적 수용, 트라우마
  • 자녀양육 — 재접근기, 양육자의 정서적 가용성
  • 교회론 — 유사가족, 융합공동체로서의 교회
  • 성화 — 성령충만 = 신과의 융합경험
