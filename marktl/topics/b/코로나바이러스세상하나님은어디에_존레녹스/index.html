신학·사회존 레녹스2026-05-05

코로나 바이러스 세상, 하나님은 어디에 계실까 (존 레녹스)

개요

  • 저자: 존 레녹스 (John Lennox) — 옥스퍼드 수학자·기독교 변증가
  • 집필 배경: COVID-19 팬데믹 초기, 고통의 의미에 대한 짧은 변증서

★ 핵심 명제

하나님을 방정식에서 제거한다고 고통이 제거되지 않는다.
고통은 그대로 남는다 — 그러나 희망이 사라진다.
그리스도인은 고통을 해결하는 사람이 아니라,
친히 고난을 겪으신 하나님을 신뢰하는 사람이다.

1. 역사적 관점 — 팬데믹은 새롭지 않다

역병 시기 사망자
안토니우스 역병 (갈레노스 역병) 165-180년 5백만
유스티니아누스 역병 (선페스트) 541-542년 2천5백만
흑사병 14세기 7천만-1억
스페인 독감 1918-1920년 2-5천만
HIV 2005-2012년 3천2백만

"우리가 예측 가능하고 안정된 세계에 익숙했을 뿐 — 사실 과거에도 늘 있었다"

2. 고통의 두 유형

유형 내용 특징
자연적 악 역병, 지진, 자연재해 인간의 직접적 책임 아님
도덕적 악 미움, 테러, 폭력, 학대, 살인 인간의 선택

※ 둘은 연결된다: 탐욕 → 벌목 → 사막화 → 지구온난화 → 자연재해

3. 세계관별 대응

범신론 — "죄 때문, 업보"

  • 욥기가 반대한다
  • 빌라도 만행·예루살렘 망대 사건: 일부는 심판이지만 우리에게 판단 권위 없다
  • 심판으로 해석하는 사람을 경계하라

무신론 — "신이 없다"

리처드 도킨스: 악을 폐기하면 코로나 바이러스도 나쁘다 할 수 없다
생명·자연·역사가 무도덕적인데 도대체 도덕이 왜 필요한가?

→ 무신론은 도움이 안 된다. 고통도 그대로이고, 희망마저 없어진다.

기독교

  • 하나님을 제거한다고 고통이 제거되지 않음
  • 하나님을 제거하면 → 궁극적인 희망이 제거된다

4. 바이러스의 본성 — 왜 존재하는가

피터 폴라드 (호주 그리피스 대학교): - 전체 바이러스 1억 가지 중 인간 파괴형은 21가지 - 대다수 바이러스는 유익하고 생명에 필수

도널드 브라운리: - 판구조의 중요성 — 지구 온도 조절 장치 - 바이러스와 지진은 생명의 필수조건

신학적 함의:

개념 내용
허용적인 뜻 악이 가능한 우주를 창조 — 로봇이 아닌 자유 존재
법령적인 뜻 하나님이 실제로 행하시는 것
  • 악은 가능하지만, 절대 하나님이 만들거나 의도하지 않으셨다
  • 심지어 이런 악을 통해서도 선을 이루신다

5. 사랑의 증거 — 성육신·십자가·부활

우리의 삶과 미래를 걸고 신뢰할 수 있는 하나님이 존재한다는 증거가 있는가?

  • 성육신: 예수님이 친히 질고를 짊어지셨다
  • 십자가: 죄와 죄책의 해결
  • 부활: 죽음을 정복, 죽음이 끝이 아님

두 개의 왕관 (Corona)

Corona = 라틴어로 '왕관', '권력과 권위의 상징'

왕관 의미
코로나바이러스 인간의 통제 불가능 — 죽음의 왕관
예수님의 가시 면류관 피조물과 창조주 간의 단절의 깊이 — 그것을 친히 짊어지심

코로나 바이러스는 확성기다 — 모든 사람이 죽는다는 확성기

6. 그리스도인의 반응

  1. 지침을 따르라 — 의학적 충고, 사회적 거리두기, 비교적 안전한 자들이 취약자를 도움
  2. 올바른 관점을 가지라 — 원래 우리는 사형선고 받은 존재, 새로운 상황이 아니다
  3. 네 이웃을 사랑하라 - 안토니우스 역병: 기독교인의 희생적 헌신 → 기독교 확산 (로드니 스타크) - 최소한 사재기하지 말라
  4. 영원을 기억하라 — 딤후 4:7-8, 의의 면류관
  5. 등반: 선한 싸움을 다 싸우고, 달려갈 길을 마치고 — 팬데믹 중의 평안
