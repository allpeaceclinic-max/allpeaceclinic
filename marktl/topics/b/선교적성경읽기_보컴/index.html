선교적 성경읽기

저자: 리처드 보컴 (Richard Bauckham)
분류: 선교적 해석학 / 성경신학 / 교육론

"선교들에 관한 성경신학이 아니라, 선교를 그 중심 관심사와 목표로 놓고 성경 전체를 읽는 방식"

1. 핵심 명제 — 성경의 중심은 선교다

보컴의 선교적 해석학은 두 가지를 구별한다:

기존 접근 선교적 성경읽기
질문 성경에서 선교 명령을 찾는다 성경 전체가 선교 이야기다
방법 선교 구절들을 모아 신학화 성경의 중심 관심사로 선교를 놓고 전체를 읽음
결과 선교에 관한 성경신학 선교를 위한 성경신학

→ 성경 이야기 전체의 중심 = 삼위일체 하나님의 선교(미씨오 데이)
→ 교회는 그 선교에 참여하는 공동체

2. 교육의 목표 전환 — 학문 길드에서 보냄받은 공동체로

제자도의 학교 → 사도적 선교 공동체 학문 길드 → 보냄 받은 공동체

  • 신학 교육의 목표: 지식 습득 → 사도적 선교 공동체 형성
  • 전략의 핵심: 증인 공동체를 세우는 것

빌 1:27 "오직 너희는 그리스도의 복음에 합당하게 생활하라 … 한마음으로 서서 한 뜻으로 복음의 신앙을 위하여 협력하는 것"

엡 4:1 "너희가 부르심을 받은 일에 합당하게 행하여"

3. 선교적 성경읽기의 위치 — vault 내 연결

보컴의 명제는 크리스토퍼 라이트의 작업과 직결된다:

보컴 라이트
선교를 중심으로 성경 전체 읽기 구약의 선교적 읽기 (하나님 백성의 선교)
삼위일체 하나님의 선교에 참여 미씨오 데이 — 교회는 하나님의 선교에 참여
사도적 선교 공동체 형성 교회론=선교론

→ 하백선시리즈 — 크리스토퍼 라이트 《하나님 백성의 선교》 15챕터 강의
→ 선교란무엇인가_스토트라이트 — 라이트의 참여선교 신학적 근거

탐구 중인 질문

Q1. "선교를 중심으로 성경 전체를 읽는다"는 것이 다른 해석 원칙(christocentric, canonical 등)과 어떤 관계인가? - 부분 답변: 선교적 읽기 ≠ 그리스도 중심 읽기를 대체 — 삼위일체적 미씨오 데이가 포괄 - 여전히 열린 질문: 강해설교 준비에서 선교적 해석학을 어떻게 실천적으로 적용하는가?

Q2. "학문 길드 → 보냄받은 공동체" 전환은 CMF/누가회 교육에 어떻게 적용되는가? - 부분 답변: CMF 스피릿 강의 — VPM 철학(Value→People→Ministry), 공동체가 먼저 - 여전히 열린 질문: 누가회 수련회/훈련 과정이 지식 전달에서 공동체 형성으로 전환되려면?
