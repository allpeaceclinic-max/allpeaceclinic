선한 영향력 — 김진수

저자: 김진수 — 비즈니스 선교사 (GMP 캄보디아)
분류: 비즈니스 선교 (BAM) 실천 가이드

"비즈니스 자체가 선교"
"두 축(비즈니스·선교)은 덧셈이 아니라 곱셈의 관계"

책 제목의 의미: "돌 제거하는 자, 옥토를 만드는 자"
→ 비즈니스 선교는 선교지의 토양 자체를 바꾸는 사역

1. BAM의 신학적 토대 — 왜 비즈니스가 선교인가

  • 비즈니스 선교는 선교의 방법이 아니라 존재 방식
  • 총체적 선교(복음 전도 + 사회 변혁 + 경제 자립)의 통합된 실천
  • 선교의 문이 닫히는 이유 = 부정직 (비자 문제 = 정직의 문제)
  • 정직(integrity) 자체가 선교 — 정직이 생명
  • 달란트 비유(마25) + 신15:9 — "우리의 악함과 하나님의 정의는 다르다" → 나누지 않는 것이 악함

2. ISI 핵심가치 — 회사의 3원칙

가치 영어 핵심 원리
정직 Integrity 노력은 덧셈, 정직은 곱셈 — 선교의 문을 여는 열쇠
나눔 Sharing 이익금의 20% 선교 + 10% 교육 / 공정한 나눔
자립 Independency 소유권 이전이 자립의 핵심 / 스스로 짓고 고쳐야

3. 비즈니스 운영 6대 원칙

원칙 핵심
소유권 투자한 사람이 소유·운영 / 후원받은 선교사의 독점 = 최악
고용·해고 인사가 만사 / 해고 후에도 관계 유지 / 해고 못하는 것이 실패 원인
손익분기점 3-5년 내 통과 / 못하면 빨리 폐업 / 직원 월급 최우선
이익분배 운영자금 저축 → 빚 갚기 → 주주배당 순서
소유권 이전 일한 대가가 돌아오지 않으면 떠남 → 지분 나눔이 해결책
선교사 노후 후원금 기록 분리 / 개인 구좌 ≠ 비즈니스 구좌 (투명성)

4. 5가지 제안 — 하지 말아야 할 것

  1. 거창하게 "새로운 선교"처럼 시작하려면 하지 말 것
  2. 개인 비즈니스 하면서 "선교한다"고 말하지 말 것
  3. 자신의 것을 투자하지 않으면 하지 말 것
  4. 생활수단으로 삼지 말 것 — 원주민을 위한 마음이 사라짐
  5. 적을 만들지 말 것 — 해고는 불가피하나 관계는 유지

5. 창업 역량

창업 전: 일에 대한 열정 / 자신감 / 리스크 테이킹 / 배수의 진 / 주인의식 / 냉철한 마음 / 성실함
창업 후: 유연성 / 흉내 내지 않기 / 직접 일하기 / 서두르지 않기 / 주인 만들어 주기 (지분 주기)

6. 핵심 통찰 — 자립과 리더십

"약한 자를 계속 포옹하는 것이 아니라, 그 중에서 리더를 선택해서 스스로 하게"
"교회를 지어 주는 것이 아니라 스스로 짓고 고쳐야"
"나누어야 커진다"
"치밀하게 준비해도 계획한 것의 70%는 일어나지 않는다. 새로운 일이 생긴다."

Q1. "정직 자체가 선교"라는 명제는 의료 선교 현장에서 어떻게 구체화되는가? - 부분 답변: 의료비 청구의 투명성, 의약품 사용의 정직성 = 선교적 행위 - 부분 답변: 선교의변질_크라이더 — 아비투스(삶의 방식) 자체가 선교 - 여전히 열린 질문: 올피스클리닉에서 ISI(정직·나눔·자립)를 어떻게 실천하는가?

Q2. 소유권 이전의 타이밍과 조건은 어떻게 판단하는가? - 부분 답변: "일한 대가가 돌아오지 않는다고 느끼면 떠난다" → 역할 있는 지분 구조 필요 - 부분 답변: 선교지에어떤교회를세울것인가 — 역량 강화(capacity building)가 선행되어야 - 여전히 열린 질문: 인터서브 파트너 사역에서 소유권 이전의 기준은?
