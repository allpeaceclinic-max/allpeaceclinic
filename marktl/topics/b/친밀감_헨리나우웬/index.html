친밀감 (Intimacy)

저자: 헨리 나우웬 (Henri J.M. Nouwen)
분류: 심리학 / 영성 / 종교적 성장

핵심 주제

종교적 성장은 발달 단계마다 다른 위기를 통과하며 이루어진다.
성숙한 신앙의 핵심은 억압이나 저항이 아닌 통합(integration)이다.

5단계 종교적 발달

1. 유아기 (출생–5세)

① 출생–생후 18개월: 객관적 실체 발견 - 종교적 차원: 하나님을 자신의 일부로 생각

② 18개월–만 3세: 언어 습득 - 종교적 차원: 기도, 감탄, 암송 - 위험: 기도가 대화의 통로가 아닌 하나님을 지배하는 방편이 될 수 있음

③ 만 4–5세: 양심의 형성 (외부 → 내면화) - 아빠의 권위 → 내면의 가치 기준으로 전환 - 아빠의 한계 인식 → 절대적 존재 필요 → '하나님'으로의 진정한 전환이 일어나야

프로이트 비판 ("환각의 미래"): 종교란 유아기 생활의 연장, 하나님은 은신처에 대한 갈망의 투사
→ 여기서 멈추면 신앙은 라이너스의 담요 — 위안의 도구에 불과

2. 유년기 (5–12세)

  • 가정에서 배운 것이 강화·수정·확장·붕괴되는 시기
  • 학교와 종교가 분리되기 쉬운 시기 ↔ 성숙한 종교는 본질상 통합적
  • 종교적 내면 갈등 없이 이 시기에 고착되는 경우 많음 → 위안만 남음

Allport의 성숙 정의: 성장하는 지능과 관련 경험이 서로 보조를 맞추려는 갈망으로 활성화될 때 일어남

3. 사춘기 (12–18세)

  • 외부 세계뿐 아니라 자기 내면의 깊고 낯선 감정을 만나는 시기
  • 종교적 발달의 중대 고비: 내면의 갈등을 이해하고 수용할 수 있는가?

세 가지 반응: 1. 외면 (억압) 2. 부정 — 어두운 욕망을 통제하려 함 3. 통합 ← 성숙한 방향

"알곡을 거두려면 가라지도 함께 두어야 한다"
그리스도는 두 강도 사이에서 십자가를 지셨다
갈등의 해답은 저항도 억압도 아닌 통합이다

4. 청년기 (대학)

  • 부모의 가정을 떠났지만 새 가정을 이루지 않은, 가장 자유로운 시기
  • 간접적인 것 → 직접적인 것으로의 전환 (실험·모험을 통해)
  • 신뢰 — 구도적 종교를 가능케 하는 토대

5. 성인기

  • 권태 = 단절된 경험 — 말, 개념, 생각, 행동이 서로 연결되지 않는 상태
  • 새로운 시각 = 믿음으로 통합 — 단절을 연결로 바꾸는 힘

핵심 통찰

단계 위기 성숙한 통과
유아기 하나님 = 위안 도구로 고착 진정한 절대자로 전환
유년기 종교와 삶의 분리 통합적 신앙 형성
사춘기 내면 갈등을 억압/부정 자기수용 + 통합
청년기 간접 신앙에 머묾 직접적 신뢰·실험
성인기 권태 (단절) 믿음으로 통합

윤명숙 전도사님 설교와의 연결

나우웬의 발달 단계론은 윤명숙 전도사님 설교에서 반복적으로 구현된다.

나우웬 윤명숙 설교
5단계 종교 발달 청소년 이해와 의사소통 (눅 2:52) — "태아기→유아기→학령기→청소년기→청년기→성인기, 인간의 일생을 통한 하나님 형상 회복"
라이너스 담요 — 신앙이 위안 도구로 고착될 위험 고전 13:11 어린아이의 모습 — "성인아이"는 신앙을 인정·위안의 수단으로만 쓴다
사춘기: 통합 vs. 억압 요 8:32 내면세계 — "외면과 내면을 성경적 관점으로 건강하게 일치시키는 것" = 통합
내재과거아 고착 고전 13:11 — "성인아이, 내재과거아, 키워서 내보내야 한다" (미실다인과 동일 개념)
성인기: 권태 = 단절 → 믿음으로 통합 렘 2:13,19 내면세계를 들여다 보는 방법 — 생수의 근원을 버리고 터진 웅덩이를 파는 것 = 단절
프로이트 비판: 종교 = 투사 요 8:32 — "예수 그리스도 없는 상담은 어느 정도는 가능하지만 근본적으로 어렵다"

핵심 공명: 나우웬이 심리학으로 진단한 "종교 발달의 고착"을 윤명숙 전도사님은 "성인아이"라는 목회 언어로 동일하게 다룬다. 두 사람 모두 심리학적 통찰을 신학적 변화의 출발점으로 삼는다.
