Has Anything Changed? — 팬데믹 이후 시대 하나님의 선교 참여

핵심 질문

코비드는 '파송으로서의 선교(Mission-as-sending)' 패러다임에 도전장을 내밀었다. 우리는 이제 어디로 가야 하는가?

하나님의 선교 참여는 전 지구적 추세뿐 아니라 지역적 변화에 의해서도 깊이 영향받는다. 하나님은 특정한 시기, 특정한 지역, 특정한 백성 가운데 일하신다. (마 16:2-4 — 시대의 표적)

VUCA 시대의 8가지 도전

Volatility·Uncertainty·Complexity·Ambiguity — 불안정·불확실·복잡·모호

  1. 전 지구적 불평등
  2. 사회-경제 시스템의 실패 → 이스라엘 경제 원칙(경작·부채·토지) 재발견 필요
  3. 기후변화와 생태계 오염
  4. 거대한 이동과 강제이주 — 초기 5세기 비자발적 이동과 유사
  5. 경제적·군사적 권력의 변화 — 미중패권 교체, 중국으로의 전환
  6. 민족주의·종교적 극단주의·폭력
  7. 테크놀로지와 인공지능
  8. 인간다움의 재구성

크리스텐덤 패러다임의 4가지 특징 (알란·엘레노어 크라이더)

알렌크라이더

  1. 선교 = 지역에 의해 정의 — 기독교 지역 → 비기독교 지역으로 흐르는 것
  2. 선교 = 선교사라 불리는 소수 특별 계층의 역할
  3. 선교 = 교회가 하는 '어떤 일' — 전문 선교기관을 통한 과업
  4. 선교의 목표 = 회심자 획득 + 교회 개척

→ 이 패러다임은 코비드로 인해 근본적으로 도전받고 있다.

글로벌 교회 패러다임으로의 전환 — 7가지 수정

(데이빗 보쉬의 에큐메니컬 선교 패러다임 → '글로벌 교회 패러다임' 으로 재명명)

  1. 선교 초점: 영토(지역) 개념 → 종족(People) 개념
  2. 서구 중심 → 모든 곳에서 모든 곳으로(From everywhere to everywhere)
  3. 서구 = 선교사 파송지 → 서구 = 선교 수신지 (후기 기독교, 역방향 선교)
  4. 장기 헌신 → 단기 선교 급증 (매년 수백만 팀 동원)
  5. 다양한 선교기관 증가
  6. 선교단체 파송 + 지역 교회 직접 파송 병행
  7. 선교 목표의 확장: 회심·교회 개척 중심 → 통합적·총체적 목표로

글로벌 교회 패러다임의 5가지 형태

1. 하나님의 선교(Missio Dei)와 삼위일체

  • 하나님이 주체 (스스로 있는 자)
  • 경계를 넘는 선교 과정을 이끄신 것은 성령
  • 에큐메니컬 신학이 높이 평가, 복음주의는 늦게 수용

2. '선교' 용어에 새로운 의미를

  • 서구 관점에서 만들어진 선교 슬로건 = 하나님의 선교가 아닌 자기 입장의 표현
  • 성경적이라기보다 문화적인 슬로건들을 점검해야

3. 복음의 총체성 품기

선교를 '과업(Task)'으로 보면, Doing이 된다. "무엇을 했는가?"보다 "어떻게 살고 있는가?"가 더 중요하다.

  • '인종적 그룹(ethnic group)' = 20세기 문화인류학이 만든 개념 → 예수가 말한 '모든 민족(열방)'에 이 개념을 적용하는 것은 시대착오적 해석
  • 선교의 성경적 증거는 완수(completion)를 지향하지 않는다 → 목표: 공의롭고 은혜로운 그리스도의 주권 아래 모든 인간, 모든 피조물의 참된 번영

4. 지역 중심성

  • 제이 마탕가: "선교의 미래는 토착인에게 있다(the future of missions is indigenous)"
  • 타문화 선교사의 역할: 방향 제시·주도가 아니라 동반자로서 섬김

5. 충성스런 증거를 위한 자리로서 공동체

  • 대안 공동체 + 복음 선포 + 급진적 관용 (초기 교회 모델)
  • → 교회론 크라이더 섹션과 연결

★ 교회를 향한 세 가지 초청

1. 겸손과 의존에의 부르심

  • 하나님의 선교를 중심으로 선교학 재정립
  • 선교사의 주도권이 아닌 성령의 주권에의 의존

2. '가라'에서 '오라'로의 부르심

막 3:13-15 — 먼저 자기와 함께 있게 하시고, 그 다음에 보내셨다

  • 파송이 아니라 동행이 먼저
  • 하나님이 이미 사람들 가운데서 일하고 계신다
  • 선교 = 하나님이 어디서 어떻게 이미 일하시는가를 볼 수 있도록 돕는 것

3. 단일 접근 → 다면적 접근으로의 부르심

  • 포스트팬데믹 시대의 선교는 하나의 모델·방법으로 수렴될 수 없다
  • VUCA 맥락에 맞는 다양한 참여 방식이 필요
