# 🏥 영닥터의 저장소 (allpeaceclinic)

가정의학과 전문의, 선교단체 이사로서의 기록과 지식 관리를 위한 공간입니다.

## 📌 주요 관심 분야
- **Medical Missions:** 선교지와 의료 현장의 연결
- **PKM (Personal Knowledge Management):** 옵시디언을 활용한 제2의 뇌 구축
- **AI Automation:** 클로드와 파이썬을 활용한 업무 자동화

## 🚀 진행 중인 프로젝트
- 원노트 데이터 옵시디언 이관 및 구조화
- CMF 재정 정책 분석 및 리포트 작성
- 선교단체, NGO 이사로서 문서 및 조직 관리
- 옵시디언을 활용한 강의안 만들기
